考向1-2

水解陷阱 + 气体摩尔体积陷阱

三年全考
🎯 考向1 — 盐溶液离子数目 = 水解陷阱重灾区
2025广东 第10题 C
C．在 1L 0.1mol/L 的 NH₄Cl 溶液中，NH₄⁺ 的数目为 0.1NA
错误。n(NH₄Cl) = 0.1 × 1 = 0.1mol，但 NH₄⁺ 是弱碱阳离子，水解：NH₄⁺ + H₂O ⇌ NH₃·H₂O + H⁺，实际 n(NH₄⁺) < 0.1mol，即数目 < 0.1NA。
连续三年完全相同的套路！
• 2025：NH₄Cl 溶液 NH₄⁺ → 水解
• 2024：NH₄NO₃ 溶液 NH₄⁺ → 水解
• 2023：NaHCO₃ 溶液 HCO₃⁻ → 水解+电离（双通道消耗）

常见可水解离子一览：
弱碱阳离子：NH₄⁺、Fe³⁺、Al³⁺、Cu²⁺、Fe²⁺、Zn²⁺
弱酸阴离子：CO₃²⁻、HCO₃⁻、SO₃²⁻、CH₃COO⁻、PO₄³⁻
酸式根（双通道）：HCO₃⁻、HSO₃⁻、H₂PO₄⁻、HPO₄²⁻
  • 看到含以上离子的盐溶液+数目 → 直接判定 "< nNA"，不需要算！
  • 只有强酸强碱盐（NaCl、KNO₃、Na₂SO₄等）的离子数目才严格等于 c·V·NA
  • 酸式根最危险：既水解又电离，消耗更彻底
🎯 考向2 — 气体体积→物质的量 = 必须"标准状况"
2024广东 第10题 D
D．Na 与 H₂O 反应生成 11.2L H₂，转移电子数目为 NA
错误。题目只说"生成 11.2L H₂"，未说明标准状况（0℃、101kPa），无法用 22.4L/mol 换算物质的量，因此无法计算转移电子数。

对比2025广东D选项（正确）："标准状况下的 22.4L Cl₂ 与足量 H₂ 反应，形成的共价键数目为 2NA" — 多了"标准状况下"四个字，天壤之别！
没有"标准状况"四个字 → 直接排除
• 标准状况：0℃、101kPa，Vm ≈ 22.4 L/mol
• 常温常压：25℃、101kPa，Vm ≈ 24.5 L/mol（不能用22.4）
• 高温高压：Vm 明显偏离 22.4

特殊记忆：SO₃ 在标况下是固体，HF、H₂O 在标况下是液体 — 都不能用 22.4 L/mol！
  • 口诀："不见标况不算 Vm，常温常压也不行"
  • 题干有"标准状况"才用 22.4 L/mol，否则直接排除涉及气体体积换算的选项
  • 标况下为液态/固态的物质：SO₃、H₂O、HF、CCl₄、CHCl₃、Br₂ — 体积≠22.4n
考向3-4

化学键计数 + 氧化还原电子转移

高频
🎯 考向3 — 共价键/σ键/π键计数
2024广东 第10题 A（✅正确选项）
A．26g C₂H₂ 中含有 σ 键的数目为 3NA
正确。
• M(C₂H₂) = 26 g/mol，n = 26/26 = 1 mol
• C₂H₂ 结构：H-C≡C-H
• 2 个 C-H σ键 + 1 个 C-C σ键（三键中的1个σ）= 3 个 σ 键
• 另外还有 2 个 π 键（不算在σ键内）
⇒ 1mol C₂H₂ 含 3NA 个 σ 键 ✅
有机分子键计数高频错误：
• 分不清 σ 键和 π 键：单键全是 σ，双键=1σ+1π，三键=1σ+2π
• 漏数 C-H 键：每个 H 原子必连 1 个 σ 键
• 苯环特殊：每个 C-C 键等同于 1.5 键（1σ + 0.5π），所以 1mol 苯含 12mol σ 键（6C-C × 1σ + 6C-H × 1σ）
• 错误变体："共价键总数"包括 σ+π，"σ键数"只数 σ
  • 化学键计数的正确选项往往是答案（可精确计算，无陷阱）
  • 步骤：写出结构式 → 数单键(C-H/C-C/C-O等) → 乘以NA
  • 常考有机物：C₂H₂(3σ)、C₂H₄(5σ)、C₂H₆(7σ)、CH₄(4σ)、C₆H₆(12σ)
  • 无机物：NH₃(3σ)、H₂O(2σ)、CO₂(2σ)、N₂(1σ+2π)
🎯 考向4 — 氧化还原电子转移数目
2025广东 第10题 B
B．1mol Fe 与水蒸气完全反应，生成 H₂ 的数目为 2NA
错误。
铁与水蒸气反应：3Fe + 4H₂O(g) → Fe₃O₄ + 4H₂
⇒ 3mol Fe → 4mol H₂ ⇒ 1mol Fe → 4/3 mol H₂ = 4/3 NA 个H₂分子
选项说 2NA，大了 50%。

注意：Fe 与水蒸气反应生成的是 Fe₃O₄（Fe 平均价态 +8/3），不是 Fe₂O₃（+3）。
电子转移题的两大陷阱：
• 1) 产物搞错：Fe+水蒸气 → Fe₃O₄（不是Fe₂O₃）；Fe+Cl₂ → FeCl₃（不是FeCl₂）；Na+O₂(常温) → Na₂O，Na+O₂(点燃) → Na₂O₂
• 2) 系数算错：电子转移数 = 变价原子数 × 化合价变化值，不是简单的 1:1

经典对比：
1mol Fe + 足量稀H₂SO₄ → 转移 2mol e⁻（Fe→Fe²⁺）
1mol Fe + 足量稀HNO₃(过量) → 转移 3mol e⁻（Fe→Fe³⁺）
两个结果差 1.5 倍，命题人最爱混淆！
  • 先确认反应方程式是否正确（产物是不是你以为的那个）
  • 再算电子转移总数 = 变价原子摩尔数 × |Δ化合价|
  • 常考反应电子转移速查：
    • 2Na₂O₂ + 2H₂O → 4NaOH + O₂↑：2mol Na₂O₂ → 转移 2mol e⁻（歧化）
    • 3Cu + 8HNO₃(稀) → 3Cu(NO₃)₂ + 2NO↑ + 4H₂O：3mol Cu → 转移 6mol e⁻
    • 2Al + 2NaOH + 2H₂O → 2NaAlO₂ + 3H₂↑：2mol Al → 转移 6mol e⁻
考向5-8

可逆反应 / 隐含条件 / 粒子结构 / pH

常考
🎯 考向5 — 可逆反应 ≠ 完全转化
综合考向（广东卷TA题中高频出现）
常见命题形式："1mol N₂ 与 3mol H₂ 充分反应，生成 NH₃ 的数目为 2NA"
典型错误："标准状况下，2mol SO₂ 与 1mol O₂ 充分反应，转移电子数目为 4NA"
以上说法全部错误。
• N₂ + 3H₂ ⇌ 2NH₃ 是可逆反应，反应不能进行到底，生成的 NH₃ < 2mol
• SO₂ + O₂ ⇌ SO₃ 也是可逆反应，不能完全转化
• 弱电解质的电离也是可逆的：1mol CH₃COOH 溶于水，电离出的 H⁺ < 1mol
看到这些关键词 → "充分反应"不等于"完全反应"
• "充分反应/一定条件下反应" → 可逆反应未进行到底
• 常考可逆反应：N₂+3H₂⇌2NH₃、2SO₂+O₂⇌2SO₃、2NO₂⇌N₂O₄、Cl₂+H₂O⇌HCl+HClO
• 弱电解质电离：CH₃COOH⇌CH₃COO⁻+H⁺、NH₃·H₂O⇌NH₄⁺+OH⁻
• 盐类水解也是可逆的！（回到考向1）
  • 做法：看到涉及可逆反应的选项 → 产物数目必定 < nNA → 直接排除
  • 记忆："是可逆就不完全，电水解也同理" — 两大可逆体系：化学平衡 + 电离水解平衡
🎯 考向6-8 — 混合物/粒子结构/pH 快速判断
考向典型陷阱判定方法示例
6. 混合物/隐含条件 "一定量的xx"、"适量的yy" — 没给具体量 数据不全 → 直接排除 "Na与H₂O反应生成11.2L H₂"（没说是标况 → 数据不全）
7. 粒子结构 同位素中子数不同、D₂O≠H₂O 核对原子组成：质子数=原子序数，中子数=质量数-质子数 2025广东A：1mol ¹⁸O₂ 比 ¹⁶O₂ 多 4NA 中子（不是2NA）
8. pH与H⁺/OH⁻ 给了pH但没给溶液体积 → 无法求数目 n(H⁺) = 10^(-pH) × V，缺V → 不能算 "pH=1的盐酸中H⁺数目为0.1NA"（没体积，无解）
🔬 同位素中子数速查
  • ¹H（氕）：0中子 | ²H/D（氘）：1中子 | ³H/T（氚）：2中子
  • ¹²C：6中子 | ¹³C：7中子 | ¹⁴C：8中子
  • ¹⁶O：8中子 | ¹⁸O：10中子
  • H₂O中子数=8（仅O），D₂O中子数=1×2+8=10，T₂O中子数=2×2+8=12
总结

NA 选择题满分攻略

必看
⚡ 做题步骤（30秒内完成）
Step 1：扫描所有选项，标记四大"直接排除"信号 🚫
  • ① 有可水解离子 + 数目 = nNA → 排除
  • ② 有气体体积 + 无"标准状况" → 排除
  • ③ 有可逆反应/弱电解质 + "数目等于" → 排除
  • ④ 数据不全（缺体积、缺浓度、缺具体量）→ 排除
Step 2：在剩余选项中，找"可精确计算"的 ✅
  • 化学键计数（给质量 + 给结构式 → 可精确算）
  • 质子/中子/电子数（给物质的量 + 给原子组成 → 可精确算）
  • 标准状况下的气体摩尔数（标况 + 22.4L → 可精确算）
Step 3：最终确认 — 算一遍

对疑似正确的选项，快速用草稿纸算一遍：n = m/M, N = n·NA，确认无误后选定。
