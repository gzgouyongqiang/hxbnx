考点 1

硫单质与二氧化硫（SO₂）的性质

必考
📝 真题举例
2025年广东卷 · 第7题（选择题）
能满足下列物质间直接转化关系，且推理成立的是：

O₂      →      O₂      →      H₂O      →      NaOH(或HCl)
单质 X → 氧化物1 → 氧化物2 → 酸(或碱) → 盐
A.X可为铝，盐的水溶液一定显酸性
B.X可为硫，氧化物1可使品红溶液褪色
C.X可为钠，氧化物2可与水反应生成H₂
D.X可为碳，盐的热稳定性：NaHCO₃ > Na₂CO₃

答案：B

硫燃烧生成 SO₂（氧化物1，可使品红褪色），SO₂可被催化氧化为SO₃（氧化物2），SO₃与水反应生成H₂SO₄（酸），再与NaOH反应生成Na₂SO₄（盐），全部转化关系成立。

A错误：Al₂O₃不与水反应；C错误：Na₂O₂与水反应生成NaOH和O₂，无H₂；D错误：Na₂CO₃热稳定性 > NaHCO₃，D的结论反了。

2023年广东卷 · 第9题（选择题）
按图装置进行实验。将稀硫酸全部加入Ⅰ中的试管，关闭活塞。下列说法正确的是（  ）

装置描述：Ⅰ中试管含Na₂SO₃固体（水浴），Ⅱ中为品红溶液，Ⅲ中为Na₂S溶液，Ⅳ中为含酚酞的NaOH溶液。

A. Ⅰ中试管内的反应，体现H₂SO₄的氧化性
B. Ⅱ中品红溶液褪色，体现SO₂的还原性
C. 在Ⅰ和Ⅲ的试管中，都出现了浑浊现象
D. 撤掉水浴，重做实验，Ⅳ中红色更快褪去
A.体现H₂SO₄的氧化性
B.体现SO₂的还原性
C.Ⅰ和Ⅲ都出现浑浊
D.撤水浴后Ⅳ更快褪色

答案：C

Ⅰ中：Na₂SO₃ + H₂SO₄ → Na₂SO₄ + SO₂↑ + H₂O，是复分解反应（非氧化还原），H元素的化合价不变，A错误。

Ⅱ中品红褪色体现SO₂的漂白性（非还原性），B错误。

Ⅰ试管中有S析出（Na₂SO₃变质部分含S），Ⅲ试管中SO₂ + 2Na₂S → 3S↓ + 2Na₂O，两者均有浑浊，C正确。

撤掉水浴，反应速率减慢，SO₂产生变慢，Ⅳ中酚酞褪色变慢，D错误。

⚠️ 陷阱提示

陷阱1：SO₂的"三重面孔"

SO₂使品红褪色 = 漂白性（可逆）；SO₂使KMnO₄褪色 = 还原性（不可逆）；SO₂使石蕊变红 = 酸性氧化物通性。高考最爱把"漂白性"和"还原性"混着考，务必分清！

陷阱2：SO₂是电解质吗？

SO₂本身不能电离，是非电解质。其水溶液能导电是因为SO₂ + H₂O → H₂SO₃（亚硫酸是弱电解质，能电离）。"水溶液能导电=电解质"是经典陷阱。

陷阱3：SO₂的漂白vs Cl₂的漂白

SO₂漂白是可逆的（与有色物质结合成不稳定化合物，加热恢复）；Cl₂漂白是不可逆的（强氧化性破坏色素分子）。如果某溶液先通SO₂再通Cl₂，漂白效果可能反而变差——因为SO₂ + Cl₂ + 2H₂O → H₂SO₄ + 2HCl，两者互相消耗！

✅ 避坑指导

SO₂性质速记口诀：一酸二还三漂白

一酸：酸性氧化物通性（与碱、碱性氧化物、水反应）

二还：还原性（与KMnO₄、Cl₂、Br₂、O₂等强氧化剂反应）

三漂白：使品红等有机色素褪色（可逆！）

同时SO₂也有弱氧化性：SO₂ + 2H₂S → 3S↓ + 2H₂O

📌 考点1核心结论：SO₂的三大性质是高考选择题的"标配"考点，每年必考1-2道。解题关键是分清"使XX褪色"到底对应的是漂白性、还原性还是酸性。
考点 2

浓硫酸的三大特性

必考
📝 真题举例
2025年广东卷 · 第8题（选择题）
下列陈述Ⅰ与陈述Ⅱ均正确，且两者间具有因果关系的是（  ）
A.Ⅰ：浓硝酸保存在棕色瓶中；Ⅱ：浓硝酸有强氧化性
B.Ⅰ：浓硫酸使蔗糖变黑；Ⅱ：浓硫酸有脱水性
C.Ⅰ：Fe²⁺遇K₃[Fe(CN)₆]变蓝；Ⅱ：KSCN可检验Fe³⁺
D.Ⅰ：MnO₂与浓盐酸生成Cl₂；Ⅱ：Cl₂和SO₂可反应

答案：B

B：浓硫酸使蔗糖变黑（C₁₂H₂₂O₁₁ → 12C + 11H₂O），是浓硫酸按水的组成比脱去有机物中的氢和氧，体现脱水性，Ⅰ和Ⅱ因果关系成立。

A：浓硝酸保存在棕色瓶中是因为见光易分解（4HNO₃ → 4NO₂↑ + O₂↑ + 2H₂O），不是强氧化性导致，因果关系不成立。

2022年广东卷 · 第5题（选择题）
若将铜丝插入热浓硫酸中进行如图所示的探究实验（a~d均为浸有相应试液的棉花），下列分析正确的是（  ）
A.Cu与浓硫酸反应，只体现H₂SO₄的酸性
B.a处紫色石蕊变红，说明SO₂是酸性氧化物
C.b或c处褪色，均说明SO₂具有漂白性
D.试管底部白色固体说明反应中无水生成

答案：B

Cu + 2H₂SO₄(浓) →⁄△ CuSO₄ + SO₂↑ + 2H₂O

A错误：生成CuSO₄体现酸性，生成SO₂体现强氧化性，浓硫酸同时体现酸性和强氧化性。

B正确：SO₂溶于水生成H₂SO₃（亚硫酸），使石蕊变红，说明SO₂是酸性氧化物。

C错误：b处品红褪色 = 漂白性（正确）；c处KMnO₄褪色 = SO₂的还原性，不是漂白性。

D错误：白色固体是CuSO₄（无水），浓硫酸的吸水性将生成的水吸收，并不代表反应中无水生成。

2023年广东卷 · 第10题（选择题改编）
下列关于浓硫酸的说法正确的是（  ）
A.蔗糖在浓硫酸作用下形成多孔炭，体现浓硫酸的氧化性和脱水性
B.浓硫酸可干燥SO₂、HCl、NH₃等气体
C.常温下铁在浓硫酸中溶解，体现浓硫酸的强氧化性
D.铜与浓硫酸反应中，被还原的H₂SO₄与参加反应的H₂SO₄之比为1∶2

答案：A

蔗糖脱水变炭（脱水性），炭进一步与浓硫酸反应（C + 2H₂SO₄ → CO₂↑ + 2SO₂↑ + 2H₂O）体现强氧化性，A正确。

B错误：NH₃是碱性气体，会与浓硫酸反应（NH₃ + H₂SO₄ → NH₄HSO₄），不能干燥。

C错误：常温下铁在浓硫酸中钝化（表面生成致密氧化膜），不溶解。

D正确（但选A更优）：Cu + 2H₂SO₄ → CuSO₄ + SO₂↑ + 2H₂O，2mol参加反应，1mol被还原，比值为1:2。此说法本身正确，但本题A更全面地体现了浓硫酸的特性。

⚠️ 陷阱提示

陷阱1：脱水性 vs 吸水性

吸水性：吸收现成的水（如干燥气体、吸水放热）。例：浓硫酸干燥Cl₂、SO₂。

脱水性：按水的组成比（H:O=2:1）脱去有机物中的H和O。例：蔗糖变黑、纸张变黑。

两者本质不同！吸水是物理+化学过程（形成水合H₂SO₄），脱水是化学变化。

陷阱2：钝化 ≠ 不反应

铁、铝在冷浓硫酸/冷浓硝酸中钝化，是化学变化（表面生成致密氧化膜），不是不反应。但题目说"常温下可以存放在铁罐中"是正确的。

陷阱3：浓硫酸与Cu反应中的量比

Cu + 2H₂SO₄(浓) → CuSO₄ + SO₂↑ + 2H₂O，参加反应的浓硫酸共2mol，其中只有1mol作氧化剂（S +6→+4），另1mol提供SO₄²⁻（表现酸性）。高考常考"被还原的H₂SO₄与参加反应的H₂SO₄之比 = 1:2"。

✅ 避坑指导

浓硫酸三大特性对照表：

特性本质典型例子不可干燥的气体
吸水性吸收游离态的水干燥Cl₂、SO₂、CO₂—
脱水性按H₂O组成比脱去有机物中的H、O蔗糖变黑、纸张炭化—
强氧化性+6价S被还原Cu→CuSO₄+SO₂↑；C→CO₂+SO₂↑NH₃、H₂S、HI（强还原性气体）
📌 考点2核心结论：浓硫酸三大特性每年必考。解题技巧：看到"变黑"→脱水性；看到"干燥"→吸水性；看到"与金属/非金属反应生成SO₂"→强氧化性。因果关系判断题要区分"原因"和"结果"。
考点 3

硫酸根（SO₄²⁻）的检验

常考
📝 真题举例
2025年广东卷 · 第4题（选择题改编）
下列有关SO₄²⁻检验的说法正确的是（  ）
A.向某溶液中加BaCl₂溶液有白色沉淀，再加稀盐酸沉淀不溶解，说明含SO₄²⁻
B.向某溶液中先加稀盐酸酸化，再加BaCl₂溶液有白色沉淀，一定能证明含SO₄²⁻
C.检验SO₄²⁻时，先加盐酸是为了排除Ag⁺和CO₃²⁻等离子的干扰
D.向燃煤中加入生石灰脱硫，不存在沉淀溶解平衡转化

答案：C

检验SO₄²⁻的标准操作：先加足量稀盐酸（排除Ag⁺产生的AgCl、CO₃²⁻产生的BaCO₃等），再加BaCl₂溶液，出现不溶于酸的白色沉淀（BaSO₄）。

A错误：未先酸化，Ag⁺也会产生不溶于盐酸的白色沉淀（AgCl），无法排除。

B错误：加入的盐酸若不足量，可能无法完全排除CO₃²⁻；且"一定能"过于绝对，需确认加入的是足量稀盐酸。

D错误：燃煤中加CaO脱硫涉及CaSO₄的沉淀溶解平衡，D的表述本身是错的（2025广东原题中A选项即此内容，被判定为错误）。

⚠️ 陷阱提示

陷阱1："先酸后钡"的试剂选择

酸化试剂必须选稀盐酸（HCl），不能选稀硫酸（H₂SO₄）——因为H₂SO₄本身会引入SO₄²⁻！

沉淀试剂常用BaCl₂或Ba(NO₃)₂，不能选Ba(OH)₂——会引入OH⁻，影响后续判断。

陷阱2：常见干扰离子

• Ag⁺：加BaCl₂直接产生AgCl（白色沉淀），加盐酸可排除
• CO₃²⁻：加BaCl₂产生BaCO₃（白色沉淀，溶于盐酸），加盐酸可排除
• SO₃²⁻：加BaCl₂产生BaSO₃（白色沉淀），加盐酸可排除（BaSO₃溶于盐酸且有SO₂气味）
• Pb²⁺：加BaCl₂产生PbCl₂（白色沉淀，微溶），需注意区分

✅ 避坑指导

SO₄²⁻检验操作口诀：

取液 → 加足量稀盐酸 → 无沉淀无气泡 → 加BaCl₂ → 白色沉淀不溶 = 含SO₄²⁻

关键三步：①取样 ②先加盐酸（排除干扰） ③再加BaCl₂（确认沉淀）

切记：盐酸要足量！加BaCl₂前溶液必须呈酸性！

📌 考点3核心结论：SO₄²⁻检验是离子检验题的"模板"，考试频率中等但从未缺席。掌握"先酸后钡"的操作顺序，理解每一步的目的，即可轻松得分。
考点 4

含硫化合物的转化关系

必考
📝 真题举例
2024年广东卷 · 第17题（大题）
含硫物质种类繁多，在一定条件下可相互转化。

(1) 实验室中，浓硫酸与铜丝反应，所产生的尾气可用______（填化学式）溶液吸收。

(2) 工业上，烟气中的SO₂可在通空气条件下用石灰石浆液吸收，生成石膏（CaSO₄·2H₂O）。该过程中，______（填元素符号）被氧化。

(3) 工业锅炉需定期除水垢，其中的硫酸钙用纯碱溶液处理：CaSO₄ + Na₂CO₃ ⇌ CaCO₃ + Na₂SO₄（反应Ⅰ）。兴趣小组探究溶液浓度对反应(Ⅰ)速率的影响，设计系列实验并记录滴定数据（数据略）。

(4) 继续探究反应(Ⅰ)平衡的建立：将硫酸钙加入Na₂CO₃溶液中，测定pH随时间的变化曲线。甲同学认为pH不变即达到平衡，乙同学提出两个假设……

答案要点：

(1) NaOH（或Na₂CO₃等碱性溶液）。Cu + 2H₂SO₄ →⁄△ CuSO₄ + SO₂↑ + 2H₂O，尾气SO₂用碱液吸收。

(2) S（硫元素）。SO₂中S为+4价，CaSO₄中S为+6价，化合价升高，被氧化。

(3) 配制溶液用到的仪器：烧杯、酸式滴定管（选BD）。滴定计算结果：x = 20.0 mL；平均反应速率之比 v(a)∶v(b) = 6∶5。

(4) 验证沉淀无CaSO₄：将反应混合物固液分离→取沉淀→加过量稀盐酸→有气体（CO₂）产生且沉淀全溶→加BaCl₂溶液无白色沉淀→说明沉淀均为CaCO₃。

建立平衡的优化方案：将最少CaSO₄加入Na₂CO₃溶液中，在50℃和搅拌条件下测定pH，当pH不变时，加入Na₂SO₄固体，若pH变大，说明反应(Ⅰ)平衡已建立。

⚠️ 陷阱提示

陷阱1：SO₂的氧化产物不止SO₃

SO₂催化氧化 → SO₃（+6价）；但工业上"烟气脱硫"的产物是CaSO₄（石膏），也是+6价。不要以为只有高温催化才是氧化，碱性条件下的氧化转化同样常见。

陷阱2：沉淀转化实验的"变量控制"

CuSO₄ + Na₂CO₃的实验中，Cu²⁺ + CO₃²⁻ → 碱式碳酸铜（绿色），不是简单的CuCO₃。浓硫酸使蔗糖变黑后生成的气体是SO₂和CO₂的混合气体。

✅ 避坑指导

含硫物质转化关系图（核心记忆）：

H₂S（-2）→⬆氧化 S（0）→⬆氧化 SO₂（+4）→⬆氧化 SO₃（+6）→ H₂SO₄

每个"⬆氧化"都可以对应具体的化学反应和试剂，构成完整的含硫转化链。

常考转化方向：

• H₂S → S：2H₂S + O₂ → 2S↓ + 2H₂O（不完全燃烧）

• S → SO₂：S + O₂ →⁄点燃 SO₂

• SO₂ → SO₃：2SO₂ + O₂ ⇌⁄V₂O₅,△ 2SO₃（可逆！催化氧化）

• SO₃ → H₂SO₄：SO₃ + H₂O → H₂SO₄

• SO₂ + H₂S → 3S↓ + 2H₂O（归中反应）

📌 考点4核心结论：含硫化合物的转化关系是高考大题（化工流程+实验探究）的高频素材。掌握S的-2、0、+4、+6四种价态之间的转化条件，理解氧化还原规律，是解题的关键。
