物质的组成、性质和转化

第一章 化学物质及其变化
📚 核心知识

🔬 元素

具有相同核电荷数（质子数）的同一类原子的总称

决定：种类由质子数决定

⚛️ 原子

化学变化中的最小粒子

组成：质子 + 中子 + 电子

💧 分子

保持物质化学性质的最小粒子

存在：气体、液体、部分固体

⚡ 离子

带电的原子或原子团

分类：阳离子（+）、阴离子（-）

原子 = 质子 + 中子 + 电子　｜　离子 = 原子 ± 电子
纯净物只含一种物质，化学式唯一

混合物含多种物质，无固定化学式

⚠️ 注意空气是混合物，但成分相对固定
🎯 考点精讲

考点1：物质分类的两条主线

树状分类法：按物质种类逐级细分（单质→金属/非金属→氧化物/酸/碱/盐）
交叉分类法：同一种物质可属于多个类别

考点2：同素异形体

由同种元素形成的不同单质

常见：O₂ 和 O₃（金刚石、石墨、C₆₀）
注意：性质不同，但化学式不同不算同素异形体！

考点3：物质转化规律

金属 → 氧化物 → 碱 → 盐
2Na + O₂ → Na₂O　　Na₂O + H₂O → NaOH

非金属 → 氧化物 → 酸 → 盐
S + O₂ → SO₂　　SO₂ + H₂O → H₂SO₃
✍️ 例题精讲
例1 下列说法正确的是（　　）

① 元素是物质的具体存在形式

② 元素的化学性质主要由最外层电子数决定

③ 原子是化学变化中的最小粒子，不可再分

④ 带电的原子或原子团叫离子

✅ 答案：②③④
💡 解题思路

① 错——物质是具体存在，元素是抽象概念
② 对——最外层电子决定化学性质
③ 对——原子是化学变化最小粒子，但物理上可分
④ 对——离子的定义

例2 下列属于同素异形体的是（　　）

A. ¹²C 和 ¹⁴C　　B. O₂ 和 O₃　　C. CO 和 CO₂　　D. Na 和 NaCl

✅ 答案：B
💡 解题思路

同素异形体 = 同元素 + 不同单质
A 错——同位素，是同一元素的不同原子
B 对——O₂ 和 O₃ 都是氧元素的不同单质
C 错——CO 和 CO₂ 是化合物，不是单质
D 错——Na 是单质，NaCl 是化合物

🏋️ 变式训练
📝 趁热打铁，做完再看答案哦~
练习1：下列关于物质分类的说法正确的是（　　）
A. 冰水共存物是混合物
B. 纯净物一定由同种分子构成
C. 只含一种元素的物质一定是纯净物
D. 空气属于混合物

本题满分：6分
🙋 点击查看答案
答案：D
A错——冰水都是H₂O，是纯净物
B错——离子化合物由离子构成
C错——O₂和O₃混合是混合物
D对——空气含N₂、O₂等多种物质
练习2：写出下列物质转化的化学方程式：
① Ca → CaO → Ca(OH)₂ → CaCO₃
② S → SO₂ → H₂SO₃ → Na₂SO₃

🙋 点击查看答案
①
2Ca + O₂ ═══ 2CaO
CaO + H₂O ═══ Ca(OH)₂
Ca(OH)₂ + CO₂ ═══ CaCO₃↓ + H₂O

②
S + O₂ ═══ SO₂
SO₂ + H₂O ⇌ H₂SO₃
H₂SO₃ + 2NaOH ═══ Na₂SO₃ + 2H₂O
📌 本讲小结
原子结构
质子数 = 核电荷数 = 原子序数
同素异形体
同元素 + 不同单质
纯净物 vs 混合物
有无固定化学式
转化规律
金属/非金属 → 氧化物 → 碱/酸 → 盐
