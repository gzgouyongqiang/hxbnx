三道经典计算题，看看你能否选择最优方法快速破解
【前测1·守恒法】 向NaOH溶液中通入一定量CO₂后，溶液中含Na⁺ 0.4mol、HCO₃⁻ 0.2mol、CO₃²⁻ 0.1mol。则通入的CO₂的物质的量为：
A. 0.15 mol   B. 0.30 mol   C. 0.45 mol   D. 0.60 mol
✅ B正确！用C元素守恒：n(CO₂) = n(HCO₃⁻) + n(CO₃²⁻) = 0.2 + 0.1 = 0.3 mol。Na⁺的量是干扰信息！守恒法一步到位，根本不需要写方程式。
【前测2·电子守恒法】 4mol Sb 与含有 NaNO₃ 和 O₂ 的混合物反应，其中 3mol O₂ 参与反应，反应中总电子转移为：
A. 12 mol   B. 16 mol   C. 20 mol   D. 24 mol
✅ C正确！用得失电子守恒：Sb作还原剂（0→+5），4mol Sb转移 4×5 = 20mol e⁻。这就是2025湖南卷第8题的核心思路——不需要配平方程式，直接盯住还原剂的化合价变化即可。
【前测3·关系式法】 工业上用铝土矿（含Al₂O₃）制备金属铝。若要生产 2.7t 铝，至少需要含 Al₂O₃ 质量分数为 80% 的铝土矿多少吨？
A. 3.2t   B. 5.1t   C. 6.4t   D. 8.5t
✅ C正确！用关系式法：Al₂O₃ → 2Al（Al守恒）
n(Al) = 2.7×10⁶/27 = 10⁵ mol，n(Al₂O₃) = 10⁵/2 = 0.5×10⁵ mol
m(纯Al₂O₃) = 0.5×10⁵×102 = 5.1t，m(铝土矿) = 5.1/80% = 6.375t ≈ 6.4t

⚠️ 三道全对说明计算基本功扎实，哪道错了就重点看对应板块！

板块一

"守恒法"——化学计算的第一思维

🔴 必须掌握
📝 核心原理

守恒法的本质：化学反应前后，某些量的总和不变。抓住这些"不变量"，可以跳过复杂的中间步骤，直接建立等量关系。

守恒类型含义典型应用场景
原子守恒反应前后某元素的原子个数不变多步反应中某元素最终产物求算、混合物组成
电子守恒氧化剂得电子总数 = 还原剂失电子总数氧化还原反应计算（转移电子数、产物比例）
电荷守恒溶液中阳离子正电荷 = 阴离子负电荷离子浓度计算、判断离子浓度大小关系
质量守恒反应前后物质总质量不变差量法的基础、固液质量变化分析
💡 守恒法思维口诀
⚡ 看到元素传递 → 原子守恒
⚖️ 看到氧化还原 → 电子守恒
➕ 看到溶液离子 → 电荷守恒
📏 看到质量变化 → 质量守恒 + 差量法
📝 真题举例
2025·新课标卷·第7题 — HF溶液中的元素守恒与电荷守恒
总浓度为 0.1 mol/L 的 HF 水溶液中存在平衡：HF ↔ H⁺ + F⁻、F⁻ + H₂O ↔ HF + OH⁻。下列叙述正确的是：

A. M、N分别为 c(F⁻)、c(HF) 关系曲线
B. Ka(HF) × Ka2(HF) = 10-7.80
C. 溶液中 c(HF) = c؃ − c(F⁻) − c(H⁺)
D. pH = 1 时，c(Na⁺) + c(H⁺) = c(F⁻) + c(OH⁻)
✅ B正确。

❌ C错误（元素守恒！）：溶液中HF的存在形式有 HF、F⁻、HF₂⁻（二氟氢根）三种！
→ 根据F元素守恒：c؃(HF) = c(HF) + c(F⁻) + c(HF₂⁻)
→ C选项漏了 HF₂⁻，所以 c(HF) ≠ c؃ − c(F⁻) − c(H⁺)

❌ D错误（电荷守恒！）：电荷守恒式右侧缺少了 HF₂⁻ 的浓度项。
→ 正确应为：c(Na⁺) + c(H⁺) = c(F⁻) + c(OH⁻) + c(HF₂⁻)

💡 本题启示：HF₂⁻ 容易被忽略！写守恒式时必须把所有含目标元素的微粒都列出来。

① 元素守恒漏算微粒——❌ HF溶液中含F的微粒有三种：HF、F⁻、HF₂⁻，不是两种！类似的，CO₂水溶液中含C微粒有CO₂、H₂CO₃、HCO₃⁻、CO₃²⁻四种

② 电荷守恒漏算离子——❌ 任何带电微粒都要计入！HF₂⁻带−1电荷，不能漏掉

③ 质子守恒与电荷守恒混淆——❌ 电荷守恒：Σ正电荷=Σ负电荷；质子守恒：c(H⁺)=c(OH⁻)+Σ其他碱消耗的H⁺。两者含义完全不同

写守恒式三步法：

  1. 全部列举。 把溶液中含目标元素/电荷的所有微粒全部列出来，不要遗漏
  2. 确定系数。 每个微粒贡献多少原子/多少电荷（注意HF₂⁻含2个F原子但只带1个负电荷）
  3. 建立等式。 左边=右边，检查是否有遗漏

常见"隐形微粒"速查：

  • HF溶液：别忘了 HF₂⁻
  • H₃PO₄溶液：别忘了 H₂PO₄⁻、HPO₄²⁻、PO₄³⁻（共四种含磷微粒）
  • H₂SO₃溶液：别忘了 HSO₄⁻
  • 任何水溶液：别忘了 H⁺ 和 OH⁻
2025·湖南卷·第8题 — Sb与NaNO₃/O₂反应的电子转移
NaSbO₃是一种合成聚酯的催化剂，可用"硝酸钠法"制备。下列说法错误的是：

C. 反应中氧化剂和还原剂的物质的量之比为 4:7
D. 反应中消耗 4mol Sb，转移 20mol e⁻
❌ C错误，D正确！

电子守恒分析：Sb是还原剂（0→+5），4mol Sb共失去 4×5 = 20mol e⁻ ✔️ D正确

氧化剂是谁？NaNO₃中的N（+5→某低价态）和 O₂（0→−2）同时做氧化剂
→ 氧化剂总量 = NaNO₃(4mol) + O₂(3mol) = 7mol
→ 还原剂 = Sb(4mol)
→ 氧化剂:还原剂 = 7:4（不是4:7！）✔️ C说反了

💡 解题关键：看到氧化还原反应——第一时间找还原剂化合价变化×物质的量 = 总电子转移，不需要完整配平！

① 氧化剂可能有多个——❌ 本题中NaNO₃和O₂同时做氧化剂，氧化剂总量=两者之和

② 氧化剂/还原剂比搞反——❌ C选项故意说成4:7，实际是7:4

③ 电子守恒不需要配平——✅ 只需盯住还原剂的化合价升高×物质的量即可

2025·浙江卷·第12题 — 电化学合成中的电子守恒
通过电化学方法制备NaBrO₃。已知氧化性：H₂O₂ > NaBrO₃。下列说法不正确的是：

D. 原料足量时，理论上电路中转移 0.2mol e⁻ 可得到 0.1mol NaBrO₃
❌ D错误！

由得失电子守恒：电路中转移 0.2mol e⁻，若只靠电解直接氧化Br⁻，确实生成0.1mol NaBrO₃。
但是！阳极上生成的 H₂O₂ 也具有氧化性（题给：氧化性 H₂O₂ > NaBrO₃），H₂O₂ 可以继续将 Br⁻ 氧化为 NaBrO₃。这部分氧化不消耗电路中的电子！
→ 实际得到的 NaBrO₃ > 0.1mol

💡 本质：电化学+化学氧化的"串联反应"中，不能只按电子守恒算电化学步骤的产物。
2025·浙江卷·第19题(2) — 草酸铜钾制备中的双元素守恒
步骤I：75℃下，水中依次加入 0.032mol K₂C₂O₄、0.020mol CuCl₂、0.012mol H₂C₂O₄充分反应。根据K元素守恒和C元素守恒可知，溶液M中含有 0.016mol KHC₂O₄、0.008mol K₂C₂O₄。求pH。
双元素守恒联立方程：

📐 K守恒：2×0.032 = 2n(KHC₂O₄) + 2n(K₂C₂O₄) → a + b = 0.032
📐 C守恒：2×0.032 + 2×0.012 = 2a + 2b → a + b = 0.044
联立：n(KHC₂O₄) = 0.016mol，n(K₂C₂O₄) = 0.008mol

📏 c(H⁺) ≈ √(Ka×c) → pH ≈ 4 ✔️

💡 两个守恒式联立，不需要知道具体反应方程式就能算出各物质的量！

板块二

"关系式法"——多步反应一步到位

🔴 高频必考
📝 核心原理

关系式法：在多步连续反应中，找到初始反应物与最终产物之间的物质的量关系，然后一步计算，跳过所有中间步骤。

🎯 关系式法的适用条件
  • 多步反应中某元素的原子完全传递（无损失）
  • 工业流程题中从原料到最终产品
  • 多个平行反应中产物合并的情况

💡 建立关系式的核心：找"不变元素"——该元素从原料到产品，原子个数不变

📝 真题举例
经典关系式法 — 工业流程中的元素传递
工业流程关系式关键
铝土矿→AlAl₂O₃ → 2AlAl守恒（注意Al₂O₃含2个Al）
硫铁矿→H₂SO₄FeS₂ → 2H₂SO₄S守恒（1个S→1个H₂SO₄）
NH₃→NO→HNO₃NH₃ → NO → HNO₃N守恒（转化率需乘）
FeS₂→SO₂→H₂SO₄FeS₂ → 2H₂SO₄S守恒（中间产物不需要管）
硫铁矿制硫酸的完整推导：
4FeS₂ + 11O₂ → 2Fe₂O₃ + 8SO₂  …(1)
2SO₂ + O₂ → 2SO₃  …………………(2)
SO₃ + H₂O → H₂SO₄  …………………(3)
从(1)：4FeS₂ → 8SO₂ → 8H₂SO₄，约分：FeS₂ → 2H₂SO₄
💡 不需要三步分别算！一步关系式搞定。实际生产中还需乘转化率。
2025·新课标卷·第8题相关 — 碘回收流程中的关系式
实验室利用沉淀法从含碘废液中回收碘。含碘废液中I⁻总浓度为 0.05mol/L，取 200mL 废液处理，最终得到 1.14g I₂ 晶体。求碘的回收率。
关系式法：2I⁻ → I₂（I守恒）
n(I⁻总) = 0.05 × 0.2 = 0.01mol
理论 n(I₂) = 0.01/2 = 0.005mol
理论 m(I₂) = 0.005 × 254 = 1.27g
实际得到 1.14g
回收率 = 1.14/1.27 × 100% = 89.8%
💡 中间的沉淀、溶解、氧化还原等步骤全部跳过，只盯住I元素从废液到I₂晶体。
⚠️ 陷阱

① 忘记乘转化率——❌ 工业生产中每一步都有损耗，实际产量 = 理论产量 × 各步转化率的乘积

② 元素传递不完整——❌ 中间步骤有元素损失（如尾气排放、废渣带走），不能直接用守恒关系式

③ 关系式系数写错——❌ Al₂O₃ → 2Al（不是1:1！），FeS₂ → 2H₂SO₄（也不是1:1！）。一定要核对化学式中原子的个数

✅ 避坑指导

关系式法标准操作流程：

  1. 找核心元素。 哪个元素从原料完整传递到产品？
  2. 写首尾化学式。 原料（含1个该元素）→ 产品（含n个该元素）
  3. 调整系数。 确保该元素左右两边原子数相等
  4. 代入计算。 n(原料) → n(产品) → m(产品)，别忘了乘转化率

高考常考关系式（建议默写）：

原料产品关系式
FeS₂H₂SO₄FeS₂ ~ 2H₂SO₄
Al₂O₃AlAl₂O₃ ~ 2Al
N₂HNO₃N₂ ~ 2HNO₃
CCO₂C ~ CO₂
Fe（+水蒸气）Fe₃O₄3Fe ~ Fe₃O₄

板块三

"差量法 + 极值法"——利用差值和极端情况速算

🔴 高频易错
📝 核心原理
📊 差量法

利用反应前后质量差、体积差、物质的量差建立比例关系。当题目给出反应前后的变化量时，直接用差量列比例。

公式：反应物消耗量 / 理论差量 = 实际差量 / 实际消耗

适用于：固体增重/减重、气体体积缩小、溶液质量变化等

🎯 极值法（平均值法）

假设混合物中只含其中一种成分（极端情况），计算出一个理论值。如果实际值介于两个极值之间，说明是混合物。

常用于：判断混合物组成、确定平均相对分子质量范围、确定化学式等

📝 真题举例
经典差量法题 — 金属与酸反应的固体质量变化
将 6.5g 锌放入足量稀硫酸中，完全反应后，溶液质量增加了 6.3g。求生成 H₂ 的体积（标况）。
差量法：Zn + H₂SO₄ = ZnSO₄ + H₂↑
理论差量：每消耗65g Zn（1mol），放出2g H₂（1mol），溶液增加 65 − 2 = 63g
设实际消耗 Zn 为 x g：x / 63 = 6.3 / 63 → x = 6.3g
n(H₂) = 6.3/65 = 0.097mol，V(H₂) = 0.097 × 22.4 = 2.17L
💡 不需要单独求Zn和H₂各自的量，直接用"差量比"一步到位！
经典极值法题 — 混合金属的平均摩尔电子质量
由两种金属组成的混合物 12g，与足量盐酸反应，收集到标况下 11.2L H₂。则该混合物可能是：
A. Fe 和 Zn   B. Mg 和 Al   C. Cu 和 Zn   D. Na 和 Mg
极值法（平均值法）：
n(H₂) = 11.2/22.4 = 0.5mol，n(e⁻转移) = 1.0mol
平均摩尔电子质量 = 12/1.0 = 12 g/mol·e⁻

各金属的 M/化合价：Fe(+2):28 | Zn(+2):32.5 | Mg(+2):12 | Al(+3):9 | Na(+1):23
两种金属的 M/化合价 必须一个≥12，一个≤12：
• A: 28和32.5 都≥12 → ❌
• B: 12和9 → Mg恰好=12 → 接近但不完全匹配
• D: 23和12 → 12~23之间包含12 → ✅
💡 答案D（Na和Mg）。极值法让你不需要分别计算每种组合！
2025·广东卷·第14题 — 水系电池理论容量计算（电子守恒法）
一种高容量水系电池，放电时电极材料每转移1mol电子对应的理论容量为 F（96500 C/mol）。充电时 16g S 能提供的理论容量为 300500 C——该说法是否正确？
电子守恒法计算容量：
n(S) = 16/32 = 0.5mol
关键：每个S原子转移多少电子？
如果 S(0→+2)：0.5×2 = 1mol e⁻ → 容量 = 96500 C
如果 S(0→+4)：0.5×4 = 2mol e⁻ → 容量 = 193000 C
D选项说 300500 C ≈ 96500×3.11 → 不是整数倍，D说法有误
💡 容量 = 转移电子数 × F。关键在于正确判断化合价变化。
⚠️ 陷阱

① 差量法的方向搞反——❌ 固体质量是增加还是减少？气体体积是增大还是缩小？要看清关键词

② 极值法边界遗漏——❌ 一种金属不反应（如Cu），极值之一应该是"只有另一种金属反应"

③ 理论差量算错——❌ 必须代入具体方程式计算。如 Fe + CuSO₄ = FeSO₄ + Cu，固体增重 = 64−56 = 8g/mol

④ 极值法只适合两组分——❌ 三组分混合物需要更多信息

✅ 避坑指导

五种计算方法选择速查表：
