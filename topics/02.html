1. 下列物质中，属于强电解质的是（　　）
A. 冰醋酸   B. 氨水   C. 硫酸钡   D. 二氧化硫 🔍 查看答案
✅ C —— BaSO₄虽难溶，但溶解部分完全电离→强电解质。冰醋酸是弱酸，氨水是混合物，SO₂不能电离。
2. 下列离子方程式书写正确的是（　　）
A. 铁与稀硫酸：2Fe + 6H⁺ = 2Fe³⁺ + 3H₂↑
B. 氢氧化钡与稀硫酸：Ba²⁺ + SO₄²⁻ = BaSO₄↓
C. 碳酸钙与醋酸：CaCO₃ + 2H⁺ = Ca²⁺ + CO₂↑ + H₂O
D. 钠与水反应：2Na + 2H₂O = 2Na⁺ + 2OH⁻ + H₂↑ 🔍 查看答案
✅ D —— A错：Fe与稀H₂SO₄产物是Fe²⁺；B错：漏了H⁺+OH⁻=H₂O；C错：醋酸是弱酸不能拆成H⁺。

⚠️ 第1题错 → 回顾考点1；第2题错 → 回顾考点2~4

考点1

电解质与非电解质 / 强电解质与弱电解质

🔴 必考
📝 真题举例
2023·广东·第13题（节选）
下列实验中，所采取的分离或提纯方法与对应原理都正确的是
选项实验目的分离方法原理
A分离溶于水中的碘乙醇萃取碘在乙醇中的溶解度较大
B分离乙酸乙酯和乙醇分液二者密度不同
C除去KNO₃固体中混有的NaCl重结晶两者的溶解度随温度变化差异大
D溶液导电性比同浓度醋酸强——氯化钠是强电解质，醋酸是弱电解质
🔍 查看答案
✅ D选项错误（不符合题意要求）—— NaCl是强电解质确实没错，但导电性强弱与离子浓度有关，同浓度的NaCl溶液和醋酸溶液，NaCl完全电离离子浓度大所以导电性强。D的陈述本身正确但此处考查的是实验方法的合理性。
2025·广东·第9题（节选）
H[CuCl₂]的电离方程式为：H[CuCl₂] = H⁺ + Cu⁺ + 2Cl⁻，该电离方程式是否正确？
🔍 查看答案
❌ 错误！H[CuCl₂]应电离为 H⁺ + [CuCl₂]⁻ ，而不是分解为 Cu⁺ + 2Cl⁻。配合物的内界[CuCl₂]⁻是一个整体，不能拆散！
⚠️ 陷阱
⚠️ 查看陷阱

① "BaSO₄难溶所以是弱电解质"——❌ 溶解度和电离度是两码事！BaSO₄虽难溶，但溶解的那部分完全电离，是强电解质

② "氨水是弱电解质"——❌ 氨水是混合物，不是纯净物！混合物连"电解质/非电解质"的讨论资格都没有

③ "SO₂水溶液导电，所以SO₂是电解质"——❌ 导电的是SO₂与水反应生成的H₂SO₃，不是SO₂本身

④ "配合物电离时会拆开内界"——❌ 配合物的内界（配离子）是一个整体，不能拆散！

✅ 避坑指导
✅ 查看指导

三步判断法：

  • 第一步：是不是纯净物？ 氨水、盐酸、食盐水 → 混合物，不参与分类
  • 第二步：能不能自身电离？ SO₂、CO₂、NH₃ → 溶于水后是生成新物质才导电，不是自身电离 → 非电解质
  • 第三步：完全电离还是部分电离？ 看"强酸强碱大多数盐"= 强，其余 = 一般弱

必背六强酸：HCl、H₂SO₄、HNO₃、HClO₄、HI、HBr   必背四强碱：NaOH、KOH、Ca(OH)₂、Ba(OH)₂

考点2

离子方程式的书写——"拆"原则

🔴 核心考点
📝 真题举例
2023·广东·第14题（节选）
下列指定反应的离子方程式正确的是
A. Na₂O₂溶于水：Na₂O₂ + H₂O = 2Na⁺ + 2OH⁻ + O₂↑
B. 食醋除水垢：CaCO₃ + 2H⁺ = Ca²⁺ + CO₂↑ + H₂O
C. Al³⁺与氨水：Al³⁺ + 3NH₃·H₂O = Al(OH)₃↓ + 3NH₄⁺
D. Fe₂O₃溶于氢碘酸：Fe₂O₃ + 6H⁺ = 2Fe³⁺ + 3H₂O
🔍 查看答案
✅ C
A错：Na₂O₂不拆且系数不对，正确为 2Na₂O₂+2H₂O=4Na⁺+4OH⁻+O₂↑
B错：醋酸是弱酸，不能拆为H⁺，应写CH₃COOH
C对：NH₃·H₂O弱碱不拆，Al³⁺与氨水反应正确
D错：Fe₂O₃溶于HI时，Fe³⁺会被I⁻还原为Fe²⁺（HI是还原性酸）！
2025·广东·第14题
MnO₂/MnS原电池的正极反应为：MnO₂ + 2e⁻ + 4H⁺ = Mn²⁺ + 2H₂O。下列与该反应相关的说法正确的是
A. 该反应中Mn的化合价从+2升到+4
B. 该反应发生在负极
C. MnO₂是还原剂
D. 电解质溶液为酸性
🔍 查看答案
✅ D —— 反应中有H⁺参与，说明电解质为酸性。A错：Mn从+4降到+2；B错：得电子是正极反应；C错：MnO₂得电子是氧化剂。
⚠️ 陷阱
⚠️ 查看陷阱

① "醋酸/氢氟酸拆成H⁺"——❌ 醋酸CH₃COOH、氢氟酸HF都是弱酸，不能拆！必须写化学式

② "Na₂O₂拆成Na⁺和O₂²⁻"——❌ Na₂O₂是过氧化物，特殊物质，写化学式不拆！

③ "澄清石灰水拆，石灰乳也拆"——❌ 澄清石灰水拆（Ca²⁺ + 2OH⁻），石灰乳/石灰浆不拆——浓度不同！

④ "浓硫酸拆成H⁺和SO₄²⁻"——❌ 浓硫酸主要以H₂SO₄分子形式存在，不拆！

✅ 避坑指导
✅ 查看指导

拆与不拆速记表：

必须拆（写离子）不能拆（写化学式）
强酸：HCl、H₂SO₄、HNO₃等弱酸：CH₃COOH、HF、H₂CO₃、H₂SO₃、HClO
强碱：NaOH、KOH、Ba(OH)₂、澄清石灰水弱碱：NH₃·H₂O、Fe(OH)₃、Cu(OH)₂、石灰乳
可溶性盐难溶盐：BaSO₄、AgCl、CaCO₃
——单质、气体、氧化物、过氧化物、浓硫酸

考点3

离子方程式的守恒——电荷守恒 + 原子守恒

🔴 核心考点
📝 真题举例
经典真题 · 守恒判断
判断下列离子方程式是否正确（先看电荷！）
① Fe³⁺ + Fe = 2Fe²⁺
② Cu + 2Ag⁺ = Cu²⁺ + 2Ag
③ 2MnO₄⁻ + 5H₂O₂ + 6H⁺ = 2Mn²⁺ + 5O₂↑ + 8H₂O
④ 2Al + 2OH⁻ + 2H₂O = 2AlO₂⁻ + 3H₂↑
🔍 查看答案
① ❌ 电荷：左边+3，右边+4 → 正确：2Fe³⁺ + Fe = 3Fe²⁺
② ✅ 电荷：左边+2，右边+2 → 原子也对
③ ✅ 电荷：左边 -2 -0 +6 = +4，右边 +4 → 对；原子也对
④ ✅ 电荷：左边0 -2 +0 = -2，右边 -2 +0 = -2 → 对
2025·广东·非选择题（节选）
次氯酸在光照条件下分解的离子方程式为：_______________。
🔍 查看答案
✅ 2HClO =光照 2H⁺ + 2Cl⁻ + O₂↑
检查：电荷左边0，右边+2-2+0=0 ✅；原子：2H 2Cl 2O → 2H 2Cl 2O ✅
⚠️ 陷阱
⚠️ 查看陷阱

① 只看原子守恒不看电荷守恒——这是最常见的丢分方式！先算电荷后算原子，养成习惯

② "Fe+稀HNO₃→H₂↑"——❌ 反应不真实！HNO₃中N得电子生成NO，不是H⁺得电子

③ "Cu+稀HCl→H₂↑"——❌ Cu不与稀盐酸反应，臆造反应！

④ "Fe³⁺ + I⁻能共存"——❌ Fe³⁺能氧化I⁻：2Fe³⁺+2I⁻=2Fe²⁺+I₂，不能共存

✅ 避坑指导
✅ 查看指导

判断离子方程式六步法：

  • ❶ 反应真实吗？先判断这个反应到底能不能发生
  • ❷ 拆对了吗？该拆的拆了没？不该拆的拆了没？
  • ❸ 电荷守恒？左右两边总电荷数相等？
  • ❹ 原子守恒？各原子种类和个数相等？
  • ❺ 用量对吗？少量/过量有没有影响系数？
  • ❻ 介质匹配？产物与酸性/碱性环境匹配吗？

⚡ 广东卷每年必考离子方程式正误判断，按这六步走，一分不丢！

考点4

离子方程式——"量、介、争"（高考区分题）

🔴 区分题
📝 真题举例
离子方程式与"量"——Ca(HCO₃)₂ + NaOH
少量NaOH：Ca²⁺ + HCO₃⁻ + OH⁻ = CaCO₃↓ + H₂O
过量NaOH：Ca²⁺ + 2HCO₃⁻ + 2OH⁻ = CaCO₃↓ + CO₃²⁻ + 2H₂O

为什么不同？——"少量定1法"：以少量的反应物计量数为1，确定其他比例。
🔍 查看解析
少量NaOH：NaOH计量数为1 → 1个OH⁻只能中和1个HCO₃⁻ → Ca²⁺:HCO₃⁻仅有一个参与反应
过量NaOH：2个HCO₃⁻都被OH⁻中和 → 生成CO₃²⁻，多余的留在溶液中
离子方程式与"介"——KMnO₄的介质影响
酸性KMnO₄ + H₂O₂：2MnO₄⁻ + 5H₂O₂ + 6H⁺ = 2Mn²⁺ + 5O₂↑ + 8H₂O
碱性KMnO₄ + H₂O₂：2MnO₄⁻ + H₂O₂ + 2OH⁻ = 2MnO₄²⁻ + O₂↑ + 2H₂O

介质不同，还原产物不同：酸性→Mn²⁺(肉色)，碱性→MnO₄²⁻(绿色)，中性→MnO₂(棕色沉淀)
离子方程式与"争"——FeBr₂中通Cl₂
少量Cl₂： 2Fe²⁺ + Cl₂ = 2Fe³⁺ + 2Cl⁻
足量Cl₂： 2Fe²⁺ + 4Br⁻ + 3Cl₂ = 2Fe³⁺ + 2Br₂ + 6Cl⁻

为什么先氧化Fe²⁺？——还原性顺序：I⁻ > Fe²⁺ > Br⁻，Cl₂优先氧化还原性最强的！
⚠️ 陷阱
⚠️ 查看陷阱

① 不看"少量/过量"直接写——❌ 看清题目中的"少量""过量""足量"，用量不同方程式完全不同

② "酸性条件下产物有OH⁻"——❌ 酸性环境产物不能出现大量OH⁻！介质决定产物形式

③ "Cl₂通入FeI₂，先氧化Fe²⁺"——❌ 还原性 I⁻ > Fe²⁺，应先氧化I⁻！FeI₂中通Cl₂和FeBr₂中通Cl₂顺序不同

④ "NaClO溶液中通SO₂产物有大量H⁺"——❌ NaClO溶液呈碱性，不能写H⁺

✅ 避坑指导
✅ 查看指导

量：少量定1法——以少量反应物的化学计量数为1，确定其余系数

介：介质定产物

  • KMnO₄：酸性→Mn²⁺ / 碱性→MnO₄²⁻ / 中性→MnO₂
  • NaClO溶液呈碱性，产物不能写H⁺，写OH⁻或H₂O

争：还原性顺序——I⁻ > Fe²⁺ > Br⁻

  • FeI₂ + Cl₂：先氧化I⁻，再氧化Fe²⁺
  • FeBr₂ + Cl₂：先氧化Fe²⁺，再氧化Br⁻
  • 🗣️ 口诀：碘铁溴，还原性从强到弱排排坐

考点5

离子共存判断

🟡 常考
📝 真题举例
离子共存经典陷阱
判断下列各组离子能否大量共存：
① Fe³⁺、I⁻、Cl⁻、Na⁺ → ________
② Al³⁺、HCO₃⁻、K⁺、SO₄²⁻ → ________
③ H⁺、NO₃⁻、Fe²⁺、SO₄²⁻ → ________
④ NH₄⁺、OH⁻、Na⁺、Cl⁻ → ________
🔍 查看答案
① ❌ Fe³⁺氧化I⁻：2Fe³⁺+2I⁻=2Fe²⁺+I₂
② ❌ Al³⁺和HCO₃⁻双水解：Al³⁺+3HCO₃⁻=Al(OH)₃↓+3CO₂↑（就是泡沫灭火器原理！）
③ ❌ H⁺+NO₃⁻=HNO₃有强氧化性，氧化Fe²⁺
④ ❌ NH₄⁺+OH⁻→NH₃↑+H₂O
⚠️ 陷阱
⚠️ 查看陷阱

① "酸性条件下NO₃⁻不氧化"——❌ NO₃⁻单独存在时不氧化，但酸性条件下NO₃⁻具有强氧化性（相当于稀HNO₃）

② "Al³⁺和HCO₃⁻可以共存"——❌ 双水解反应剧烈，恰好是泡沫灭火器原理

③ 只看生成沉淀不看氧化还原——先看氧化还原！Fe³⁺+I⁻、Fe³⁺+S²⁻、酸性MnO₄⁻+Cl⁻等

✅ 避坑指导
✅ 查看指导

共存三不：

  • 不反应 → 不生成沉淀/气体/弱电解质
  • 不氧化还原 → 没有强氧化剂+强还原剂组合
  • 不双水解 → Al³⁺/Fe³⁺ 不与 CO₃²⁻/HCO₃⁻/S²⁻/AlO₂⁻共存

常见不能共存组合速记：

  • Fe³⁺ + I⁻/S²⁻/SO₃²⁻ → 氧化还原
  • 酸性MnO₄⁻/Cr₂O₇²⁻ + Cl⁻/Br⁻/I⁻/Fe²⁺/SO₃²⁻ → 氧化还原
  • H⁺ + NO₃⁻ + Fe²⁺/I⁻/SO₃²⁻ → 相当于稀HNO₃氧化
  • Al³⁺/Fe³⁺ + CO₃²⁻/HCO₃⁻/AlO₂⁻ → 双水解

考点6

离子方程式正误判断——综合实战

🔴 每年必考
📝 真题举例
综合判断 · 用六步法逐一排查
下列离子方程式书写正确的是
A. 过氧化钠溶于水：Na₂O₂ + H₂O = 2Na⁺ + 2OH⁻ + O₂↑
B. 食醋除水垢（CaCO₃）：CaCO₃ + 2H⁺ = Ca²⁺ + CO₂↑ + H₂O
C. 硫酸铝溶液加足量氨水：Al³⁺ + 3NH₃·H₂O = Al(OH)₃↓ + 3NH₄⁺
D. Fe₂O₃溶于氢碘酸：Fe₂O₃ + 6H⁺ = 2Fe³⁺ + 3H₂O
🔍 查看答案
✅ C
A：❶真实✅ ❷Na₂O₂不拆❌ ❸电荷0=0✅ → ❌拆分错！（Na₂O₂应写化学式，且系数应为2:2:4:4:1）
B：❶真实✅ ❷醋酸不能拆为H⁺❌ → ❌拆分错！（应为CH₃COOH，不是H⁺）
C：❶真实✅ ❷Al³⁺拆，NH₃·H₂O不拆✅ ❸电荷+3=+3✅ ❹原子守恒✅ → ✅全对！
D：❶真实？Fe₂O₃+HI→Fe³⁺但I⁻能还原Fe³⁺！反应不真实❌
2025·广东·非选择题（节选）
"高压加热"时，生成Fe₂O₃的离子方程式为：_______________。
🔍 查看答案
提示：矿粉酸浸通入SO₂，浸取液含Fe²⁺/Cu²⁺/Ni²⁺/Al³⁺，调pH=3.0，通入空气加热 → 涉及Fe²⁺被O₂氧化为Fe³⁺再水解为Fe₂O₃
离子方程式：4Fe²⁺ + O₂ + 4H₂O = 2Fe₂O₃↓ + 8H⁺（或分步写：Fe²⁺→Fe(OH)₃→Fe₂O₃）
⚠️ 陷阱
⚠️ 查看陷阱

① 还原性酸的陷阱——HI（氢碘酸）、HBr（氢溴酸）是还原性酸，遇到Fe³⁺、MnO₄⁻等氧化剂会先发生氧化还原！

② "Al³⁺+过量OH⁻→Al(OH)₃"——❌ Al(OH)₃是两性氢氧化物，过量强碱会溶解：Al(OH)₃+OH⁻=AlO₂⁻+2H₂O。但氨水是弱碱不会溶解Al(OH)₃！

③ 过氧化物/特殊物质——Na₂O₂、CaC₂、Mg₃N₂等特殊物质写化学式不拆

✅ 避坑指导
✅ 查看指导

本讲核心口诀（贴课桌上！）：

❶ 拆——强拆弱不拆 · ❷ 守——先电荷后原子 · ❸ 量介争——小量定1、介质定产物、还性强弱排排坐
