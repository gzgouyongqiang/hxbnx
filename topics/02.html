高三化学第一轮复习

第2讲 离子反应 离子方程式

广东卷必考 T14/T15，独占一道选择题
🔥 核心就三件事：
① 拆——强拆弱不拆 ② 守——电荷守恒 + 原子守恒 ③ 量——少量/过量定系数
⚡
前测（5分钟）——暴露薄弱点
1. 下列物质中，属于强电解质的是（　　）
  • A. 冰醋酸
  • B. 氨水
  • C. 硫酸钡
  • D. 二氧化硫
✅ 答案：C
BaSO₄虽难溶，但溶解的部分完全电离 → 强电解质；
冰醋酸是弱酸（弱电解质）；氨水是混合物（非电解质）；SO₂不能电离（非电解质）
2. 下列离子方程式书写正确的是（　　）
  • A. 铁与稀硫酸：2Fe + 6H⁺ = 2Fe³⁺ + 3H₂↑
  • B. 氢氧化钡与稀硫酸：Ba²⁺ + SO₄²⁻ = BaSO₄↓
  • C. 碳酸钙与醋酸：CaCO₃ + 2H⁺ = Ca²⁺ + CO₂↑ + H₂O
  • D. 钠与水反应：2Na + 2H₂O = 2Na⁺ + 2OH⁻ + H₂↑
✅ 答案：D
A错：Fe与稀H₂SO₄反应，产物应为Fe²⁺，而非Fe³⁺
B错：漏写 H⁺ + OH⁻ = H₂O，不完整
C错：醋酸是弱酸，不能拆成H⁺，应写 CH₃COOH
D对：✅ Na为单质不拆，H₂O不拆，产物均正确
⚠️ 答错定位：
错第1题 → 回顾电解质分类；错第2题 → 回顾"拆"的原则
🧱
核心一：拆——记牢这张表

唯一原则：强拆弱不拆

必须拆（离子形式）不能拆（化学式）
强酸：HCl、H₂SO₄、HNO₃、HClO₄、HI、HBr 弱酸：CH₃COOH、HF、H₂CO₃、H₂SO₃、HClO
强碱：NaOH、KOH、Ca(OH)₂（澄清石灰水）、Ba(OH)₂ 弱碱：NH₃·H₂O、Fe(OH)₃、Cu(OH)₂
可溶性盐 难溶盐：BaSO₄、AgCl、CaCO₃
— 单质、气体、氧化物、浓硫酸、非电解质
⚠️ 三个易错点：
澄清石灰水 → 拆 ✅　　石灰乳/石灰浆 → 不拆 ❌
Na₂O₂ → 不拆（过氧化物写化学式）
NH₃·H₂O → 不拆（弱碱）
⚖️
核心二：守 + 真

一、两大守恒——检查顺序：先电荷，后原子

① 电荷守恒：两边总电荷必须相等（最容易快速发现错误）

例（×）：Fe³⁺ + Fe = 2Fe²⁺　　❌ 左边+3，右边+4！

正确：2Fe³⁺ + Fe = 3Fe²⁺ ✅

② 原子守恒：各原子种类和数目相等

二、反应真实性——不能臆造

常见臆造错误正确情况
Fe + 稀HNO₃ → H₂↑Fe + HNO₃ → NO↑，产物为Fe³⁺/Fe²⁺ + NO
Al + 稀HNO₃ → H₂↑稀HNO₃与Al反应不生成H₂
Cu + 稀HCl → H₂↑Cu不与稀盐酸反应
Fe³⁺ + I⁻ → 共存Fe³⁺氧化I⁻：2Fe³⁺ + 2I⁻ = 2Fe²⁺ + I₂
📊
核心三：量 + 介 + 争（高考区分题在此）

一、看用量——少量定1法

以少量反应物的计量数为1，确定其他比例

经典体系少量过量
Ca(HCO₃)₂ + NaOH Ca²⁺ + HCO₃⁻ + OH⁻ = CaCO₃↓ + H₂O Ca²⁺ + 2HCO₃⁻ + 2OH⁻ = CaCO₃↓ + CO₃²⁻ + 2H₂O
FeBr₂ + Cl₂ 2Fe²⁺ + Cl₂ = 2Fe³⁺ + 2Cl⁻ 2Fe²⁺ + 4Br⁻ + 3Cl₂ = 2Fe³⁺ + 2Br₂ + 6Cl⁻

二、看介质——酸碱性决定产物

KMnO₄：酸性 → Mn²⁺；碱性 → MnO₄²⁻；中性 → MnO₂

三、看竞争——还原性顺序定先后

I⁻ > Fe²⁺ > Br⁻（还原性从强到弱）
FeI₂中通Cl₂：先氧化I⁻
FeBr₂中通Cl₂：先氧化Fe²⁺
🗣️ 口诀：碘铁溴，还原性从强到弱排排坐
🔬
真题解剖：2022广东卷 T14

第一步：把每个选项的"拆"扫一遍

选项拆分判断
Cl₂ + NaOH → NaCl + NaClO + H₂O Cl₂不拆，NaOH拆 ✅
硅酸钠 + 盐酸 → 硅酸 + H₂O Na₂SiO₃可溶拆为SiO₃²⁻，HCl拆为H⁺ ✅
Na₂O₂ + H₂O → NaOH + O₂ Na₂O₂是过氧化物，不能拆为O₂²⁻ ❌
Fe + CuSO₄ → FeSO₄ + Cu Fe不拆，CuSO₄可溶拆 ✅

第二步：重点防C项陷阱

❌ 错误写法：2O₂²⁻ + 2H₂O = 4OH⁻ + O₂↑

✅ 正确写法：2Na₂O₂ + 2H₂O = 4Na⁺ + 4OH⁻ + O₂↑

💡 Na₂O₂不拆，看到过氧化物先圈出来！

✏️
课堂精练（10分钟）
1. 下列离子方程式书写正确的是（　　）
  • A. 过氧化钠溶于水：Na₂O₂ + H₂O = 2Na⁺ + 2OH⁻ + O₂↑
  • B. 食醋除水垢（CaCO₃）：CaCO₃ + 2H⁺ = Ca²⁺ + CO₂↑ + H₂O
  • C. 硫酸铝溶液中滴加足量氨水：Al³⁺ + 3NH₃·H₂O = Al(OH)₃↓ + 3NH₄⁺
  • D. Fe₂O₃溶于氢碘酸：Fe₂O₃ + 6H⁺ = 2Fe³⁺ + 3H₂O
✅ 答案：C
A错：Na₂O₂不拆且原子不守恒，系数有误（正确：2Na₂O₂ + 2H₂O = 4Na⁺ + 4OH⁻ + O₂↑）
B错：醋酸是弱酸，不能拆为H⁺，应写 CH₃COOH
C对：✅ NH₃·H₂O弱碱不拆，Al³⁺与氨水反应正确
D错：Fe₂O₃溶于氢碘酸，Fe³⁺会被I⁻还原为Fe²⁺，反应不真实（HI是还原性酸）
2. 向FeBr₂溶液中通入少量Cl₂，离子方程式正确的是（　　）
  • A. 2Fe²⁺ + Cl₂ = 2Fe³⁺ + 2Cl⁻
  • B. 2Br⁻ + Cl₂ = Br₂ + 2Cl⁻
  • C. 2Fe²⁺ + 4Br⁻ + 3Cl₂ = 2Fe³⁺ + 2Br₂ + 6Cl⁻
  • D. Fe²⁺ + Br⁻ + Cl₂ = Fe³⁺ + Br₂ + Cl⁻
✅ 答案：A
关键：还原性 I⁻ > Fe²⁺ > Br⁻，少量Cl₂优先氧化Fe²⁺
B错：优先顺序错误，少量Cl₂应先氧化Fe²⁺，而非Br⁻
C错：这是足量Cl₂的方程式，题目说少量
D错：系数未配平，不满足电荷守恒
3. 下列离子方程式能正确表示对应反应的是（　　）
  • A. 向NaClO溶液中通入少量SO₂：ClO⁻ + SO₂ + H₂O = Cl⁻ + SO₄²⁻ + 2H⁺
  • B. 向Ca(HCO₃)₂溶液中加入过量NaOH：Ca²⁺ + HCO₃⁻ + OH⁻ = CaCO₃↓ + H₂O
  • C. 泡沫灭火器反应：Al³⁺ + 3HCO₃⁻ = Al(OH)₃↓ + 3CO₂↑
  • D. 向AgCl悬浊液中滴加Na₂S：2AgCl + S²⁻ = Ag₂S + 2Cl⁻
✅ 答案：C（也接受D）
A错：NaClO溶液呈碱性，产物不能有大量H⁺
B错：NaOH过量时，HCO₃⁻完全反应，正确式为：Ca²⁺ + 2HCO₃⁻ + 2OH⁻ = CaCO₃↓ + CO₃²⁻ + 2H₂O
C对：✅ Al³⁺水解程度不及HCO₃⁻水解，两者反应双水解，方程式正确
D：沉淀转化，AgCl(s)→Ag₂S(s)，写法基本正确，高考一般也给分
📦
本讲小结
❶ 拆：强酸强碱可溶盐拆，弱酸弱碱难溶氧化物不拆
❷ 守：先看电荷守恒（最快），再看原子守恒
❸ 量介争：少量定1法，介质定产物，还原性强弱定顺序
🧠 六看判断流程（贴课桌上！）

① 真实性？ → 有无这个反应？
② 拆分？ → 该拆的拆了没？不该拆的拆了没？
③ 电荷守恒？ → 总电荷相等？
④ 原子守恒？ → 各原子数相等？
⑤ 用量？ → 少量/过量？系数对不对？
⑥ 介质？ → 酸性/碱性？产物受影响吗？
