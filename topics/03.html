考点1

氧化还原反应的判断——标化合价，看升降

🔴 必考
📝 真题举例
2025·山东·第1题
下列在化学史上产生重要影响的成果中，不涉及氧化还原反应的是
A. 侯德榜发明了以NH₃、CO₂和NaCl为原料的联合制碱法
B. 戴维电解盐酸得到H₂和Cl₂，从而提出了酸的含氢学说
C. 拉瓦锡基于金属和O₂的反应提出了燃烧的氧化学说
D. 哈伯发明了以N₂和H₂为原料合成氨的方法
🔍 查看答案
✅ A
逐一标化合价：
A → NaCl+NH₃+CO₂+H₂O→NaHCO₃+NH₄Cl，各元素化合价不变 → 非氧化还原反应 ✅
B → HCl电解：H⁺(+1→0 降)、Cl⁻(-1→0 升) → 氧化还原 ❌
C → 金属(0→+n 升)、O₂(0→-2 降) → 氧化还原 ❌
D → N₂(0→-3 降)、H₂(0→+1 升) → 氧化还原 ❌
2023·广东·第7题（节选）
擦干已洗净的铁锅，以防生锈，防止生成氧化铁；铁丝在O₂中燃烧生成Fe₃O₄。以上两个现象涉及的化学知识关联判断正确的是？
🔍 查看答案
铁锅防锈 → Fe被O₂氧化为Fe₂O₃·xH₂O，涉及氧化还原
铁丝在O₂中燃烧 → 3Fe + 2O₂ =点燃 Fe₃O₄，Fe从0升到+2/+3，O从0降到-2，涉及氧化还原
两者都涉及氧化还原，但本质不同（缓慢氧化 vs 剧烈燃烧），不能简单等同。
⚠️ 陷阱
⚠️ 查看陷阱

① "有O₂参与的反应一定是氧化还原反应"——❌ 2Na₂O + O₂ = 2Na₂O₂，Na₂O₂中O为-1价，但该反应为化合反应且化合价有变化，所以是氧化还原。但反过来，CaO + H₂O = Ca(OH)₂无化合价变化，虽含O元素却不是氧化还原！判断依据只有一个：有无化合价变化

② "分解反应/化合反应一定是氧化还原反应"——❌ 2H₂O =通电 2H₂↑ + O₂↑ 是氧化还原（化合价变了），但 CaCO₃ =高温 CaO + CO₂↑ 不是氧化还原（化合价没变）！四大基本反应类型与氧化还原无必然对应关系

③ "浓硫酸的脱水性 = 氧化性"——❌ 蔗糖遇浓硫酸变黑：先脱水（C₁₂H₂₂O₁₁ → 12C + 11H₂O，无化合价变化），再氧化（C + 2H₂SO₄(浓) = CO₂↑ + 2SO₂↑ + 2H₂O，有化合价变化）。脱水性和氧化性是两步不同过程！

✅ 避坑指导
✅ 查看指导

判断氧化还原反应唯一标准：有无化合价变化

  • 第一步：标出反应物和生成物中各元素的化合价
  • 第二步：对比同种元素反应前后化合价是否变化
  • 第三步：有变化 → 氧化还原反应；无变化 → 非氧化还原反应

常见非氧化还原反应：复分解反应（酸碱盐之间的反应）、部分分解反应（CaCO₃分解）、部分化合反应（CaO+H₂O）

常见必为氧化还原的反应：燃烧、金属与酸反应、电解、歧化反应、归中反应

考点2

氧化剂、还原剂的判断——"升失氧还，降得还氧"

🔴 核心考点
📝 真题举例
2025·广东·第11题（节选）
利用如图装置进行实验：打开K₁、K₂，一定时间后，a中溶液变蓝；关闭K₁，打开K₃，点燃酒精灯加热数分钟后，滴入无水乙醇。下列说法错误的是
A. a中现象体现了I⁻的还原性
B. b中H₂O₂既作氧化剂也作还原剂
C. 乙醇滴加过程中，c中的铜丝由黑变红，说明乙醇被氧化
D. d中有银镜反应发生，说明c中产物有乙酸
🔍 查看答案
✅ D（选"错误"的选项）
A → a中I⁻被氧化为I₂，使淀粉变蓝，体现I⁻的还原性 ✅正确（不选）
B → b中H₂O₂分解：2H₂O₂ = 2H₂O + O₂↑，O从-1变为-2（被还原，作氧化剂）和0（被氧化，作还原剂），既作氧化剂也作还原剂 ✅正确（不选）
C → 铜丝由黑(CuO)变红(Cu)，CuO中Cu从+2→0被还原，乙醇被氧化为乙醛 ✅正确（不选）
D → 银镜反应检验的是醛基，乙醛可发生银镜反应。但c中CuO催化乙醇氧化可生成乙醛，乙醛可发生银镜反应。实际上d中能发生银镜反应，说明c中产物有乙醛或进一步氧化为乙酸后再还原……该选项需要结合实验细节判断，真题答案为D错误。
2024·广东·第13题（节选）
由氨制硝酸过程中，NH₃做还原剂，体现了NH₃的还原性——该陈述是否正确？
🔍 查看答案
✅ 正确 —— 4NH₃ + 5O₂ =催化 4NO + 6H₂O，NH₃中N从-3→+2升高，失电子被氧化，作还原剂，体现还原性。
⚠️ 陷阱
⚠️ 查看陷阱

① "氧化剂得电子 → 化合价降低 → 发生氧化反应"——❌ 氧化剂得电子，化合价降低，发生的是还原反应！口诀：氧化剂被还原，还原剂被氧化

② "H₂O₂一定既作氧化剂又作还原剂"——❌ 看反应！H₂O₂与KMnO₄反应时只作还原剂（O从-1→0升），与KI反应时只作氧化剂（O从-1→-2降）。只有自身分解时才既作氧化剂又作还原剂！

③ "还原性强的物质氧化性一定弱"——❌ 物质可以同时具有氧化性和还原性（如H₂O₂、Fe²⁺、SO₂），强弱取决于具体反应中的角色

④ "含氧酸都有氧化性"——❌ H₃PO₄、H₂CO₃等含氧酸无强氧化性（P、C已是最高价态，但不能氧化常见还原剂）

✅ 避坑指导
✅ 查看指导

口诀表——一张表搞定所有判断：

变化电子被……作……产物叫……
化合价升高失电子被氧化还原剂氧化产物
化合价降低得电子被还原氧化剂还原产物

速记口诀：升失氧还（升→失电子→被氧化→还原剂），降得还氧（降→得电子→被还原→氧化剂）

强弱规律：氧化性：氧化剂 > 氧化产物；还原性：还原剂 > 还原产物

H₂O₂的三种角色：

  • 遇强氧化剂（KMnO₄）→ 作还原剂
  • 遇强还原剂（KI、Fe²⁺）→ 作氧化剂
  • 自身分解（2H₂O₂→2H₂O+O₂↑）→ 既作氧化剂又作还原剂

考点3

电子转移的表示——单线桥与双线桥

🟡 常考
📝 真题举例
经典真题 · 电子转移表示
对于反应 MnO₂ + 4HCl(浓) =△ MnCl₂ + Cl₂↑ + 2H₂O
用双线桥和单线桥分别表示电子转移方向和数目。
🔍 查看答案
双线桥：
Mn：MnO₂中Mn(+4) → MnCl₂中Mn(+2)，得2e⁻，被还原
Cl：HCl中Cl(-1) → Cl₂中Cl(0)，失e⁻×2=2e⁻（仅被氧化的Cl），被氧化
转移电子总数：2e⁻

单线桥：
箭头从失电子元素(Cl⁻)指向得电子元素(Mn)：
Cl⁻ → Mn，标注转移电子数 2e⁻

⚠️ 注意：4个HCl中只有2个Cl被氧化（另外2个Cl⁻化合价不变），不能写4e⁻！
2024·广东·非选择题（节选）
工业上，烟气中的SO₂可在通空气条件下用石灰石的浆液吸收，生成石膏。该过程中，______（填元素符号）被氧化。
🔍 查看答案
✅ S —— SO₂→CaSO₄，S元素的化合价由+4价上升到+6价，失去电子，被氧化。
⚠️ 陷阱
⚠️ 查看陷阱

① "单线桥上写'得/失电子'"——❌ 单线桥只标电子转移总数（如 2e⁻），不写得/失！只有双线桥才写得/失

② "4HCl被氧化，转移4e⁻"——❌ 4HCl中只有2个HCl作还原剂（Cl从-1→0），另外2个Cl⁻化合价不变。转移2e⁻！

③ "歧化反应中电子转移总数=升高的电子数+降低的电子数"——❌ 歧化反应中同种元素既升又降，电子转移总数=升高数=降低数（它们相等！），不是相加！

✅ 避坑指导
✅ 查看指导

两种线桥对比：

单线桥双线桥
箭头从失电子元素→得电子元素反应物→生成物（各画一条）
标注只标转移电子总数（如 2e⁻）写得/失 ne⁻
适用快速判断电子转移方向详细展示每个元素的化合价变化

电子转移计算三步法：

  • ❶ 找出化合价变化的元素，算出每个原子变化量
  • ❷ 乘以该元素的原子个数
  • ❸ 确保得电子总数 = 失电子总数（守恒验证）

考点4

歧化反应与归中反应

🔴 区分题
📝 真题举例
经典真题 · 歧化反应判断
关于反应 Cl₂ + 2NaOH = NaCl + NaClO + H₂O，下列说法正确的是
A. Cl₂只作氧化剂
B. Cl₂只作还原剂
C. Cl₂既作氧化剂又作还原剂
D. NaCl是氧化产物
🔍 查看答案
✅ C
Cl₂中Cl为0价 → NaCl中Cl为-1价（降低，被还原，作氧化剂）
Cl₂中Cl为0价 → NaClO中Cl为+1价（升高，被氧化，作还原剂）
同种元素既升又降 = 歧化反应 → 既作氧化剂又作还原剂 ✅
D → NaCl是Cl降低的产物 → 还原产物 ❌；NaClO才是氧化产物
经典真题 · 归中反应计算
关于反应 8NH₃ + 6NO₂ = 7N₂ + 12H₂O，下列说法正确的是
A. NH₃中H元素被氧化
B. NO₂在反应过程中失去电子
C. 还原剂与氧化剂的物质的量之比为3∶4
D. 氧化产物与还原产物的质量之比为4∶3
🔍 查看答案
✅ D
标化合价：NH₃中N(-3)，NO₂中N(+4)，N₂中N(0)

归中反应分析：
NH₃中N(-3) → N₂(0)：升3，失电子，被氧化 → 还原剂 → 氧化产物
NO₂中N(+4) → N₂(0)：降4，得电子，被还原 → 氧化剂 → 还原产物

A → H元素化合价不变（+1→+1）❌
B → NO₂中N从+4→0，得电子，被还原 ❌
C → 8个NH₃全部被氧化（8mol还原剂），6个NO₂全部被还原（6mol氧化剂），比 = 8:6 = 4:3 ❌
D → 氧化产物（来自NH₃的N₂，4mol）: 还原产物（来自NO₂的N₂，3mol）= 4:3 ✅
验证：8mol×3 = 24e⁻失 = 6mol×4 = 24e⁻得 ✓
⚠️ 陷阱
⚠️ 查看陷阱

① "歧化反应中氧化产物和还原产物一定是不同物质"——❌ 看反应！Cl₂+NaOH产物为NaCl和NaClO（不同物质），但2H₂O₂=2H₂O+O₂↑歧化产物也是不同物质。关键看化合价变化！

② "归中反应中还原剂:氧化剂 = 物质总量比"——❌ 归中反应中，所有反应物都参与了电子转移！8NH₃全部被氧化=全部是还原剂，6NO₂全部被还原=全部是氧化剂，不是"部分参与"

③ "归中反应的产物N₂中，氧化产物和还原产物是不同物质"——❌ 归中反应产物是同种物质N₂，但其中N原子来源不同！来自NH₃的N₂是氧化产物，来自NO₂的N₂是还原产物

✅ 避坑指导
✅ 查看指导

歧化 vs 归中 一图搞懂：

歧化反应归中反应
特征同种元素从同一价态→一升一降同种元素从不同价态→中间价态
起点一种反应物（Cl₂、H₂O₂等）两种反应物（NH₃+NO₂等）
口诀中间价→两头（0→+1和-1）两头→中间（-3和+4→0）
常见例子Cl₂+NaOH、3Cl₂+6KOH(热)NH₃+NOx、H₂S+SO₂

归中反应计算关键：所有反应物都完全参与电子转移！不是"部分参与"！

考点5

氧化性/还原性强弱比较

🔴 核心考点
📝 真题举例
2023·广东·第12题（节选）
电解饱和食盐水实验中，b处出现蓝色，发生Cl₂ + 2KI = I₂ + 2KCl。这说明还原性：______
🔍 查看答案
✅ 还原性：I⁻ > Cl⁻
反应中Cl₂是氧化剂，I⁻是还原剂；I₂是氧化产物，Cl⁻是还原产物
根据规律：还原性：还原剂(I⁻) > 还原产物(Cl⁻)
2025·山东·第9题（节选）
用肼(N₂H₄)的水溶液处理核冷却系统内壁上的铁氧化物时，通常加入少量Fe³⁺。下列关于还原性强弱判断正确的是？
🔍 查看答案
关键判断：Fe³⁺催化N₂H₄还原Fe₂O₃ → 还原性：N₂H₄ > Fe³⁺对应的还原产物
由反应推断还原性顺序，再结合金属活动性顺序综合判断。
氧化性/还原性强弱比较的四种方法：
① 化学反应：氧化剂 > 氧化产物，还原剂 > 还原产物
② 金属活动性：K Ca Na Mg Al Zn Fe Sn Pb (H) Cu Hg Ag Pt Au（还原性从左→右递减）
③ 反应条件：条件越容易（温度低、不需催化），氧化（还原）性越强
④ 电化学：原电池负极→还原性强；电解池阳极先放电→还原性强
⚠️ 陷阱
⚠️ 查看陷阱

① "氧化性强的物质对应的离子还原性也强"——❌ 氧化性：F₂ > Cl₂ > Br₂ > I₂，但还原性：F⁻ < Cl⁻ < Br⁻ < I⁻。氧化性越强的单质，其阴离子还原性越弱！

② "反应越剧烈，氧化性（还原性）越强"——❌ 反应剧烈程度还受浓度、温度、接触面积等影响。比较强弱必须控制变量，看"相同条件下谁先反应"

③ "最高价态一定有最强氧化性"——❌ HNO₃(浓)氧化性远强于H₃PO₄，虽然N(+5)和P(+5)都是最高价态。氧化性不仅看价态，还看物质本身性质

✅ 避坑指导
✅ 查看指导

四种方法优先级：化学反应 > 金属活动性 > 反应条件 > 电化学

必背强弱顺序：

  • 氧化性：F₂ > Cl₂ > Br₂ > I₂ > S；KMnO₄(H⁺) > Cl₂ > Br₂ > I₂ > S
  • 还原性：I⁻ > Fe²⁺ > Br⁻ > Cl⁻ > F⁻
  • 还原性（金属）：K > Ca > Na > Mg > Al > Zn > Fe > Sn > Pb > (H) > Cu > Hg > Ag

口诀："谁强谁先反应，产物一定弱于剂"

特别注意：I⁻ > Fe²⁺ > Br⁻ 这个顺序在"竞争反应"中考查频率极高（如Cl₂通入FeI₂/FeBr₂）！

考点6

氧化还原反应的综合判断——以考定教实战

🔴 每年必考
📝 真题举例
2024·广东·第14题
部分含Na或Fe或Cu物质的分类与相应化合价关系如图。下列推断合理的是
A. 若a在沸水中可生成e，则a→f的反应一定是化合反应
B. 在g→f→e→d转化过程中，一定存在物质颜色的变化
C. 加热c的饱和溶液，一定会形成能产生丁达尔效应的红棕色分散系
D. 若b和d均能与同一物质反应生成c，则组成a的元素一定位于周期表p区
🔍 查看答案
✅ B
该图考查含Fe/Na/Cu物质的化合价与分类。关键是理解化合价与物质类别的对应关系。

A → a可为Mg(在沸水中生成Mg(OH)₂即e)，a→f可以是Mg+2HCl=MgCl₂+H₂↑(置换反应)，不一定是化合反应 ❌

部分含Na或Cu物质的分类与相应化合价关系如图所示。下列推断不合理的是
A. 可存在c→d→e的转化
B. 能与H₂O反应生成c的物质只有b
C. 新制的d可用于检验葡萄糖中的醛基
D. 若b能与CO₂反应生成c，则b中含共价键
🔍 查看答案
✅ B（选"不合理"的）
a、b、c对应物质：Na、Na₂O(或Na₂O₂)、NaOH
或a、b、e、d对应：Cu、Cu₂O、CuO、Cu(OH)₂

A → NaOH+CuSO₄→Cu(OH)₂，Cu(OH)₂受热→CuO，存在c→d→e ✅合理
B → Na和Na₂O都能与H₂O反应生成NaOH，不只是b ❌不合理 → 选B！
C → 新制Cu(OH)₂检验醛基是经典实验 ✅合理
D → b=Na₂O₂时含O-O共价键 ✅合理
