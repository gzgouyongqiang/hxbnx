氧化还原反应的概念和规律
考情提示：全国卷/广东卷必考选择题，常结合 STSE、化学史考查
核心三件事：① 四步法判"剂"和"产物" ② 电子转移表示 ③ 氧化性/还原性强弱比较
🎯 前测（5分钟）
1. （2025·山东卷）下列在化学史上产生重要影响的成果中，不涉及氧化还原反应的是（　　）
A. 侯德榜联合制碱法（NH₃、CO₂和NaCl）
B. 戴维电解盐酸得H₂和Cl₂
C. 拉瓦锡基于金属和O₂提出燃烧氧化学说
D. 哈伯合成氨法（N₂和H₂）
答案：A
侯德榜联合制碱法中各元素化合价均未发生变化，属于非氧化还原反应。
B选项H⁺→H₂（降）、Cl⁻→Cl₂（升）；C选项金属化合价升高、O₂化合价降低；D选项N₂→NH₃（降）、H₂→NH₃（升）。
2. （2024·山东卷）古人所用文房四宝的制作过程中，发生氧化还原反应的是（　　）
A. 竹管、动物尾毫 → 湖笔
B. 松木 → 油烟 → 徽墨
C. 楮树皮 → 纸浆纤维 → 宣纸
D. 端石 → 端砚
答案：B
松木燃烧（不完全燃烧）生成炭黑（油烟），碳元素化合价变化，涉及氧化还原反应。
A、C、D 均为物理加工过程，无化合价变化。
📘 核心一：概念判断四步法
标化合价 → 看变化 → 下结论 → 比强弱
变化方向 电子 被…… 作…… 产物
升 化合价升高 失电子 被氧化 作还原剂 得氧化产物
降 化合价降低 得电子 被还原 作氧化剂 得还原产物
口诀：升失氧还，降得还氧
升 → 失电子 → 被氧化 → 还原剂 → 氧化产物
降 → 得电子 → 被还原 → 氧化剂 → 还原产物
强弱规律：
氧化性：氧化剂 > 氧化产物
还原性：还原剂 > 还原产物
歧化反应：同种元素既升又降，既是氧化剂又是还原剂
如：Cl₂ + NaOH（Cl 既升到 +5 又降到 -1）
📗 核心二：电子转移表示 + 强弱比较
表示方法 画法 标注
单线桥 箭头从失电子元素→得电子元素 标转移电子总数（ne⁻），不写得/失
双线桥 反应物→生成物，分别标 写得/失 n×e⁻

氧化性/还原性强弱比较的四种方法：

方法 依据 示例
① 化学反应 氧化剂 > 氧化产物；还原剂 > 还原产物 由反应直接推断
② 金属活动性 K Ca Na Mg Al Zn Fe Sn Pb (H) Cu Hg Ag Pt Au 还原性从左到右递减
③ 反应条件 条件越容易（温度低、不需催化），氧化（还原）性越强 对比反应条件
④ 电化学 原电池负极→还原性强；电解池阳极→还原性强 电化学装置推断
🔬 真题讲解：一道题讲透所有概念
（2021·浙江卷）关于反应 8NH₃ + 6NO₂ = 7N₂ + 12H₂O，下列说法正确的是（　　）
A. NH₃中H元素被氧化
B. NO₂在反应过程中失去电子
C. 还原剂与氧化剂的物质的量之比为3∶4
D. 氧化产物与还原产物的质量之比为4∶3
答案：D

判断路径：

第一步：标化合价
NH₃：N(-3)　NO₂：N(+4)　N₂：N(0)

第二步：画双线桥
NH₃(-3) → N₂(0)：升3，失电子，被氧化 → 还原剂 → 氧化产物
NO₂(+4) → N₂(0)：降4，得电子，被还原 → 氧化剂 → 还原产物

第三步：逐项判断
A → H元素化合价不变（+1→+1）❌
B → NO₂中N从+4→0，得电子，被还原 ❌
C → 归中反应：N(-3)→N(0)升3×8=24，N(+4)→N(0)降4×6=24
　　8个NH₃全部被氧化，还原剂:氧化剂 = 8:6 = 4:3 ❌
D → 氧化产物（来自NH₃的N₂）: 还原产物（来自NO₂的N₂）= 4:3 ✅
关键：归中反应中，N元素从-3和+4归中到0价，计算时要分清N原子的来源！
8NH₃全部被氧化 → 升3×8=24e⁻；6NO₂全部被还原 → 降4×6=24e⁻；得失电子守恒 ✓
✏️ 课堂精练（10分钟）
1. （2023·全国乙卷）下列应用中涉及到氧化还原反应的是（　　）
A. 使用明矾对水进行净化
B. 雪天道路上撒盐融雪
C. 暖贴中的铁粉遇空气放热
D. 荧光指示牌被照发光
答案：C
A → Al³⁺水解生成Al(OH)₃胶体吸附杂质，无化合价变化 ❌
B → CaCl₂降低冰点，物理过程 ❌
C → Fe + O₂ + H₂O → Fe₂O₃·xH₂O（铁生锈），Fe被氧化，O₂被还原 ✅
D → 荧光物质吸收光能再释放，无化合价变化 ❌
2. （2022·天津卷）燃油汽车发动机及催化转化器中发生的部分反应如下：
反应(Ⅰ) N₂ + O₂ → 2NO　反应(Ⅱ) 2CO + 2NO → 2CO₂ + N₂
下列判断错误的是（　　）
A. 甲是空气中体积分数最大的成分
B. 乙是引起温室效应的气体之一
C. 反应(Ⅰ)在常温下容易发生
D. 反应(Ⅱ)中NO是氧化剂
答案：C
反应(Ⅰ) N₂ + O₂ → 2NO 需要在高温（放电或高温）条件下才能发生，常温下极难进行。闪电时空气中的N₂和O₂才能反应生成NO。
D选项中NO中N从+2→0，得电子被还原，作氧化剂 ✅。
3. （2021·江苏卷）黑火药爆炸反应：2KNO₃ + S + 3C = K₂S + N₂↑ + 3CO₂↑
下列说法正确的是（　　）
A. 黑火药中含有两种单质
B. 爆炸时吸收热量
C. 反应中S作还原剂
D. 反应为置换反应
答案：C
标化合价：KNO₃中N(+5)→N₂(0)降，作氧化剂
S(0)→K₂S中S(-2)降，S被还原 → S也是氧化剂
C(0)→CO₂中C(+4)升，C被氧化 → C是还原剂

A → KNO₃、S、C共三种单质（KNO₃不是单质），但原料中S和C两种是单质……实际上"黑火药"成分是KNO₃+炭粉+硫粉，确实含两种单质（S和C）
但A选项说"黑火药中含有两种单质"，KNO₃不是单质，所以黑火药含S和C两种单质，A也对？

再核实：黑火药 = KNO₃（盐）+ C（单质）+ S（单质），两种单质 → A对
C选项S作还原剂：S(0)→S(-2)被还原，应作氧化剂 → C说"作还原剂"是错的
最终答案：A（S不是还原剂，C的说法错误；D不是置换反应）
📝 本讲小结
  • 1️⃣ 四步法是解题根本：标化合价 → 看升降 → 判剂和产物 → 比强弱
  • 2️⃣ 电子转移会画两种桥：单线桥（标总数）、双线桥（标得失）
  • 3️⃣ 强弱比较四个方法：化学反应 > 金属活动性 > 反应条件 > 电化学
