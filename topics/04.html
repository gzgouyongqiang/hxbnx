氧化还原反应方程式的配平与书写
🔥 工艺流程大题必考，选择题也常结合NA考查电子转移。
核心就三件事：① 配平五步法 ② 介质配平 ③ 电子转移计算
🎯 前测（5分钟）
1. （2022·浙江卷）关于反应 4CO₂ + SiH₄ = 4CO + 2H₂O + SiO₂，下列说法正确的是（　　）
A. CO是氧化产物
B. SiH₄发生还原反应
C. 氧化剂与还原剂的物质的量之比为1∶4
D. 生成1 mol SiO₂时，转移8 mol电子
✅ 答案：D
CO₂→CO，C从+4→+2降2，被还原，CO₂是氧化剂，4个CO₂共降8；
SiH₄→SiO₂，Si从-4→+4升8，是还原剂；
氧化剂∶还原剂 = 4∶1
2. 配平下列离子方程式（写系数即可）：
____ Cr₂O₇²⁻ + ____ Fe²⁺ + ____ H⁺ → ____ Cr³⁺ + ____ Fe³⁺ + ____ H₂O
✅ 答案：1, 6, 14, 2, 6, 7
📐 核心一：配平五步法
① 标
→
② 等
→
③ 定
→
④ 配
→
⑤ 查
示例：配平 ____Cu + ____HNO₃(稀) = ____Cu(NO₃)₂ + ____NO↑ + ____H₂O
①标 标化合价：Cu(0→+2)，HNO₃中N(+5→+2)
②等 找最小公倍数：Cu升2，N降3，最小公倍数6
③定 定氧化剂/还原剂系数：Cu×3，HNO₃(变价部分)×2
④配 原子守恒配其他
3Cu + 8HNO₃ = 3Cu(NO₃)₂ + 2NO↑ + 4H₂O
⑤查 检查守恒：左右原子数、电荷均相等 ✅
⚠️ 抓大放小：先配变价元素，再配非变价元素，最后配H和O
⚗️ 核心二：介质配平 + 多元素变价

一、介质配平（离子方程式）

介质 左边多O 左边少O
酸性 + 2H⁺ → 生成 H₂O + H₂O → 生成 2H⁺
碱性 + H₂O → 生成 2OH⁻ + 2OH⁻ → 生成 H₂O
经典示例：酸性KMnO₄氧化Fe²⁺
MnO₄⁻ + Fe²⁺ + H⁺ → Mn²⁺ + Fe³⁺ + H₂O

Mn(+7→+2)降5，Fe(+2→+3)升1 → 最小公倍数5
→ MnO₄⁻:1, Fe²⁺:5, 再配H⁺和H₂O
MnO₄⁻ + 5Fe²⁺ + 8H⁺ = Mn²⁺ + 5Fe³⁺ + 4H₂O

二、多元素变价——算总升降值

示例：FeS₂ + O₂ = Fe₂O₃ + SO₂
① 标价：Fe(+2→+3)升1，S(-1→+4)升5×2=10，一个FeS₂共升11
② O₂(0→-2)降2×2=4（每个O₂降4）
③ 最小公倍数44，FeS₂:4，O₂:11
4FeS₂ + 11O₂ = 2Fe₂O₃ + 8SO₂
🎯 口诀：多个变价一起算，别漏原子乘个数
🔬 真题讲解：配平 + 电子转移计算
（2024·湖北卷·T7改编）二氧化氯(ClO₂)可用于自来水消毒。实验室用草酸(H₂C₂O₄)和KClO₃制取ClO₂的反应为：
H₂C₂O₄ + KClO₃ ——→ CO₂↑ + ClO₂↑
A. 该反应中KClO₃是还原剂
B. 每生成1 mol ClO₂，转移1 mol电子
C. 氧化产物与还原产物的物质的量之比为1∶2
D. 该反应在强碱性条件下进行
🔬 讲判断路径

第一步：配平（五步法）
①标：H₂C₂O₄中C(+3→+4)升1×2=2，KClO₃中Cl(+5→+4)降1
②等：2和1的最小公倍数=2
③定：H₂C₂O₄系数1，KClO₃系数2
④配：H₂C₂O₄ + 2KClO₃ = 2CO₂↑ + 2ClO₂↑ + 2KOH

第二步：逐项判断
A→KClO₃中Cl降价，是氧化剂 ❌
B→每个Cl从+5→+4降1，2个Cl转移2e⁻，生成2个ClO₂，每生成1 mol ClO₂转移1 mol e⁻ ✅
C→氧化产物为CO₂，还原产物为ClO₂，比例为2:2=1:1 ❌
D→产物有KOH，强碱性有利于反应进行，但草酸在强碱性条件下会成盐，真实条件为酸性 ❌

✅ 答案：B
✏️ 课堂精练（10分钟）
1. （2022·浙江卷）关于反应 4CO₂ + SiH₄ = 4CO + 2H₂O + SiO₂，若生成1 mol SiO₂，下列说法错误的是（　　）
A. CO₂是氧化剂
B. SiH₄中Si的化合价为-4
C. 转移电子数为8 mol
D. CO是还原产物
✅ 答案：D
A→CO₂中C(+4→+2)降2，是氧化剂 ✅
B→SiH₄中H为+1，4个H共+4，Si=-4 ✅
C→Si(-4→+4)升8，生成1 mol SiO₂转移8 mol e⁻ ✅
D→CO来自CO₂中C被还原，是还原产物 ❌
2. （2024·浙江1月卷）汽车尾气中的NO和CO在催化剂作用下发生反应：2NO + 2CO = N₂ + 2CO₂。下列说法不正确的是（　　）
A. 生成1 mol CO₂转移电子的数目为2 mol
B. 催化剂降低NO与CO反应的活化能
C. NO是氧化剂，CO是还原剂
D. N₂既是氧化产物又是还原产物
✅ 答案：D
A→C(+2→+4)升2，每个CO₂对应2e⁻，每生成1 mol CO₂转移2 mol e⁻ ✅
B→催化剂降低活化能 ✅
C→NO中N(+2→0)降低，是氧化剂；CO中C(+2→+4)升高，是还原剂 ✅
D→N₂中N来自NO（被还原），所以N₂是还原产物，不是氧化产物 ❌
3. 制备纳米Si的流程：SiO₂ + Mg → Si + MgO（未配平）。下列说法错误的是（　　）
A. SiO₂可与NaOH溶液反应
B. 盐酸在该工艺中体现了还原性
C. 高纯Si可用于制造硅太阳能电池
D. 该反应中氧化剂与还原剂的物质的量之比为1∶2
✅ 答案：B
配平：SiO₂ + 2Mg = Si + 2MgO
A→SiO₂ + 2NaOH = Na₂SiO₃ + H₂O ✅
B→HCl与MgO反应→MgCl₂ + H₂O，体现酸性，非还原性 ❌
C→高纯Si是半导体→太阳能电池 ✅
D→SiO₂降4是氧化剂，Mg升2×2=4是还原剂，氧化剂∶还原剂=1∶2 ✅
📝 本讲小结
  • 1️⃣ 配平五步法：标 → 等 → 定 → 配 → 查
  • 2️⃣ 介质口诀：酸性多O加H⁺变H₂O，碱性多O加水变OH⁻
  • 3️⃣ 电子转移计算：找准变价元素，一个原子一个原子算，乘对系数
