考点1

氧化还原反应配平的核心原理

必考
📘 核心知识
⚡ 配平三大守恒
  • 电子守恒：氧化剂得到的电子数 = 还原剂失去的电子数（配平的核心依据）
  • 质量守恒：反应前后各元素原子数相等
  • 电荷守恒：离子方程式两边电荷代数和相等
📋 电子守恒配平法（五步法）
1
标化合价：找出化合价发生变化的元素，标注反应前后化合价
2
列升降：计算每摩尔该物质化合价升高/降低的总数
3
求最简比：用最小公倍数使化合价升高总数 = 降低总数（电子守恒）
4
配其他：根据原子守恒和电荷守恒，配平其他物质的系数
5
验证：检验两边原子数和电荷数是否守恒
K2Cr2O7 + HCl → KCl + CrCl3 + Cl2 + H2O
Cr：+6 → +3（降低3，得3e⁻，共2个Cr，得6e⁻）
Cl：-1 → 0（升高1，失1e⁻，需6个Cl，失6e⁻）
∴ 配平系数：K₂Cr₂O₇:3Cl₂ = 1:3，然后补充其余系数
✅ 配平结果
K₂Cr₂O₇ + 14HCl → 2KCl + 2CrCl₃ + 3Cl₂↑ + 7H₂O
考点2

氧化剂、还原剂的判断与分析

必考
🎯 真题精析
2025年广东高考 第11题（选择题）
利用如图装置进行实验：打开K₁、K₂，一定时间后，a 中溶液变蓝；关闭K₁，打开K₃，点燃酒精灯加热数分钟后，滴入无水乙醇。下列说法错误的是

A．a 中现象体现了 I⁻ 的还原性

B．b 中 H₂O₂ 既作氧化剂也作还原剂

C．乙醇滴加过程中，c 中的铜丝由黑变红，说明乙醇被氧化

D．d 中有银镜反应发生，说明 c 中产物有乙酸

🔍 查看答案
【答案】D
C项正确：乙醇在铜催化下被氧化为乙醛，Cu作为催化剂（先被氧化为CuO，再被乙醇还原为Cu）。铜丝由黑变红是乙醛生成的证据，而非乙酸。d中银镜反应说明产物是乙醛，而不是乙酸（乙酸不能发生银镜反应），故D错误。
⚠️ 查看陷阱
陷阱：混淆乙醛与乙酸的检验
乙醛（CH₃CHO）含醛基，能发生银镜反应；乙酸（CH₃COOH）含羧基，不能发生银镜反应。看到"银镜反应"就应想到含醛基的物质，而非乙酸。
✅ 查看指导
  • H₂O₂ 在 MnO₂ 催化下分解：2H₂O₂ → 2H₂O + O₂↑，其中O化合价 -1→ -2（还原）和 -1→ 0（氧化），故H₂O₂既作氧化剂又作还原剂，B选项正确。
  • 判断"被氧化"：化合价升高的物质中元素被氧化，该物质作还原剂
  • 判断"被还原"：化合价降低的物质中元素被还原，该物质作氧化剂
🎯 真题精析
2024年广东高考 第17题 第(2)小题
工业上，烟气中的 SO₂ 可在通空气条件下用石灰石的浆液吸收，生成石膏（主要成分 CaSO₄）。该过程中，________（填元素符号）被氧化。
🔍 查看答案
【答案】S
分析：SO₂ 中 S 化合价为 +4，CaSO₄ 中 S 化合价为 +6，化合价升高 → S 被氧化。O₂ 中 O 化合价 0 → -2，降低，O 被还原（或 O₂ 作氧化剂）。
⚠️ 查看陷阱
陷阱：填元素符号 vs 填物质名称
题目要求填"元素符号"，答"S"而非"硫"或"SO₂"。审题时特别注意：被氧化/被还原的是"元素"，作氧化剂/还原剂的是"物质"。
✅ 查看指导
反应方程式：2CaCO₃ + 2SO₂ + O₂ → 2CaSO₄ + 2CO₂
  • SO₂ → CaSO₄：S化合价 +4 → +6，升高2，被氧化，SO₂ 是还原剂
  • O₂ → CaSO₄（或H₂O）：O化合价 0 → -2，降低2，被还原，O₂ 是氧化剂
  • 记忆口诀：升失氧（化合价升高、失去电子、被氧化）；降得还（化合价降低、得到电子、被还原）
🎯 真题精析
2023年广东高考 相关题（氧化性分析）
在如图装置中，I中试管发生反应：Cu + 2H₂SO₄（浓）△→ CuSO₄ + SO₂↑ + 2H₂O。
分析：Ⅰ中氢元素化合价不变，H₂SO₄ 不体现氧化性的理由是什么？
🔍 查看答案
【分析】本题考查"哪种H₂SO₄体现氧化性"：浓H₂SO₄中S为+6价，生成SO₂中S为+4价（化合价降低），浓H₂SO₄作氧化剂，体现氧化性。而H中的H化合价从+1到+1未变，H不体现氧化性。
【注意】题目说的是"Ⅰ试管中氢元素化合价不变"——H₂SO₄ 中的 H 在产物 CuSO₄、H₂O 中化合价始终为+1，确实没有变化。所谓的氧化性，是指 S+6→+4 那部分体现的，而不是 H 的变化。
⚠️ 查看陷阱
陷阱：浓H₂SO₄与稀H₂SO₄氧化性的区分
稀H₂SO₄：H⁺体现氧化性（如 Fe + H₂SO₄稀 → FeSO₄ + H₂↑，H: +1→0）
浓H₂SO₄：S⁺⁶体现氧化性（如 Cu + 2H₂SO₄浓 → CuSO₄ + SO₂↑ + 2H₂O，S: +6→+4）
✅ 查看指导
  • 判断一个物质"体现什么性质"的关键：看该物质中哪个元素化合价发生了变化
  • 浓H₂SO₄与Cu反应：S +6→+4（降低），体现的是H₂SO₄的氧化性；同时也体现酸性（提供H⁺使Cu溶解）
  • 高考频考：区分"氧化性"与"酸性"，不能混淆
考点3

氧化还原方程式书写规范与典型类型

常考
📘 常见配平类型汇总
类型 例子 配平思路
同种元素升降（歧化） Cl₂ + NaOH →
NaCl + NaClO + H₂O		 Cl: 0→-1（得1e⁻），0→+1（失1e⁻），数目相等，直接配平
多个元素价态变化 Fe + 浓H₂SO₄ (热) →
Fe₂(SO₄)₃ + SO₂ + H₂O		 Fe: 0→+3（失3e⁻），S: +6→+4（得2e⁻）
最小公倍数6：2Fe，3S，系数比=2:3
含H⁺或OH⁻的离子方程式 MnO₄⁻ + Fe²⁺ + H⁺ →
Mn²⁺ + Fe³⁺ + H₂O		 先电子守恒→再电荷守恒→最后补H₂O
歧化与归中反应 H₂O₂ → H₂O + O₂（歧化）
NO + NO₂ + H₂O → HNO₂（归中）		 歧化：同一元素价态"一升一降"
归中：两种不同价态同元素"趋向中间价"
🎯 真题精析（KMnO₄配平典型题）
典型考题（基于广东卷命题规律整理）
酸性 KMnO₄ 溶液与 FeSO₄ 溶液反应，写出该反应的离子方程式。
🔍 查看答案
【配平过程】
1. 化合价分析：Mn: +7→+2（降低5），Fe: +2→+3（升高1）
2. 电子守恒：1个Mn得5e⁻，1个Fe失1e⁻，需5个Fe²⁺
3. 初步方案：MnO₄⁻ + 5Fe²⁺ → Mn²⁺ + 5Fe³⁺
4. 补电荷：左边净电荷 = (-1 + 5×2) = +9，右边 = (+2 + 5×3) = +17，差8个正电荷需8H⁺
5. 补H₂O：8H⁺ → 4H₂O

离子方程式：MnO₄⁻ + 5Fe²⁺ + 8H⁺ = Mn²⁺ + 5Fe³⁺ + 4H₂O
⚠️ 查看陷阱
陷阱1：漏写 H⁺ 或 H₂O
含氧酸根参与的氧化还原反应，O元素要靠H₂O来平衡，离子方程式中必须有H⁺（酸性）或OH⁻（碱性）。

陷阱2：忘记检验电荷守恒
写完后必须验证两边电荷代数和是否相等。左边：(-1)+5(+2)+8(+1)=+17；右边：(+2)+5(+3)=+17。✓
✅ 查看指导
  • KMnO₄ 在酸性条件下：MnO₄⁻ → Mn²⁺（得5e⁻）
  • KMnO₄ 在中性条件下：MnO₄⁻ → MnO₂（得3e⁻）
  • KMnO₄ 在碱性条件下：MnO₄⁻ → MnO₄²⁻（得1e⁻）
  • 高考口诀：酸中得5，中性得3，碱性得1——KMnO₄的三种还原产物要熟记
2025年广东高考 工业流程大题（第18题节选）
矿粉酸浸通入 SO₂，浸取液中含有 Fe²⁺、Cu²⁺、Ni²⁺、Al³⁺等，调节 pH=3.0 并通入空气加热，生成 Fe₂O₃↓。写出该步骤的离子方程式：
4Fe²⁺ + O₂ + 4H₂O = 2Fe₂O₃↓ + 8H⁺（已给出，分析配平思路）
🔍 查看答案
【配平分析】
Fe²⁺ → Fe³⁺（在Fe₂O₃中）：化合价 +2→+3，升高1，每个Fe失1e⁻
O₂ → O²⁻：化合价 0→-2，降低2，每个O得2e⁻，1mol O₂得4e⁻
电子守恒：4mol Fe²⁺（失4e⁻）= 1mol O₂（得4e⁻）
补O和H：4Fe²⁺ + O₂ → 2Fe₂O₃，O差：(2×3)-1=5个O，来自4个H₂O，产生8H⁺
最终：4Fe²⁺ + O₂ + 4H₂O = 2Fe₂O₃↓ + 8H⁺
（注意：生成H⁺而非消耗H⁺，因为体系中需要不断调节pH>3防止Fe以外的离子沉淀）
⚠️ 查看陷阱
陷阱：Fe₂O₃中的O来自哪里？
Fe₂O₃ 中有3个O，而 Fe²⁺和O₂本身只提供2个O（1个O₂），还缺1个O → 来自H₂O。
这种"O要靠水补"的思维在无机大题中极其常见，务必熟练。
✅ 查看指导
  • 含氧化合物生成时的配平口诀：先电子守恒定主系数，再看O的来源（加H₂O），最后检验H（加H⁺）
  • 工业流程中"通空气氧化Fe²⁺"是高频考点：O₂ 作氧化剂，Fe²⁺ 作还原剂
  • 生成 H⁺ 说明酸度增大，这就是为什么流程中要"调节pH"——要持续补碱来维持pH
考点4

氧化性还原性强弱的比较

常考
📘 核心规律
⚖️ 比较强弱的依据
  • 位移反应：A + BCl₂ → ACl₂ + B，则 A 的还原性强于 B（活泼金属置换不活泼金属）
  • 氧化还原方向：较强氧化剂 + 较强还原剂 → 较弱氧化剂 + 较弱还原剂
  • 金属活动性顺序：K > Ca > Na > Mg > Al > Zn > Fe > Ni > Sn > Pb > (H) > Cu > Hg > Ag > Pt > Au（还原性从强到弱）
⚡ 常考氧化剂/还原剂的判断
物质常见化合价变化体现性质
KMnO₄、MnO₂Mn: +7→+2；Mn: +4→+2强氧化剂
浓H₂SO₄、稀H₂SO₄S: +6→+4；H: +1→0氧化性（不同机理）
HNO₃（浓稀）N: +5→+4或+2强氧化性
Na₂O₂O: -1→-2（H₂O）和O: -1→0（O₂）既作氧化剂又作还原剂
H₂O₂O: -1→-2（还原剂得）或 O: -1→0（氧化剂失）既可作氧化剂又可作还原剂
SO₂S: +4→+6（还原剂）或 S: +4→0（弱氧化剂）主要作还原剂
Fe²⁺Fe: +2→+3（还原剂）还原剂
🎯 真题精析
2024年广东高考 第13题（陈述Ⅰ和陈述Ⅱ因果判断）
下列陈述Ⅰ与陈述Ⅱ均正确，且具有因果关系的是______

C．陈述Ⅰ：由氨制硝酸过程中，NH₃ 作还原剂，体现了 NH₃ 的还原性
　 陈述Ⅱ：NH₃ 与 HNO₃ 反应时，NH₃ 作碱

（实际答案为B，本题考查C项分析）
🔍 查看答案
C选项错误分析：
陈述Ⅰ：工业制硝酸中，4NH₃ + 5O₂ →(催化剂)→ 4NO + 6H₂O，N: -3→+2，升高5，NH₃ 确实作还原剂，陈述I正确。
陈述Ⅱ：NH₃ + HNO₃ → NH₄NO₃，NH₃作碱，陈述II也正确。
但两者无因果关系：NH₃体现还原性是因为N的化合价能升高；NH₃体现碱性是因为N上有孤对电子能接受H⁺。这是两种不同性质，互不影响。
⚠️ 查看陷阱
陷阱：还原性 ≠ 碱性，两者不能互为因果
NH₃ 既有还原性（N为-3价，能升高）又有碱性（N有孤对电子），但这是两种独立性质：
• 还原性：N化合价升高（N-3 → N+x）
• 碱性：接受H⁺形成NH₄⁺
高考"陈述I和II因果判断"的关键：两个陈述都对，但逻辑上必须是前者能推出后者，才算因果关系。
✅ 查看指导
  • 做"陈述I和II"题的步骤：①先判断两句话各自对不对；②再判断能否用"因为"连接（I是II的原因）
  • 常见有因果关系的类型：电负性强→极性键极性大、粒子半径小→与水结合能力强、键能大→化学性质稳定等
  • 常见无因果关系的干扰：性质A（如还原性）与性质B（如碱性）是独立性质，不能互推
考点5

离子方程式中氧化还原反应的书写

常考
📘 书写要点
📝 书写氧化还原离子方程式的完整步骤
1
拆分强电解质：强酸、强碱、可溶盐写成离子形式；弱酸、弱碱、难溶盐、气体保留化学式
2
电子守恒配平氧化还原：先确定氧化剂和还原剂的系数比
3
电荷守恒补 H⁺/OH⁻：酸性环境加 H⁺，碱性环境加 OH⁻
4
质量守恒补 H₂O：哪边缺O就加H₂O，同时调整H的个数
5
双重验证：原子数守恒 + 电荷守恒，缺一不可
🎯 真题精析
2023年广东高考 第19题节选（配合物氧化还原）
某有机物能与 Co²⁺ 形成橙红色的配离子，该配离子可被 H₂O₂ 氧化成淡蓝色的配离子（Co³⁺）。
完成反应的离子方程式（已知氧化剂为 H₂O₂）：
______Co²⁺ + ______H₂O₂ + ______H⁺ → ______Co³⁺ + ______H₂O
🔍 查看答案
【配平分析】
Co: +2→+3（升高1，失1e⁻）
O₂⁻: -1→-2（降低1，得1e⁻，每个H₂O₂得2e⁻）
电子守恒：2Co²⁺ 失2e⁻ = 1H₂O₂ 得2e⁻
补H₂O：右边有2个H₂O，左边需2H⁺

离子方程式：2Co²⁺ + H₂O₂ + 2H⁺ = 2Co³⁺ + 2H₂O
（如果是配离子参与，将Co²⁺换成对应配离子化学式即可，配平系数不变）
⚠️ 查看陷阱
陷阱：H₂O₂ 作氧化剂时，注意其每mol转移电子数为2
H₂O₂ 中 O 为 -1 价：
• 作氧化剂：O -1 → -2（每mol O₂²⁻ 得 2e⁻）→ 生成 H₂O
• 作还原剂：O -1 → 0（每mol O₂²⁻ 失 2e⁻）→ 生成 O₂
本题H₂O₂作氧化剂，所以产物是H₂O，而非O₂。
✅ 查看指导
  • H₂O₂ 是两用试剂：
    遇强氧化剂（如KMnO₄）→ 作还原剂（O: -1→0，放O₂）
    遇还原剂（如Fe²⁺、Co²⁺）→ 作氧化剂（O: -1→-2，生成H₂O）
  • 配合物题中，Co²⁺ 被氧化为 Co³⁺ 是典型的1e⁻转移，记住系数关系：2Co²⁺ : 1H₂O₂
总结

第4讲 · 高频考点与易错点速查

⚡ 满分攻略
高频考点常见陷阱避坑方法
判断被氧化/被还原元素 问"元素"写"物质"，或搞混氧化/还原方向 升失氧（化合价升→失e⁻→被氧化）；降得还（化合价降→得e⁻→被还原）
配平含氧酸参与的反应 漏写H⁺或H₂O，导致O原子不守恒 先电子守恒，再电荷守恒加H⁺/OH⁻，最后O守恒加H₂O
H₂O₂ 和 Na₂O₂ 双重身份 不明确何时作氧化剂何时作还原剂 看对方：遇氧化性强的物质→作还原剂；遇还原性物质→作氧化剂
KMnO₄ 不同条件下产物 忘记区分酸性/中性/碱性条件下的还原产物 酸→Mn²⁺（得5e⁻）；中性→MnO₂（得3e⁻）；碱→MnO₄²⁻（得1e⁻）
浓稀H₂SO₄氧化机理不同 混淆"酸性"与"氧化性" 稀H₂SO₄酸性（H⁺氧化）；浓H₂SO₄氧化性（S+6氧化）
