⚡ 难点突破1  与量有关的离子方程式书写技巧

高三化学第一轮复习 · 高考高频失分点，全国卷T11/T12常考
📋 前测（5分钟）——快速诊断
1.（2021·全国甲卷）向NaHCO₃溶液中加入少量Ba(OH)₂溶液，下列离子方程式正确的是（　　）
A. HCO₃⁻ + Ba²⁺ + OH⁻ = BaCO₃↓ + H₂O

B. 2HCO₃⁻ + Ba²⁺ + 2OH⁻ = BaCO₃↓ + CO₃²⁻ + 2H₂O
✅ A
解析：少量Ba(OH)₂ → Ba(OH)₂计量数为1 → 1个Ba²⁺，1个OH⁻参与反应。
OH⁻只中和1个HCO₃⁻，余下1个HCO₃⁻不变。
❌ B是过量Ba(OH)₂的写法。
关键：以少量物质的计量数为1来配平。
2.（2020·全国卷）向Al₂(SO₄)₃溶液中滴加NaOH溶液至过量，反应分步为：
第一步（少量NaOH）：Al³⁺ + 3OH⁻ = Al(OH)₃↓
第二步（NaOH过量）：Al(OH)₃ + OH⁻ = Al(OH)₄⁻
若直接加过量NaOH，总方程式是什么？
Al³⁺ + 4OH⁻ = Al(OH)₄⁻

解析：过量NaOH → Al(OH)₃会继续溶解。合并两步：Al³⁺ + 3OH⁻ + OH⁻ = Al(OH)₃ + OH⁻ → Al(OH)₄⁻
即 Al³⁺ + 4OH⁻ = Al(OH)₄⁻
📌 核心方法——少量定1法（万能模板）
💡 核心原则：以少量（不足量）反应物的计量数为1，不足的物质按实际需要取用
1
判断谁少谁多
题目说"少量/少量滴加"就以此为1
2
少量物质系数=1
写出其离子，系数定为1
3
按需要取用过量物
过量物够用，需要多少写多少
4
配平检验
检查原子守恒+电荷守恒
🔬 题型一：酸式盐 + 碱（最常考）
经典模型1：Ca(HCO₃)₂ + NaOH
🟢 少量NaOH
Ca²⁺ + HCO₃⁻ + OH⁻ = CaCO₃↓ + H₂O
NaOH为1→1个OH⁻→中和1个HCO₃⁻
🔴 过量NaOH
Ca²⁺ + 2HCO₃⁻ + 2OH⁻ = CaCO₃↓ + CO₃²⁻ + 2H₂O
OH⁻充足→HCO₃⁻全部被中和
⚠️ 陷阱：少量时OH⁻只中和1个HCO₃⁻，另一个HCO₃⁻不动！系数比是 1:1:1 不是 1:2:2。
经典模型2：Ba(OH)₂ + NaHCO₃
🟢 少量Ba(OH)₂
Ba²⁺ + OH⁻ + HCO₃⁻ = BaCO₃↓ + H₂O
Ba(OH)₂为1→1个Ba²⁺+1个OH⁻
🔴 过量Ba(OH)₂
Ba²⁺ + 2OH⁻ + 2HCO₃⁻ = BaCO₃↓ + CO₃²⁻ + 2H₂O
Ba(OH)₂充足，HCO₃⁻完全反应
经典模型3：NH₄HSO₄ + Ba(OH)₂
🟢 n[Ba(OH)₂] : n[NH₄HSO₄] = 1:1
Ba²⁺ + 2OH⁻ + NH₄⁺ + H⁺ + SO₄²⁻ = BaSO₄↓ + NH₃·H₂O + H₂O
OH⁻刚好中和H⁺，余下NH₄⁺部分与OH⁻反应生成NH₃·H₂O
🔴 n[Ba(OH)₂] : n[NH₄HSO₄] = 1:2
Ba²⁺ + 2OH⁻ + 2H⁺ + SO₄²⁻ = BaSO₄↓ + 2H₂O
OH⁻只够中和H⁺，NH₄⁺不参与反应
⚠️ 难点：NH₄HSO₄中H⁺优先被中和！OH⁻先与H⁺反应，再与NH₄⁺反应。
1:2时OH⁻只够中和2个H⁺，NH₄⁺完全不动。
经典模型4：Na₂CO₃ + HCl
🟢 少量HCl
CO₃²⁻ + H⁺ = HCO₃⁻
HCl为1→1个H⁺→只生成HCO₃⁻
🔴 过量HCl
CO₃²⁻ + 2H⁺ = CO₂↑ + H₂O
HCl充足→CO₃²⁻完全转化为CO₂
🔬 题型二：氧化剂/还原剂 + 多种可被氧化/还原离子
核心规则——还原性/氧化性强弱顺序决定反应先后
💡 还原性：I⁻ > Fe²⁺ > Br⁻   |   氧化性：Cl₂ > Br₂ > I₂
少量氧化剂 → 优先氧化还原性最强的离子
反应少量Cl₂过量Cl₂
FeBr₂ + Cl₂ 2Fe²⁺ + Cl₂ = 2Fe³⁺ + 2Cl⁻
先氧化Fe²⁺（还原性更强）		 2Fe²⁺ + 4Br⁻ + 3Cl₂ = 2Fe³⁺ + 2Br₂ + 6Cl⁻
Fe²⁺和Br⁻全部被氧化
FeI₂ + Cl₂ 2I⁻ + Cl₂ = I₂ + 2Cl⁻
先氧化I⁻（还原性最强）		 2Fe²⁺ + 4I⁻ + 3Cl₂ = 2Fe³⁺ + 2I₂ + 6Cl⁻
Fe²⁺和I⁻全部被氧化
经典变式：Cl₂通入FeBr₂溶液（物质的量比不同）
n(Cl₂) : n(FeBr₂)离子方程式说明
1:2（少量） 2Fe²⁺ + Cl₂ = 2Fe³⁺ + 2Cl⁻ 只氧化Fe²⁺，Br⁻不反应
1:1 2Fe²⁺ + 2Br⁻ + 2Cl₂ = 2Fe³⁺ + Br₂ + 4Cl⁻ Fe²⁺全氧化，Br⁻氧化一半
3:2（过量） 2Fe²⁺ + 4Br⁻ + 3Cl₂ = 2Fe³⁺ + 2Br₂ + 6Cl⁻ 全部氧化
FeBr₂ + Cl₂ 的通用计算思路：
① Fe²⁺还原性强于Br⁻，Cl₂先氧化Fe²⁺
② 每摩尔FeBr₂含1mol Fe²⁺ + 2mol Br⁻
③ Fe²⁺ + ½Cl₂ → Fe³⁺ + Cl⁻（消耗½mol Cl₂/mol FeBr₂）
④ 2Br⁻ + Cl₂ → Br₂ + 2Cl⁻（消耗1mol Cl₂/mol FeBr₂）
⑤ 所以完全氧化1mol FeBr₂共需 1.5mol Cl₂，即 n(Cl₂):n(FeBr₂) = 3:2
⑥ 比例在 1:2~3:2 之间时，Fe²⁺全氧化，Br⁻部分氧化
🔬 题型三：滴加顺序不同 → 现象和产物不同
经典模型：NaOH + AlCl₃ vs AlCl₃ + NaOH
🟢 NaOH滴入AlCl₃（碱过量趋势）
先：Al³⁺ + 3OH⁻ = Al(OH)₃↓ （沉淀出现）
后：Al(OH)₃ + OH⁻ = Al(OH)₄⁻ （沉淀消失）
现象：先有白色沉淀，后沉淀溶解
🔴 AlCl₃滴入NaOH（酸过量趋势）
先：Al³⁺ + 4OH⁻ = Al(OH)₄⁻ （不生成沉淀！）
后：Al³⁺ + 3Al(OH)₄⁻ = 4Al(OH)₃↓ （后来才出沉淀）
现象：开始无沉淀，后出现白色沉淀且不消失
⚠️ 原理：谁滴入谁，谁就是"少量"。NaOH滴入AlCl₃ → NaOH先少量后过量 → 沉淀先增后溶。反过来AlCl₃滴入NaOH → AlCl₃先少量 → 局部过量碱使Al³⁺直接变成Al(OH)₄⁻。
经典模型2：Na₂CO₃ + HCl vs HCl + Na₂CO₃
🟢 Na₂CO₃滴入HCl
先：CO₃²⁻ + 2H⁺ = CO₂↑ + H₂O （立即冒泡）
现象：一开始就产生气泡（HCl始终过量）
🔴 HCl滴入Na₂CO₃
先：CO₃²⁻ + H⁺ = HCO₃⁻ （无气泡）
后：HCO₃⁻ + H⁺ = CO₂↑ + H₂O （后来冒泡）
现象：开始无气泡，一段时间后产生气泡
📝 真题演练
2021·全国甲卷下列离子方程式正确的是（　　）
A. 向Na₂SO₃溶液中通入过量Cl₂：SO₃²⁻ + Cl₂ + H₂O = SO₄²⁻ + 2Cl⁻ + 2H⁺

B. 向FeI₂溶液中通入少量Cl₂：2Fe²⁺ + Cl₂ = 2Fe³⁺ + 2Cl⁻

C. 向NaAlO₂溶液中通入少量CO₂：AlO₂⁻ + CO₂ + 2H₂O = Al(OH)₃↓ + HCO₃⁻

D. 向NH₄HCO₃溶液中加入少量NaOH：NH₄⁺ + OH⁻ = NH₃↑ + H₂O
✅ C
A ❌ 应为SO₃²⁻ + Cl₂ + H₂O = SO₄²⁻ + 2Cl⁻ + 2H⁺，但注意Na₂SO₃溶液中碱性环境，产物应为SO₄²⁻ + 2Cl⁻ + 2H⁺没错。但题目说"过量Cl₂"，Cl₂足量，方程式本身是对的... 实际上A的配平没问题。
B ❌ FeI₂中I⁻还原性强于Fe²⁺，少量Cl₂应先氧化I⁻：2I⁻ + Cl₂ = I₂ + 2Cl⁻
C ✅ NaAlO₂ + 少量CO₂ → CO₂不足生成HCO₃⁻（不是CO₃²⁻），正确
D ❌ NH₄HCO₃ + 少量NaOH，OH⁻先与HCO₃⁻反应（HCO₃⁻酸性更强），不是与NH₄⁺反应。
应为：HCO₃⁻ + OH⁻ = CO₃²⁻ + H₂O
2022·山东卷下列说法正确的是（　　）
向等体积等浓度的NaAlO₂和NaHCO₃溶液中分别滴加稀盐酸：
A. 均立即产生气泡
B. 均逐渐产生气泡
C. NaAlO₂中立即产生气泡，NaHCO₃中逐渐产生气泡
D. NaAlO₂中逐渐产生气泡，NaHCO₃中立即产生气泡
✅ D
NaHCO₃ + HCl：HCl滴入NaHCO₃ → HCl少量 → HCO₃⁻ + H⁺ = CO₂↑ + H₂O → 立即冒泡
NaAlO₂ + HCl：HCl滴入NaAlO₂ → HCl少量 → 先中和OH⁻/生成Al(OH)₃（不冒泡），HCl过量后才与Al(OH)₃反应生成Al³⁺ → 逐渐冒泡
✏️ 课堂精练
1. 写出下列反应的离子方程式：
（1）向NaHSO₄溶液中逐滴加入Ba(OH)₂溶液至沉淀恰好完全
（2）向Ba(OH)₂溶液中逐滴加入NaHSO₄溶液至沉淀恰好完全
（1）NaHSO₄溶液中滴加Ba(OH)₂至沉淀恰好完全：
Ba²⁺ + 2OH⁻ + 2H⁺ + SO₄²⁻ = BaSO₄↓ + 2H₂O
（Ba(OH)₂滴入→少量→但"至沉淀恰好完全"说明SO₄²⁻恰好全沉淀→Ba²⁺:SO₄²⁻=1:1）

（2）Ba(OH)₂溶液中滴加NaHSO₄至沉淀恰好完全：
2H⁺ + SO₄²⁻ + Ba²⁺ + 2OH⁻ = BaSO₄↓ + 2H₂O
（结果相同，但过程不同：NaHSO₄少量时H⁺先中和OH⁻，然后SO₄²⁻与Ba²⁺沉淀）
答案相同：Ba²⁺ + 2OH⁻ + 2H⁺ + SO₄²⁻ = BaSO₄↓ + 2H₂O
2.（2019·全国卷）向含有0.1mol FeBr₂的溶液中通入0.15mol Cl₂，写出离子方程式。
n(FeBr₂) = 0.1mol → 含Fe²⁺ 0.1mol，Br⁻ 0.2mol
n(Cl₂) = 0.15mol
① 先氧化Fe²⁺（还原性强）：0.1mol Fe²⁺需0.05mol Cl₂
② 剩余Cl₂ = 0.15 - 0.05 = 0.1mol，可氧化0.2mol Br⁻
③ 0.2mol Br⁻恰好全部被氧化
→ Fe²⁺和Br⁻全部被氧化
2Fe²⁺ + 4Br⁻ + 3Cl₂ = 2Fe³⁺ + 2Br₂ + 6Cl⁻
3. 下列各组中的两个离子方程式，能正确表示对应反应的是（　　）
A. AlCl₃溶液加过量氨水：Al³⁺ + 3OH⁻ = Al(OH)₃↓
B. Na₂S溶液中加少量FeCl₃：2Fe³⁺ + 3S²⁻ = 2FeS↓ + S↓
C. CuSO₄溶液加少量Na₂S：Cu²⁺ + S²⁻ = CuS↓
D. 小量SO₂通入NaOH溶液：SO₂ + 2OH⁻ = SO₃²⁻ + H₂O
✅ B
A ❌ 氨水是弱碱，不拆为OH⁻：Al³⁺ + 3NH₃·H₂O = Al(OH)₃↓ + 3NH₄⁺
B ✅ Fe³⁺氧化性>S²⁻还原性，发生氧化还原反应：2Fe³⁺ + 3S²⁻ = 2FeS↓ + S↓
C ✅ Cu²⁺ + S²⁻ = CuS↓ 没问题... 等等，这个确实是对的。
D ❌ 少量SO₂应为：SO₂ + 2OH⁻ = SO₃²⁻ + H₂O... 这确实对了。
再审查：D其实是对的。但C CuSO₄+少量Na₂S中，S²⁻会被Fe³⁺氧化吗？这里没有Fe³⁺。
C：Cu²⁺ + S²⁻ = CuS↓ ✅ 正确
D：SO₂ + 2OH⁻ = SO₃²⁻ + H₂O ✅ 也正确
注意：本题答案为 C 和 D。但B也是正确的。
实际高考中，A错（氨水不拆），BCD均对。选BCD。
📦 方法速查 + 小结
与量有关的离子方程式——三大题型速查
题型核心规则关键操作
酸式盐+碱少量定1法以少量物计量数为1，按需取用过量物
氧化还原竞争还原性/氧化性强弱顺序I⁻ > Fe²⁺ > Br⁻，先强后弱
滴加顺序谁滴入谁是"少量"NaOH+AlCl₃ vs AlCl₃+NaOH现象不同
🎯 最佳策略

① 先判断题型（酸式盐+碱 / 氧化还原 / 滴加顺序）
② 酸式盐+碱 → 立即使用"少量定1法"
③ 氧化还原 → 先排还原性/氧化性顺序
④ 滴加顺序 → 谁滴入谁就是少量，分阶段分析
