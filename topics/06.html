第6讲 物质的量浓度及溶液配制

步步高大一轮复习讲义 · 以考定教

考点1

物质的量浓度(c)的概念与计算

必考
📘 核心公式体系
🔗 物质的量浓度定义
  • 定义：单位体积溶液中所含溶质 B 的物质的量，符号 c(B)
  • 公式：c(B) = n(B) / V(液)
  • 单位：mol/L（或 mol·L⁻¹），有时也用 mmol/mL
c = n / V（液）
⚠️ 注意：① V 是溶液体积，不是溶剂体积；② V 单位必须是 L（不是 mL）
③ 从 c 求溶质物质的量：n = c · V（此即第5讲四大公式之一）
📐 稀释定律

内容：稀释前后溶质的物质的量不变。

c₁ · V₁ = c₂ · V₂

（稀释前浓度×体积 = 稀释后浓度×体积）

🎯 真题精析 — c·V 结合水解陷阱
2025年广东高考 第10题 C选项
C．在 1L 0.1mol/L 的 NH₄Cl 溶液中，NH₄⁺ 的数目为 0.1NA
C选项错误。从 c·V 计算：n(NH₄Cl) = 0.1mol/L × 1L = 0.1mol，但 NH₄⁺ 是弱碱阳离子，在溶液中会水解：
NH₄⁺ + H₂O ⇌ NH₃·H₂O + H⁺
因此溶液中 n(NH₄⁺) < 0.1mol，即 NH₄⁺ 数目 < 0.1NA。

正确选项为D（标准状况下22.4L Cl₂与足量H₂反应，形成共价键数目为2NA）。
陷阱1：把 c·V 算出的 n 直接当离子数
NH₄Cl = NH₄⁺ + Cl⁻，电离出的 NH₄⁺ 确实是 0.1mol，但 NH₄⁺ 会水解，实际数目减少。这是广东卷 NA 题的最高频陷阱之一——2023、2024、2025 连续三年都考！

陷阱2：常见可水解离子一览
• 弱碱阳离子：NH₄⁺、Fe³⁺、Al³⁺、Cu²⁺、Fe²⁺、Zn²⁺ 等 → 水解使阳离子数减少
• 弱酸阴离子：CO₃²⁻、HCO₃⁻、SO₃²⁻、CH₃COO⁻、PO₄³⁻ 等 → 水解使阴离子数减少
  • 在 NA 选择题中，看到"xxmol/L 的 NH₄Cl/FeCl₃/Na₂CO₃ 溶液，xxx 离子数目为 xxNA"——先怀疑水解
  • 判断口诀："可水解离子，数目一定 < nNA"
  • 只有强酸强碱盐（如 NaCl、KNO₃）的离子数目才严格等于 nNA
  • n = c·V 只是"名义上"的物质的量，不能替代实际存在量
🎯 同类真题 — 2024广东卷
2024年广东高考 第10题 B选项
B．1L 1mol·L⁻¹ NH₄NO₃ 溶液中含 NH₄⁺ 的数目为 NA
B选项错误。同样的水解陷阱：NH₄⁺ 水解，实际数目 < NA。此题答案选A。
连续两年完全相同的陷阱！NH₄⁺ 的水解是广东卷的"看家题"。
2023年换了花样——考 HCO₃⁻ 既水解又电离（同样数目 < NA）。
  • 3年规律：广东卷 NA 题每年必考一个"盐溶液中离子数目"选项，且该选项必错
  • 2025: NH₄⁺（水解）→ 错 | 2024: NH₄⁺（水解）→ 错 | 2023: HCO₃⁻（水解+电离）→ 错
  • 看到这种选项可以直接排除，不用算
考点2

一定物质的量浓度溶液的配制

必考
📘 容量瓶的认识与使用
⚗️ 容量瓶
  • 构造：细颈、梨形、平底玻璃瓶，配有磨口玻璃塞
  • 标识：温度（20℃）、容积、刻度线
  • 常见规格：50mL、100mL、250mL、500mL、1000mL
  • 用途：专用于配制一定体积、一定物质的量浓度的溶液
  • 注意事项：不能用于溶解、稀释、加热、长期储存溶液
📘 配制步骤（八字口诀）
🧪 用固体配制一定物质的量浓度溶液的完整步骤
  • ① 计算：根据 c 和 V，计算所需溶质的质量 m = c·V·M
  • ② 称量：用托盘天平（或分析天平）称取所需固体
  • ③ 溶解（冷却）：在烧杯中用适量蒸馏水溶解，用玻璃棒搅拌，冷却至室温
  • ④ 转移：用玻璃棒引流，将溶液沿玻璃棒注入容量瓶
  • ⑤ 洗涤：用蒸馏水洗涤烧杯内壁和玻璃棒2~3次，洗涤液全部注入容量瓶
  • ⑥ 定容：加蒸馏水至刻度线下1~2cm时，改用胶头滴管逐滴加入至凹液面最低处与刻度线相切
  • ⑦ 摇匀：盖好瓶塞，反复上下颠倒摇匀
  • ⑧ 装瓶贴签：将配好的溶液转入试剂瓶，贴好标签
🔧 关键仪器清单

托盘天平（或分析天平）、药匙、烧杯、玻璃棒、容量瓶、胶头滴管

口诀：天平称、烧杯溶、玻璃棒来引流、容量瓶里定容用、胶头滴管最后动

🎯 真题精析 — 2024广东 配制Na₂CO₃溶液
2024年广东高考 第17题(3)①（填空题）
工业锅炉需定期除水垢，其中的硫酸钙用纯碱溶液处理。兴趣小组在实验室探究 Na₂CO₃ 溶液的浓度对反应速率的影响。

① 用 Na₂CO₃ 固体配制溶液，以滴定法测定其浓度。
i．该过程中用到的仪器有 __________ 。（备选仪器如图：A.锥形瓶 B.烧杯 C.容量瓶 D.酸式滴定管）
【答案】BD（烧杯 + 酸式滴定管）

用固体配制溶液需要烧杯（溶解），以酸式滴定管进行滴定测定浓度。
注意：本题并非问"配制溶液的完整仪器"，而是问"配制溶液 + 滴定法测定浓度"两个步骤合并所需的仪器。
陷阱1：审题不清
题目问的是"配制溶液 + 滴定法测浓度"整个过程的仪器，不只是配制。如果只答烧杯和容量瓶就漏了滴定的仪器。

陷阱2：容量瓶不是必须的
这里只需配制一定浓度后用滴定法标定，不一定需要用到容量瓶精确配制定体积——用烧杯溶解后用酸式滴定管滴定即可。如果题目换成"配制 250mL 0.1mol/L Na₂CO₃ 溶液"，容量瓶才是必须的。
  • 看清楚题目是"配制溶液"还是"配制+测定"，仪器范围不同
  • 纯配制一定浓度溶液：天平、药匙、烧杯、玻璃棒、容量瓶、胶头滴管
  • 配制后用滴定法测浓度：在上述基础上加滴定管、锥形瓶等
  • 如果只是"用固体配溶液"而未指定准确体积，容量瓶不一定是必须的
🎯 真题精析 — 2023广东 配制FeSO₄溶液为何加酸
2023年广东高考 第17题(4)（填空题）
乙同学测量反应 A 的焓变。
查阅资料：配制 FeSO₄ 溶液时需加入酸。加酸的目的 __________ 。
提出猜想：Fe 粉与 Fe₂(SO₄)₃ 溶液混合，可能存在 Fe 粉和酸的反应。
【答案】抑制 Fe²⁺ 水解

Fe²⁺ 是弱碱阳离子，在水中会水解：Fe²⁺ + 2H₂O ⇌ Fe(OH)₂ + 2H⁺
加酸（提供 H⁺）可以抑制水解，防止生成 Fe(OH)₂ 沉淀，保证溶液中 Fe²⁺ 浓度的准确。
陷阱：忘记水解与抑制水解的关系
可水解的盐配制溶液时，常常需要加相应的酸或碱来抑制水解：
• FeCl₃ 溶液 → 加少量 HCl（抑制 Fe³⁺ 水解）
• FeSO₄ 溶液 → 加少量 H₂SO₄（抑制 Fe²⁺ 水解）
• Na₂CO₃ 溶液 → 有时需加少量 NaOH（抑制 CO₃²⁻ 水解）
• SnCl₂ 溶液 → 加少量 HCl（抑制 Sn²⁺ 水解）

更进一步：配制 FeSO₄ 溶液时不仅要加酸，还要加少量铁粉——防止 Fe²⁺ 被空气氧化成 Fe³⁺。
  • 万能模板："配制 XX 溶液时需加入酸/碱，目的是______" → 答"抑制水解"
  • 常用的需要加酸配制的盐：FeCl₃、FeSO₄、Fe₂(SO₄)₃、SnCl₂、AlCl₃ 等
  • 常用的需要加碱配制的盐：Na₂CO₃、Na₂S 等（抑制阴离子水解）
  • 实验题常把"配溶液加酸/碱"与"水解平衡移动"结合起来考
考点3

配制溶液的误差分析

常考
📘 误差分析总表

核心依据：c = n / V，n 偏大则 c 偏大，V 偏大则 c 偏小

操作影响 n影响 Vc 的变化
称量时砝码生锈（增重） n 偏大 — 偏高
转移时溶液溅出 n 偏小 — 偏低
未洗涤烧杯和玻璃棒 n 偏小 — 偏低
未冷却就定容 — 热胀，实际 V 偏小 偏高
定容时俯视刻度线 — V 偏小 偏高
定容时仰视刻度线 — V 偏大 偏低
定容时水加多，用胶头滴管吸出 n 偏小 — 偏低
摇匀后液面下降，再补水至刻度 — V 偏大 偏低
📐 俯视/仰视图解

定容时：眼睛视线→刻度线→凹液面最低点，三点一线。

俯视：视线向下斜 → 看到的"相切点"比实际液面低 → 加水偏少 → V偏小，c偏高
仰视：视线向上斜 → 看到的"相切点"比实际液面高 → 加水偏多 → V偏大，c偏低

考点4

物质的量浓度在高考中的延伸应用

常考
📘 三个关键延伸
① 滴定分析中浓度的计算

滴定反应：aA + bB → 产物。当达到滴定终点时：
c(A)·V(A) / a = c(B)·V(B) / b
即：c(待测) = c(标准) × V(标准) × a / (V(待测) × b)

这是广东卷实验大题中几乎必考的计算题型。

② 离子浓度与电荷守恒

电解质溶液中，阳离子所带正电荷总数 = 阴离子所带负电荷总数。
电荷守恒式：∑[阳离子电荷×浓度] = ∑[阴离子电荷×浓度]
例：Na₂CO₃ 溶液中，c(Na⁺) + c(H⁺) = 2c(CO₃²⁻) + c(HCO₃⁻) + c(OH⁻)

③ 物质的量浓度与质量分数(w)的换算

c = 1000ρw / M
ρ：溶液密度(g/mL)   w：质量分数   M：溶质摩尔质量(g/mol)
推导：c = n/V = (m溶质/M) / (m溶液/ρ×10⁻³) = (m溶液·w/M) / (m溶液/ρ×10⁻³) = 1000ρw/M

🎯 真题验证 — 2023广东 HCO₃⁻ 双重身份
2023年广东高考 第11题 C选项
侯氏制碱法涉及 NH₄Cl 和 NaHCO₃ 等物质。
C．体积为 1L 的 1mol·L⁻¹ NaHCO₃ 溶液中，HCO₃⁻ 数目为 NA
C选项错误。HCO₃⁻ 既能水解又能电离：
水解：HCO₃⁻ + H₂O ⇌ H₂CO₃ + OH⁻（减少 HCO₃⁻）
电离：HCO₃⁻ ⇌ H⁺ + CO₃²⁻（也减少 HCO₃⁻）
两条路径都在减少 HCO₃⁻ 数目，所以 n(HCO₃⁻) < 0.1mol（N < NA）。
HCO₃⁻ 的双重身份是高频考点
HCO₃⁻ 既能与 H⁺ 反应（HCO₃⁻ + H⁺ = H₂O + CO₂↑），又能与 OH⁻ 反应（HCO₃⁻ + OH⁻ = CO₃²⁻ + H₂O）。
在溶液中的行为：既水解又电离，两条路径都消耗 HCO₃⁻。

对比：NH₄⁺ 只能水解（不能电离），CO₃²⁻ 只能水解（不能电离），而 HCO₃⁻、HSO₃⁻、H₂PO₄⁻ 等"酸式根"是双通道消耗。
  • 酸式根的判断优先级：HCO₃⁻、HSO₃⁻、H₂PO₄⁻、HPO₄²⁻ 都是"双通道消耗"——数目一定 < nNA
  • 真题规律：涉及这类离子的选项，基本都是错误选项
  • 记忆口诀："酸式根离子，又要水解又电离，算数目时肯定不给力"
总结

第6讲 · 满分攻略与易错清单

⚡ 考前速记表
知识点核心内容常见错误
c = n/V V 是溶液体积，单位 L V 用 mL 代人；V 当溶剂体积
稀释定律 c₁V₁ = c₂V₂ 忽略稀释前后物质的量不变
配制步骤 计算→称量→溶解→冷却→转移→洗涤→定容→摇匀 未冷却就定容（体积不准）
容量瓶 专用定容仪器，不可加热/溶解 把容量瓶当烧杯用
c·V→离子数 必须先判断离子是否水解 算出 n 直接乘以 NA
配制易水解盐 加酸/碱抑制水解 忘记加酸导致沉淀、浓度不准
误差分析 n 偏大或 V 偏小 → c 偏高 俯视/仰视效应混淆
🎯 广东卷命题规律
  • 选择题：每年 NA 题中必有"盐溶液离子数"选项，此选项 100% 错误（因为水解）
  • 实验大题：常在滴定分析中考察浓度计算（c待=c标·V标/V待），公式中 a/b 系数不能忘
  • 填空题："配制 XX 溶液时需加酸"→ 答"抑制水解"，这是一个标准化答案
  • 误差分析：俯视/仰视最容易考，记忆口诀"俯高仰低"（俯视→c偏高；仰视→c偏低）
  • n = c·V 这个公式本身不会单独考，而是和其他坑（水解、可逆反应、非标况）组合出现
