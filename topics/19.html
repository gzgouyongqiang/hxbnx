第19讲 硫及其化合物的相互转化

考点 1

H₂S的性质与硫的归中反应

常考
📝 真题举例
2023年广东卷 · 第9题（同第18讲，深化解析）
按图装置进行实验。将稀硫酸全部加入Ⅰ中的试管（含Na₂SO₃），关闭活塞。

Ⅲ中为Na₂S溶液，观察到浑浊。

该反应中SO₂与Na₂S的反应方程式为______，其中氧化剂是______。

答案：SO₂ + 2Na₂S → 3S↓ + 2Na₂O（或写离子方程式：SO₂ + 2S²⁻ → 3S↓ + 2O²⁻）

分析：这是典型的归中反应——SO₂中S为+4价，Na₂S中S为-2价，两者反应生成S（0价）。

氧化剂：SO₂（S +4→0，得电子）；还原剂：Na₂S（S -2→0，失电子）。

经典题型 · H₂S的燃烧与氧化
将H₂S气体通入下列溶液中，无沉淀生成的是（  ）

A. 氯水   B. 溴水   C. 碘水   D. FeCl₃溶液

答案：均会产生沉淀（此题为"陷阱型"——所有选项都有沉淀！）

H₂S + Cl₂ → S↓ + 2HCl；H₂S + Br₂ → S↓ + 2HBr；H₂S + I₂ → S↓ + 2HI

H₂S + 2FeCl₃ → S↓ + 2FeCl₂ + 2HCl

这类题的变体是"将H₂S分别通入X溶液，有沉淀/无沉淀/有气体"，需要逐一判断。

⚠️ 陷阱提示

陷阱1：归中反应 vs 歧化反应

归中：同种元素不同价态→中间价态。如 SO₂ + H₂S → S（+4和-2→0）

歧化：同种元素同一价态→一高一低。如 Cl₂ + 2NaOH → NaCl + NaClO + H₂O（0→-1和+1）

SO₂本身不能发生歧化反应（+4价不能同时变高变低到相邻价态），但H₂O₂可以（0→-1和-1，不对，H₂O₂歧化：2H₂O₂ → 2H₂O + O₂，0→-2和0，OH₂O₂中O为-1价→-2和0）

陷阱2：H₂S的实验室制备

FeS + 2HCl → FeCl₂ + H₂S↑。注意：不能用浓硫酸或硝酸（会氧化H₂S），只能用稀盐酸或稀硫酸。

✅ 避坑指导

H₂S核心性质速记：

① 强还原性：可被Cl₂、Br₂、O₂、KMnO₄等氧化为S或SO₂

② 弱酸性：H₂S ⇌ H⁺ + HS⁻ ⇌ 2H⁺ + S²⁻（二元弱酸）

③ 毒性：剧毒气体，有臭鸡蛋气味

④ 不完全燃烧：2H₂S + 3O₂ → 2SO₂ + 2H₂O（完全）；2H₂S + O₂ → 2S + 2H₂O（不完全）

📌 考点1核心结论：H₂S和SO₂的归中反应是选择题中常考的氧化还原反应。解题关键是判断化合价升降方向——归中反应"高价+低价→中间价"。
考点 2

亚硫酸与亚硫酸盐的性质

常考
📝 真题举例
2022年广东卷 · 选择题（改编）
下列关于H₂SO₃（亚硫酸）的说法正确的是（  ）
A.H₂SO₃是强酸，在水溶液中完全电离
B.Na₂SO₃溶液显碱性，因为SO₃²⁻水解
C.Na₂SO₃与稀硫酸反应，不能产生气体
D.Na₂SO₃在空气中久置不会变质

答案：B

A错误：H₂SO₃是弱酸（中强酸），分步电离：H₂SO₃ ⇌ H⁺ + HSO₃⁻（Ka₁较大）；HSO₃⁻ ⇌ H⁺ + SO₃²⁻（Ka₂较小）。

B正确：Na₂SO₃是强碱弱酸盐，SO₃²⁻ + H₂O ⇌ HSO₃⁻ + OH⁻，溶液显碱性。

C错误：Na₂SO₃ + H₂SO₄ → Na₂SO₄ + SO₂↑ + H₂O，有SO₂气体产生。

D错误：Na₂SO₃在空气中会被氧化：2Na₂SO₃ + O₂ → 2Na₂SO₄（变质）。

⚠️ 陷阱提示

陷阱1：H₂SO₃ vs H₂SO₄ 的本质区别

性质H₂SO₃（亚硫酸）H₂SO₄（硫酸）
酸性强弱弱酸（中强酸）强酸
稳定性不稳定，易分解为SO₂和H₂O稳定
氧化性S(+4)，有氧化性（较弱）稀硫酸无氧化性；浓硫酸有强氧化性
还原性有较强还原性（+4可升为+6）无还原性（S已为最高价+6）
盐的稳定性Na₂SO₃易被氧化为Na₂SO₄Na₂SO₄稳定

陷阱2：Na₂SO₃变质检验

检验Na₂SO₃是否被氧化：取样品→加过量稀盐酸→有气泡（排除SO₃²⁻）→加BaCl₂溶液→若有白色沉淀（BaSO₄）说明已变质（含SO₄²⁻）。

✅ 避坑指导

亚硫酸盐核心考点：

• Na₂SO₃ + 强酸（稀H₂SO₄/HCl）→ SO₂↑（实验室制SO₂的原理）

• Na₂SO₃ 变质（被O₂氧化）→ 检验：加盐酸酸化后加BaCl₂

• Na₂SO₃有还原性：能与KMnO₄、Cl₂、Br₂、I₂等反应

• Na₂SO₃有弱氧化性：能与H₂S反应（归中）

📌 考点2核心结论：亚硫酸/亚硫酸盐是高考常考的"还原性物质"。关键对比H₂SO₃与H₂SO₄的差异：前者有还原性+不稳定，后者有强氧化性（浓）+稳定。
考点 3

硫的四种价态与转化规律

必考
📝 真题举例
2025年广东卷 · 第7题（转化关系判断）
下列关于S元素价态变化的判断正确的是（  ）

A. S（0价）→ SO₂（+4价），硫被还原
B. SO₂（+4价）→ SO₃（+6价），硫被氧化
C. H₂S（-2价）→ S（0价），氢元素被还原
D. SO₃（+6价）→ H₂SO₄（+6价），硫被还原
A.S→SO₂，硫被还原
B.SO₂→SO₃，硫被氧化
C.H₂S→S，氢被还原
D.SO₃→H₂SO₄，硫被还原

答案：B

A错误：S（0价）→ SO₂（+4价），化合价升高，硫被氧化。

B正确：SO₂（+4价）→ SO₃（+6价），化合价升高，硫被氧化。

C错误：H₂S→S，是硫元素（-2→0）化合价升高被氧化，与氢无关。

D错误：SO₃→H₂SO₄，S仍为+6价，无化合价变化，不是氧化还原反应。

⚠️ 陷阱提示

陷阱1：非氧化还原反应的转化

SO₃ → H₂SO₄：S化合价不变（+6→+6），是非氧化还原反应（化合反应）

H₂SO₄(浓) → CuSO₄：产物中S仍为+6，浓硫酸在此反应中既表现酸性又表现强氧化性，但生成物CuSO₄的S价态没变。

陷阱2：相邻价态不反应

含硫化合物的氧化还原遵循"相邻价态不反应"规律：如SO₂与H₂SO₄不反应（+4与+6相邻），但SO₂与浓H₂SO₄可以反应（因为浓H₂SO₄的强氧化性可将SO₂进一步氧化）——这个例外要记住！

陷阱3：归中反应的条件

归中反应要求"高价+低价→中间价"，且中间价不能越过两端。如H₂S(-2)与浓H₂SO₄(+6)反应，产物可以是S(0)或SO₂(+4)，但不会生成比+6更高的价态。

✅ 避坑指导

硫的四种价态转化规律：

价态代表物质升高（被氧化）的条件降低（被还原）的条件
-2H₂S、Na₂SO₂、Cl₂、Br₂、浓H₂SO₄、KMnO₄H⁺（置换出H₂）
0S（硫磺）O₂（点燃）→SO₂；浓HNO₃→H₂SO₄H₂（加热）、金属（如Fe→FeS）
+4SO₂、H₂SO₃、Na₂SO₃O₂（催化）→SO₃；Cl₂、KMnO₄H₂S（归中）→S
+6H₂SO₄、SO₃、Na₂SO₄—（已是最高价）金属（Cu、Fe）、C等还原剂
📌 考点3核心结论：硫有-2、0、+4、+6四种常见价态，掌握每种价态的氧化还原方向和常见试剂，是解决硫的转化关系题的基础。牢记"相邻价态不反应"的一般规律。
考点 4

含硫转化的综合应用——工业脱硫与实验探究

必考
📝 真题举例
2024年广东卷 · 第17题（大题，完整呈现）
含硫物质种类繁多，在一定条件下可相互转化。

(1) 实验室中，浓硫酸与铜丝反应，所产生的尾气可用______（填化学式）溶液吸收。

(2) 工业上，烟气中的SO₂可在通空气条件下用石灰石浆液吸收，生成石膏（CaSO₄·2H₂O）。该过程中，______（填元素符号）被氧化。

(3) 工业锅炉需定期除水垢，其中的硫酸钙用纯碱溶液处理：CaSO₄ + Na₂CO₃ ⇌ CaCO₃ + Na₂SO₄（反应Ⅰ）。
① 用Na₂CO₃固体配制溶液，以滴定法测定其浓度。该过程中用到的仪器有______。
② 探究浓度对反应速率的影响，部分实验数据如下表：
实验编号Na₂CO₃溶液/mL水/mLCaSO₄/g反应后消耗盐酸/mL
a40.00过量24.0
b30.010.0过量18.0
cx20.0过量12.0
则 x = ______，测得的平均反应速率之比 v(a)∶v(b) = ______。

(4) 探究反应(Ⅰ)平衡的建立。将硫酸钙加入Na₂CO₃溶液中，测定pH随时间的变化。
① 甲同学认为pH不变即达平衡；乙同学提出假设1（硫酸钙已完全消耗）和假设2（被碳酸钙包裹）。
② 验证方案：取反应混合物固液分离→取沉淀→加过量稀盐酸→有气体产生且完全溶解→加BaCl₂无白色沉淀。说明沉淀均为______。
③ 写出建立平衡的优化方案及判断依据。

答案要点：

(1) NaOH（或Na₂CO₃等碱性溶液均可）

(2) S（硫元素）。SO₂(+4)→CaSO₄(+6)，化合价升高，被氧化。

(3) ① 烧杯、酸式滴定管（选BD）
② x = 20.0 mL（总体积相同均为40.0 mL）；v(a)∶v(b) = 6∶5（由消耗盐酸体积比推算浓度变化量比）

(4) ② CaCO₃（碳酸钙）
③ 将最少CaSO₄加入Na₂CO₃溶液中，50℃搅拌，测pH，当pH不变时加Na₂SO₄固体，若pH变大，说明反应(Ⅰ)平衡已建立。

解题关键：本题融合了含硫转化（SO₂→CaSO₄）、沉淀溶解平衡（CaSO₄ ⇌ Ca²⁺ + SO₄²⁻）、平衡移动原理等知识点。

⚠️ 陷阱提示

陷阱1：工业脱硫的化学原理

"烟气脱硫"不只有一种方法：①石灰石-石膏法（CaCO₃+SO₂→CaSO₄）②氨法（SO₂+2NH₃·H₂O→(NH₄)₂SO₃）③双碱法等。但不管哪种方法，SO₂中的S最终都变成+6价（被氧化）。

陷阱2：平衡判断的证据充分性

甲同学只看"pH不变"就判断平衡，忽略了"固体是否已完全反应"的可能。这是高考实验题中常见的"证据不足"陷阱——必须排除其他可能性才能确认平衡。

陷阱3：2025广东化工流程中的SO₂

2025广东大题中，矿粉酸浸"通入SO₂"的作用是还原剂——将Fe³⁺还原为Fe²⁺（防止Fe³⁺影响后续分离），体现SO₂的还原性。SO₂在化工流程中既可以是"要去除的污染物"，也可以是"有用的还原剂"！

✅ 避坑指导

含硫转化综合题解题策略：

① 先判断S的化合价（起点→终点），确定是氧化还是还原

② 注意反应条件：催化氧化需要V₂O₅、加热；工业脱硫需要空气（O₂）

③ 实验探究题要"证据链完整"——不能只看一个现象就下结论

④ 化工流程中SO₂的角色：多数情况下是需要去除的废气，但也可以利用其还原性

📌 考点4核心结论：含硫转化的综合题是广东卷大题的"标配"（2024年考沉淀平衡+反应速率，2025年化工流程考SO₂还原性）。解题核心是掌握S的价态变化规律+实验探究的规范思路。
