答题规范2 物质的分离提纯操作规范

步步高大一轮复习讲义 · 以考定教

⚡ 前测（3分钟）—— 你的分离提纯操作描述踩分了吗？
两道填空题，测测你的答题用语是否规范
【前测①】 用重结晶法提纯苯甲酸粗品（含少量泥沙和NaCl），写出完整操作流程。
（先自己写，再对照后面的答题模板——关键词一个都不能少！） 🔍 参考答案
加热溶解 → 趁热过滤 → 冷却结晶 → 过滤 → 洗涤 → 干燥。
其中"趁热"去掉、"洗涤"遗漏，是最高频失分点！
【前测②】 过滤后如何规范描述沉淀的洗涤操作？写出关键词。
（"洗涤"两字够吗？阅卷老师要的是完整句式！） 🔍 参考答案

⚠️ 关键词："沿玻璃棒""蒸馏水""浸没沉淀""重复2~3次"，缺一扣分！

板块一

过滤 & 洗涤 —— 最基础的固液分离操作规范

🔴 必须掌握
📝 核心操作与规范用语

适用场景：不溶性固体与液体的分离（如除去泥沙、收集沉淀）。

✅ 过滤操作满分模板
过滤，沿玻璃棒将浊液注入漏斗中，使液面低于滤纸边缘。

💡 "一贴二低三靠"是操作细节，答题时重点说"沿玻璃棒""液面低于滤纸边缘"。

📝 沉淀洗涤操作规范（高频考点！）
✅ 沉淀洗涤满分模板
沿玻璃棒向漏斗中注入蒸馏水至浸没沉淀，待水自然流下后，重复操作2~3次。

🗣️ 口诀："沿棒注水→浸没沉淀→自然流下→重复2~3次"

📝 沉淀是否洗净的检验规范
✅ 检验模板
取少量最后一次洗涤液于试管中，滴加×××试剂，若无×××现象，则说明沉淀已洗涤干净。
⚠️ 常见失分陷阱

陷阱1：只写"过滤、洗涤"，无操作细节

→ 化工流程题常考操作填空题，只写"过滤"可能只拿一半分！要补"洗涤"。

陷阱2："洗涤"不写"沿玻璃棒""蒸馏水"

→ 阅卷老师按关键词给分，漏一个扣一分。

陷阱3：检验是否洗净时取"洗涤液"而非"最后一次洗涤液"

→ 必须取最后一次！前面的洗涤液本来就有杂质。

板块二

蒸发结晶 vs 冷却结晶 —— 别搞混！答题关键词完全不同

🔴 必须掌握
📝 两种结晶方式的本质区别
对比维度蒸发结晶冷却结晶（降温结晶）
适用对象 溶解度随温度变化不大的物质（如NaCl） 溶解度随温度升高显著增大的物质（如KNO₃、苯甲酸）
操作描述 蒸发浓缩至有大量晶体析出（或有晶膜出现）→ 趁热过滤 蒸发浓缩 → 冷却结晶 → 过滤 → 洗涤 → 干燥
关键差异 靠蒸发溶剂使晶体析出，要趁热 靠降温降低溶解度使晶体析出
📝 蒸发结晶 · 答题模板
✅ 满分模板（NaCl型）
蒸发浓缩至有大量晶体析出（或液面出现晶膜）时，停止加热，用余热蒸干。
✅ 满分模板（需趁热过滤型）
蒸发浓缩至有大量晶体析出 → 趁热过滤 → 洗涤 → 干燥。

💡 要不要"趁热过滤"，取决于杂质在高温下是否也析出。若杂质低温析出则必须趁热。

📝 冷却结晶 · 答题模板
✅ 满分模板（KNO₃/苯甲酸型）
蒸发浓缩 → 冷却结晶 → 过滤 → 洗涤 → 干燥

🗣️ 五字诀："蒸→冷→滤→洗→干"，一步都不能少！

⚠️ 常见失分陷阱

陷阱1：NaCl型写成"冷却结晶"

→ NaCl溶解度随温度变化很小，冷却几乎不析晶！必须用蒸发结晶。

陷阱2：冷却结晶忘记"过滤"后的"洗涤""干燥"

→ 特别是化工流程题的操作填空，每一步都是分！

陷阱3："蒸发结晶"与"蒸发浓缩"混用

→ "蒸发浓缩"是步骤，"蒸发结晶"是最终目的。描述操作流程时写"蒸发浓缩"。

板块三

重结晶 —— 高考最高频的分离提纯操作！

🔴 必须掌握
📝 什么叫重结晶？和冷却结晶什么关系？

重结晶 = 先溶解 + 再结晶。利用杂质和产物在溶剂中溶解度的差异进行提纯。

适用条件：① 杂质含量较少；② 目标产物的溶解度随温度变化大。

1 加热溶解 —— 用适量溶剂（通常是水）加热使粗品全部溶解
2 趁热过滤 —— 除去不溶性杂质（如泥沙），此时温度高，产物不析出
3 冷却结晶 —— 降温使产物因溶解度降低而析出
4 过滤 —— 分离晶体与母液（母液中溶有可溶性杂质如NaCl）
5 洗涤 —— 用少量冷溶剂洗去晶体表面的母液
6 干燥 —— 获得纯品
📝 标准答题模板（高考标准答案句式）
✅ 满分模板
加热溶解 → 趁热过滤（除去不溶性杂质） → 冷却结晶 → 过滤 → 洗涤 → 干燥

🗣️ 六字诀："溶→趁→冷→滤→洗→干"。其中"C趁热过滤"是区分冷却结晶的核心！

📝 真题情境举例
2025·广东卷·第17题(2)
某苯甲酸粗品含少量泥沙和氯化钠。用重结晶法提纯该粗品过程中，需要的操作及其顺序为：加热溶解、______（填下列操作编号）。
备选操作：a. 冷却结晶   b. 常温过滤   c. 蒸发结晶   d. 趁热过滤   e. 萃取分液
🔍 查看答案
✅ d a b（趁热过滤 → 冷却结晶 → 常温过滤）

🔑 解析：重结晶法提纯苯甲酸的方法为：加热溶解 → 趁热过滤 → 冷却结晶 → 常温过滤。
① 趁热过滤(d)：除去不溶性泥沙，此时温度高，苯甲酸在溶液中不析出。
② 冷却结晶(a)：降温后苯甲酸因溶解度降低而大量析出。
③ 常温过滤(b)：分离晶体与含有NaCl的母液。

⚠️ 易错辨析："蒸发结晶(c)"不适用于此处——NaCl溶解度随温度变化不大，若蒸发会使NaCl也析出，达不到分离目的！
⚠️ "趁热过滤"vs"常温过滤"：重结晶中两次"过滤"目的不同——第一次趁热除去不溶杂质，第二次常温收集产物晶体。阅卷时"趁热"是关键词！
2024·广东卷·第6题
提纯苯甲酸粗品（含少量NaCl和泥沙）的过程中，操作X为：
A. 加热蒸馏   B. 加水稀释   C. 冷却结晶   D. 萃取分液
🔍 查看解析
✅ 选C。

📝 苯甲酸粗品（含少量NaCl和泥沙）加水加热溶解 → 趁热过滤除泥沙 → 滤液含苯甲酸和NaCl → 由于苯甲酸溶解度受温度影响大，NaCl溶解度受温度影响小 → 冷却结晶使苯甲酸析出 → 过滤后洗涤得到苯甲酸。

💡 连续两年考同一知识点（苯甲酸重结晶），说明这是高考命题组的"心头好"！
⚠️ 常见失分陷阱

陷阱1："趁热过滤"写成"过滤"，漏了"趁热"

→ 重结晶中"趁热"是关键词！趁热过滤是为了防止产物在过滤时降温析出混入不溶性杂质中。

陷阱2：顺序写错——"冷却结晶"放在"趁热过滤"前面

→ 必须先趁热过滤除去不溶杂质，再冷却让产物结晶。顺序一错全题0分！

陷阱3：重结晶与萃取分液混淆

→ 看到"苯甲酸+NaCl+水溶液"就想萃取？错！苯甲酸在水中溶解度受温度影响大，重结晶即可。

陷阱4：重结晶后忘记"洗涤"

→ 晶体表面沾有母液（含NaCl），不洗涤的话产品纯度不够！

板块四

蒸馏 —— 利用沸点差分离液体混合物

🟡 选学了解
📝 适用条件

适用：分离互溶且沸点相差较大（一般>30℃）的液体混合物。

不适用：不互溶液体（用分液）、沸点接近（用分馏）。

📝 操作规范答题模板
✅ 蒸馏操作描述
将混合液加入蒸馏烧瓶中，加入沸石（防暴沸），温度计水银球位于蒸馏烧瓶支管口处，冷凝管中的冷却水下口进、上口出，加热，收集××℃馏分。
⚠️ 高频失分点

陷阱1：温度计水银球位置 → 蒸馏时在支管口，不是液面下！（测的是蒸气温度，不是液体温度）

陷阱2：冷凝管水流方向 → 下进上出，反了就扣分！

陷阱3：忘记加沸石 → 防暴沸是必写项！

板块五

萃取 & 分液 —— 不互溶液体的分离

🟡 选学了解
📝 萃取原理与条件

原理：利用溶质在互不相溶的两种溶剂中溶解度不同，将溶质从一种溶剂转移到另一种溶剂中。

萃取剂选择三原则：① 与原溶剂互不相溶；② 溶质在萃取剂中溶解度远大于在原溶剂中；③ 萃取剂与原溶剂、溶质均不反应。

📝 萃取分液操作规范
1 检漏 —— 检查分液漏斗是否漏液
2 加液 —— 将混合液和萃取剂加入分液漏斗
3 振荡 —— 盖好瓶塞，倒转分液漏斗，用力振荡（过程中不时打开活塞放气）
4 静置 —— 放在铁架台上静置分层
5 分液 —— 打开上方瓶塞（或使瓶塞凹槽对准漏斗口小孔），旋开下方活塞，将下层液体从下口放出，上层液体从上口倒出
📝 标准答题模板
✅ 萃取分液满分模板
将混合液注入分液漏斗中，加入×××（萃取剂），振荡、静置分层后，打开上方瓶塞，将下层液体从下口放出，上层液体从上口倒出。
⚠️ 高频失分点

陷阱1：忘记"打开上方瓶塞" → 不开瓶塞，气压原因液体放不出来！是经典考点。

陷阱2：上下层倒出方式写错 → 下层从下口放出，上层从上口倒出（防止上层被下层残留污染）。

陷阱3：忘记振荡时"放气" → 有机溶剂易挥发，振荡产生蒸气压，不放气可能冲开瓶塞。

板块六

洗气 —— 气体除杂的标准操作

🟡 选学了解
📝 洗气原理

将混合气体通过盛有液体除杂试剂的洗气瓶，杂质气体被吸收或反应，目标气体通过。

关键原则：洗气试剂只能与杂质反应，不能与目标气体反应，且不能引入新杂质。

📝 常见洗气搭配速查表
目标气体杂质气体除杂试剂装置
Cl₂HCl饱和食盐水洗气瓶
CO₂HCl饱和NaHCO₃溶液洗气瓶
CO₂SO₂饱和NaHCO₃溶液（或KMnO₄/H⁺）洗气瓶
SO₂HCl饱和NaHSO₃溶液洗气瓶
H₂HClNaOH溶液洗气瓶
N₂/O₂CO₂NaOH溶液洗气瓶
📝 真题情境举例
2024·广东卷·第8题（实验装置判断）
实验室制得的氯气中有HCl杂质，可以通过饱和食盐水洗气除杂，再通过无水氯化钙干燥，可以达到净化、干燥Cl₂的目的。
🔍 查看解析
🔑 为什么用饱和食盐水除Cl₂中的HCl？
① HCl极易溶于水 → 被饱和食盐水吸收。
② Cl₂在饱和食盐水中溶解度很小（同离子效应：饱和食盐水中Cl⁻浓度大，抑制Cl₂ + H₂O ⇌ H⁺ + Cl⁻ + HClO正向移动）。

⚠️ 不能用NaOH溶液！Cl₂也会与NaOH反应：Cl₂ + 2NaOH = NaCl + NaClO + H₂O。
⚠️ 不能用NaHCO₃溶液！会产生CO₂新杂质：HCl + NaHCO₃ = NaCl + H₂O + CO₂↑。
⚠️ 常见失分陷阱

陷阱1：洗气试剂也跟目标气体反应

→ 选试剂必须考虑目标气体！如除CO₂中的SO₂不能用NaOH（CO₂也反应），应用饱和NaHCO₃。

陷阱2：洗气后不干燥

→ 水洗后气体含水分，若后续实验需干燥气体则必须加干燥装置！

总结

分离提纯操作规范 · 得分关键词总表

🔴 考前必看
📝 六大操作得分关键词速查
操作答题关键词典型真题
过滤 沿玻璃棒、液面低于滤纸边缘 化工流程操作填空
洗涤沉淀 沿玻璃棒、蒸馏水、浸没沉淀、自然流下、重复2~3次 沉淀洗涤操作描述
检验洗净 取最后一次洗涤液、滴加×××、无×××现象 沉淀是否洗净的判断
蒸发结晶 蒸发浓缩、大量晶体析出（晶膜）、停止加热、余热蒸干 NaCl型物质的提纯
冷却结晶 蒸发浓缩、冷却结晶、过滤、洗涤、干燥 KNO₃型物质的提纯
重结晶 加热溶解、趁热过滤、冷却结晶、过滤、洗涤、干燥 2025广东17(2)、2024广东6
蒸馏 沸石、温度计水银球在支管口、下进上出 液液分离操作描述
萃取分液 检漏、振荡放气、静置、打开上方瓶塞、下出上倒 分液漏斗操作
📝 操作选择决策树

看到分离提纯题目，按以下顺序判断操作方法：

  • 🔍 Step1：是固体+液体混合物？ → ✅ 过滤
  • 🔍 Step2：是两种互溶液体？ → ✅ 蒸馏（沸点差大） 或 分馏（沸点差小）
  • 🔍 Step3：是两种不互溶液体？ → ✅ 分液
  • 🔍 Step4：溶液中提取溶解度随温度变化大的溶质？ → ✅ 冷却结晶
  • 🔍 Step5：溶液中提取溶解度随温度变化小的溶质？ → ✅ 蒸发结晶
  • 🔍 Step6：固体粗品（含少量杂质）需提纯？ → ✅ 重结晶
  • 🔍 Step7：气体中有杂质气体？ → ✅ 洗气

⚠️ 选错方法 = 全题0分！先判断混合物类型，再选操作方法！

⚠️ 五大高频失分点汇总
  1. 重结晶漏"趁热"："趁热过滤"写成"过滤"，关键词丢了
  2. 洗涤不规范：只写"洗涤"两字，没有"沿玻璃棒""蒸馏水""2~3次"
  3. 结晶方式选错：NaCl型用冷却结晶、KNO₃型用蒸发结晶——全错！
  4. 重结晶顺序乱："冷却结晶"放在"趁热过滤"前面
  5. 分液忘记"打开上方瓶塞"：不开瓶塞放不出液体，经典考点
