🔥 专项突破1  六角度判断离子方程式正误

高三化学第一轮复习 · 广东卷T14/T15必考，全国卷T11/T12年年出现
📋 前测（5分钟）——快速诊断
1.（2021·广东卷T15）下列离子方程式书写正确的是（　　）
A. SO₂通入酸性KMnO₄溶液：5SO₂ + 2MnO₄⁻ + 2H₂O = 5SO₄²⁻ + 2Mn²⁺ + 4H⁺

B. Na₂O₂与水反应：Na₂O₂ + H₂O = 2Na⁺ + 2OH⁻ + O₂↑

C. Al与NaOH溶液反应：Al + 2OH⁻ = AlO₂⁻ + H₂↑

D. Cl₂通入FeCl₂溶液：Fe²⁺ + Cl₂ = Fe³⁺ + 2Cl⁻
✅ A（B系数配平错；C漏H₂O、H₂系数错；D电荷不守恒）
❌ B：2Na₂O₂+2H₂O=4Na⁺+4OH⁻+O₂↑（系数错）
❌ C：2Al+2OH⁻+6H₂O=2Al(OH)₄⁻+3H₂↑（漏H₂O，配平错）
❌ D：2Fe²⁺+Cl₂=2Fe³⁺+2Cl⁻（电荷不守恒）
2.（2023·北京卷）下列离子方程式与所给事实不相符的是（　　）
A. 制84消毒液：Cl₂ + 2OH⁻ = Cl⁻ + ClO⁻ + H₂O

B. 食醋去除水垢中的CaCO₃：CaCO₃ + 2H⁺ = Ca²⁺ + CO₂↑ + H₂O

C. FeCl₃腐蚀覆铜板：2Fe³⁺ + Cu = 2Fe²⁺ + Cu²⁺

D. 用Na₂S去除废水中的Hg²⁺：Hg²⁺ + S²⁻ = HgS↓
✅ B（醋酸是弱酸，不能拆为H⁺，应为CaCO₃+2CH₃COOH=Ca²⁺+2CH₃COO⁻+CO₂↑+H₂O）
A✅制84消毒液，Cl₂与冷碱歧化，正确
C✅Fe³⁺氧化Cu，正确
D✅Hg²⁺与S²⁻生成HgS沉淀，正确
📌 核心知识——六看全景图

看到离子方程式 → 六看逐一排查，哪个角度错就选哪个

①看真实性
这个反应真实存在吗？（Fe+稀HNO₃生成H₂？❌）
②看拆分
该拆的拆了？不该拆的拆了？（醋酸拆H⁺？❌）
③看电荷
两边电荷相等吗？（Fe²⁺+Cu=Fe+Cu²⁺？❌）
④看守恒
原子数和电子得失守恒吗？（配平对吗？）
⑤看用量
少量/过量/滴加顺序对吗？（FeBr₂+Cl₂少量？）
⑥看介质
酸碱性影响产物吗？（KMnO₄酸性→Mn²⁺？）
🎯 最佳策略：从最容易判断的角度入手

先看拆分（最快），再看电荷（次快），最后看用量和介质（最慢）。

🔍 角度①真实性 + 角度②拆分——最快出答案
角度①：反应是否真实存在？
常见"造假"事实正确事实
Fe + 稀HNO₃ → H₂↑❌ HNO₃有强氧化性，生成NO
Cu + 稀HCl → H₂↑❌ Cu不置换H₂
Na + CuSO₄ → Cu❌ Na先与水反应
CaCO₃ + HNO₃ → Ca(NO₃)₂ + H₂CO₃❌ H₂CO₃分解为CO₂
S²⁻ + H⁺(HNO₃) → H₂S↑❌ HNO₃氧化性，生成S
角度②：拆分正确吗？——强拆弱不拆
💡 拆分规则：强酸、强碱、可溶性盐 → 拆；弱酸、弱碱、难溶物、氧化物、单质 → 不拆
物质是否拆分常见错误
醋酸（CH₃COOH）❌ 不拆（弱酸）误拆为CH₃COO⁻+H⁺
NH₃·H₂O❌ 不拆（弱碱）误拆为NH₄⁺+OH⁻
Na₂O₂❌ 不拆（化学式形式）误拆为2Na⁺+O₂²⁻
石灰乳（Ca(OH)₂）❌ 不拆（微溶，悬浊液）误拆为Ca²⁺+2OH⁻
CaCO₃❌ 不拆（难溶）误拆为Ca²⁺+CO₃²⁻
Fe³⁺ + I⁻❌ 不共存（发生氧化还原反应）误认为能共存
📝 真题解剖：六看逐项演练
2022·全国卷能正确表示下列反应的离子方程式为（　　）
A. 硫化铵溶液和硝酸混合：S²⁻ + 2H⁺ = H₂S↑
B. 明矾溶液与过量氨水混合：Al³⁺ + 4NH₃ + 2H₂O = AlO₂⁻ + 4NH₄⁺
C. 硅酸钠溶液中通入二氧化碳：SiO₃²⁻ + CO₂ + H₂O = HSiO₃⁻ + HCO₃⁻
D. 将等浓度Ba(OH)₂和NH₄HSO₄以体积比1:2混合：Ba²⁺ + 2OH⁻ + 2H⁺ + SO₄²⁻ = BaSO₄↓ + 2H₂O
🔍 逐项六看：
①真实？②拆分？③电荷？④守恒？⑤用量？

A ❌ HNO₃有强氧化性，S²⁻会被氧化为S，不生成H₂S
B ❌ Al(OH)₃不溶于氨水；且NH₃·H₂O不拆
C ❌ CO₂少量生成CO₃²⁻，过量生成HCO₃⁻；此处按过量写但题目没说，且HSiO₃⁻写法不规范（应为H₂SiO₃↓）
D ✅✅ 1:2时OH⁻全部与H⁺反应，NH₄⁺不参与；Ba²⁺+SO₄²⁻=BaSO₄↓，正确
✅ 答案：D
🔍 角度④守恒 + 角度⑤用量 + 角度⑥介质/竞争
角度⑤：用量——少量定1法
💡 以少量反应物的计量数为1，不足的物质按需要取用
反应少量过量
Ca(HCO₃)₂ + NaOHCa²⁺+HCO₃⁻+OH⁻=CaCO₃↓+H₂OCa²⁺+2HCO₃⁻+2OH⁻=CaCO₃↓+CO₃²⁻+2H₂O
Ba(OH)₂ + NaHCO₃Ba²⁺+OH⁻+HCO₃⁻=BaCO₃↓+H₂OBa²⁺+2OH⁻+2HCO₃⁻=BaCO₃↓+CO₃²⁻+2H₂O
NH₄HSO₄ + Ba(OH)₂先中和H⁺：Ba²⁺+2OH⁻+2H⁺+SO₄²⁻=BaSO₄↓+2H₂O
角度⑥：介质/竞争——高考区分度
💡 还原性竞争：I⁻ > Fe²⁺ > Br⁻；少量Cl₂先氧化I⁻
场景产物
Cl₂通入FeBr₂（少量Cl₂）先氧化Fe²⁺：2Fe²⁺+Cl₂=2Fe³⁺+2Cl⁻
Cl₂通入FeI₂（少量Cl₂）先氧化I⁻：2I⁻+Cl₂=I₂+2Cl⁻
KMnO₄酸性介质还原产物为Mn²⁺（不是MnO₂！）
KMnO₄中性/碱性介质还原产物为MnO₂或MnO₄²⁻
✏️ 课堂精练（10分钟）
1.（2022·湖南卷）下列离子方程式正确的是（　　）
A. Cl₂通入冷的NaOH溶液：Cl₂ + 2OH⁻ = Cl⁻ + ClO⁻ + H₂O

B. 用醋酸和淀粉-KI溶液检验加碘盐中的IO₃⁻：IO₃⁻ + 5I⁻ + 6H⁺ = 3I₂ + 3H₂O

C. FeSO₄溶液中加入H₂O₂产生沉淀：2Fe²⁺ + H₂O₂ + 4H₂O = 2Fe(OH)₃↓ + 4H⁺

D. NaHCO₃溶液与少量Ba(OH)₂混合：HCO₃⁻ + Ba²⁺ + OH⁻ = BaCO₃↓ + H₂O
✅ A、C
A→Cl₂+冷NaOH：歧化反应 ✅
B→醋酸弱酸不能拆H⁺，应为IO₃⁻+5I⁻+6CH₃COOH=3I₂+3H₂O+6CH₃COO⁻ ❌
C→Fe²⁺被H₂O₂氧化→Fe³⁺，水解为Fe(OH)₃↓ ✅
D→少量Ba(OH)₂：Ba²⁺+2HCO₃⁻+2OH⁻=BaCO₃↓+CO₃²⁻+2H₂O（系数错）❌
✅ 答案：AC
2.（2022·广东卷）下列关于Na的化合物之间转化反应的离子方程式书写正确的是（　　）
A. 碱转化为酸式盐：OH⁻ + H⁺ = H₂O

B. 碱转化为两种盐：Cl₂ + 2OH⁻ = Cl⁻ + ClO⁻ + H₂O

C. 过氧化物转化为碱：2Na₂O₂ + 2H₂O = 4Na⁺ + 4OH⁻ + O₂↑

D. 盐转化为另一种盐：NaHCO₃ + OH⁻ = Na⁺ + CO₃²⁻ + H₂O
✅ B、C（注意本题B为正确表达）
A→OH⁻+H⁺=H₂O是中和反应，HCO₃⁻才是酸式盐 ❌
B→Cl₂+2OH⁻=Cl⁻+ClO⁻+H₂O，生成两种盐（NaCl+NaClO）✅
C→2Na₂O₂+2H₂O=4NaOH+O₂↑，过氧化物转化为碱 ✅
D→NaHCO₃应拆为HCO₃⁻，Na⁺是多余项 ❌
✅ 答案：BC
3.（2023·浙江卷）下列反应的离子方程式不正确的是（　　）
A. Cl₂通入NaOH溶液：Cl₂ + 2OH⁻ = Cl⁻ + ClO⁻ + H₂O

B. Al₂O₃溶于NaOH溶液：Al₂O₃ + 2OH⁻ = 2AlO₂⁻ + H₂O

C. 过量CO₂通入饱和Na₂CO₃溶液：CO₂ + 2Na⁺ + CO₃²⁻ + H₂O = 2NaHCO₃↓

D. Na₂SO₃溶液中滴入CaCl₂溶液：SO₃²⁻ + Ca²⁺ = CaSO₃↓
✅ D
A→Cl₂+2OH⁻=Cl⁻+ClO⁻+H₂O ✅
B→Al₂O₃+2OH⁻=2AlO₂⁻+H₂O ✅
C→析出NaHCO₃晶体，SO₃²⁻写对了（但注意是CO₃²⁻）✅
本题C选项正确
D→H₂SO₃酸性弱于HCl，Na₂SO₃+CaCl₂不反应！D ❌
✅ 答案：D
📦 六看速查表 + 小结
本讲就一句话——六看逐项扫，错在哪一目了然
角度快速自查常考陷阱
①真实性这个反应能发生吗？HNO₃/H₂SO₄(浓)的强氧化性
②拆分弱酸/弱碱/难溶拆了吗？醋酸、NH₃·H₂O、石灰乳
③电荷左右电荷总数相等吗？Fe³⁺+Cu=Fe+Cu²⁺
④守恒原子配平对吗？Na₂O₂+H₂O配平
⑤用量少量/过量/滴加顺序对吗？Ca(HCO₃)₂+NaOH少量/过量
⑥介质酸碱性影响产物吗？KMnO₄酸性→Mn²⁺；FeBr₂+Cl₂
