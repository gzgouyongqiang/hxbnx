以考定教 · 2027届高三化学第一轮复习 · 真题来源：2023—2025广东卷

⚡ 前测（5分钟）——六角度快速检测
每道题对应一个判断角度，看看你最薄弱的环节在哪里
【角度①·反应真实性】 下列离子方程式书写正确的是（　　）
A. Cu + 2H⁺ = Cu²⁺ + H₂↑   B. Fe + 2H⁺ = Fe²⁺ + H₂↑
C. CuO + 2H⁺ = Cu²⁺ + H₂O   D. CaCO₃ + 2H⁺ = Ca²⁺ + CO₂↑ + H₂O 🔍 查看答案
✅ BCD都正确，A错误！Cu在金属活动性顺序中位于H之后，不能与稀酸（非氧化性酸）置换出H₂。→ 角度①：先问"这个反应真的能发生吗？"
【角度②·拆分规则】 下列拆分正确的是（　　）
A. CH₃COOH → H⁺ + CH₃COO⁻   B. BaSO₄ → Ba²⁺ + SO₄²⁻
C. Na₂O₂ → 2Na⁺ + O₂²⁻   D. NaOH → Na⁺ + OH⁻ 🔍 查看答案
✅ 只有D正确！A：醋酸是弱酸不拆；B：BaSO₄难溶不拆；C：Na₂O₂是过氧化物特殊物质不拆。→ 角度②：强拆弱不拆是铁律！
【角度③·电荷守恒】 判断：Fe³⁺ + Fe = 2Fe²⁺ （电荷是否守恒？）
🔍 查看答案
❌ 不守恒！左边电荷 = +3 + 0 = +3，右边 = 2×(+2) = +4。正确应为 2Fe³⁺ + Fe = 3Fe²⁺。→ 角度③：先算电荷再算原子！

角度①

反应的真实性 —— "这个反应真能发生吗？"

🔴 基础必查
📝 真题举例
2025·广东·第4题（节选）——劳动项目中的离子方程式
选项劳动项目化学知识
A向燃煤中加入生石灰脱硫减排CaSO₃ + CO₃²⁻ ⇌ CaCO₃ + SO₃²⁻
B用BaCl₂和盐酸检验粗盐中含SO₄²⁻Ba²⁺ + SO₄²⁻ = BaSO₄↓
CMgCl₂溶液点卤制豆腐MgCl₂使蛋白质盐析
D用铁粉、活性炭、食盐等制暖贴使用时铁粉被氧化，反应放热
🔍 查看答案
✅ D选项表述错误（即为本题答案）。A中CaSO₃+CO₃²⁻的转化是沉淀转化反应，真实存在；B中Ba²⁺+SO₄²⁻=BaSO₄↓是检验SO₄²⁻的经典反应；C中MgCl₂使豆浆（蛋白质胶体）聚沉（注意：严格说是聚沉不是盐析）；D中铁粉氧化放热没错但暖贴中还涉及食盐加速电化学腐蚀的过程。
💡 考点：判断离子方程式首先要确认反应本身能否真实发生！
经典高频错例 · 臆造反应
判断下列反应是否能发生：
① Cu + H₂SO₄(稀) → CuSO₄ + H₂ ↑
② Cu + 2AgNO₃ → Cu(NO₃)₂ + 2Ag
③ 2FeCl₃ + Zn → 2FeCl₂ + ZnCl₂
④ 2NaCl + 2H₂O → 2NaOH + H₂↑ + Cl₂↑（电解除外）
🔍 查看答案
① ❌ 不能！Cu在H后面，不能置换稀硫酸中的氢（浓硫酸可以，因为强氧化性）
② ✅ 能！Cu比Ag活泼，发生置换反应
③ ✅ 能但不完整！Zn更活泼，应直接把Fe³⁺还原到Fe（不是Fe²⁺），或理解为分步进行
④ ❌ 不能！这是电解熔融NaCl或电解饱和食盐水才能发生的反应，普通混合不反应
⚠️ 陷阱
⚠️ 查看陷阱

① "Cu和稀H₂SO₄能产生H₂"——❌ 这是初中就学过的常识！金属活动性顺序表中Cu在H后面，不与非氧化性酸反应产生H₂

② "Fe和稀HNO₃生成H₂"——❌ HNO₃有强氧化性，N得电子生成NO（稀）或NO₂（浓），不是H⁺得电子！产物中没有H₂

③ "H⁺和OH⁻在任何情况下都生成H₂O"——❌ 如果还有其他能反应的离子，可能优先发生别的反应！如H⁺+NO₃⁻+Fe²⁺体系，优先发生氧化还原

④ "忽略隐含的氧化还原"——❌ 如Fe₂O₃+HI体系中，I⁻会还原Fe³⁺，不能简单写Fe₂O₃+6H⁺=2Fe³⁺+3H₂O

✅ 避坑指导
✅ 查看指导

角度① 三步核查法：

  • 第一步：查金属活动性。 前面的金属能置换后面的（K/Ca/Na/Mg/Al/Zn/Fe/Sn/Pb/(H)/Cu/Hg/Ag/Pt/Au）
  • 第二步：查酸的氧化性。 浓H₂SO₄、HNO₃是氧化性酸，产物不是H₂而是SO₂/NO/NO₂
  • 第三步：查隐含反应。 有强氧化剂+强还原剂时优先发生氧化还原，而不是简单的复分解

🗣️ 口诀："先问能不能，再问怎么写"

角度②

物质的拆分规则 —— "该拆的不拆？不该拆的拆了？"

🔴 核心得分点
📝 真题举例
2025·广东·第12题（节选）——配合物的电离
CuCl微溶于水，但在Cl⁻浓度较高的溶液中因形成[CuCl₂]⁻和[CuCl₃]²⁻而溶解。将适量CuCl完全溶于盐酸得到溶液，下列叙述正确的是
A. 加水稀释，[CuCl₃]²⁻浓度一定下降
B. 加入少量NaCl固体，[CuCl₂]⁻浓度一定上升
C. H[CuCl₂]的电离方程式为：H[CuCl₂] = H⁺ + Cu⁺ + 2Cl⁻ ← 判断此式正误！
D. 体系中 c(Cu⁺)+c(H⁺)=c(Cl⁻)+c([CuCl₂]⁻)+c([CuCl₃]²⁻)+c(OH⁻)
🔍 查看答案
✅ C选项错误！H[CuCl₂]的正确电离应为：
H[CuCl₂] = H⁺ + [CuCl₂]⁻
💡 关键：[CuCl₂]⁻是配离子（内界整体），不能拆成Cu⁺和2Cl⁻！这就像NH₄⁺不能拆成N³⁻和4H⁺一样，配位键连接的内界是一个整体。
2023·广东·第14题同类变式（经典拆分题）
下列指定反应的离子方程式正确的是
A. Na₂O₂溶于水：Na₂O₂ + H₂O = 2Na⁺ + 2OH⁻ + O₂↑
B. 食醋除水垢：CaCO₃ + 2H⁺ = Ca²⁺ + CO₂↑ + H₂O
C. Al³⁺与氨水：Al³⁺ + 3NH₃·H₂O = Al(OH)₃↓ + 3NH₄⁺
D. Fe₂O₃溶于氢碘酸：Fe₂O₃ + 6H⁺ = 2Fe³⁺ + 3H₂O
🔍 查看答案
✅ C 正确
❌ A：Na₂O₂不应拆写为离子形式，且系数不对（应为2Na₂O₂+2H₂O=4Na⁺+4OH⁻+O₂↑）
❌ B：醋酸CH₃COOH是弱酸，不能拆成H⁺，应保留分子式
❌ D：HI中的I⁻具有还原性，会把Fe³⁺还原为Fe²⁺，反应真实性有问题
2024·广东·第11题（节选）——涉及NH₄⁺和OH⁻的反应
八水氢氧化钡与氯化铵固体混合反应，滴加浓盐酸后观察现象。该实验中涉及的离子反应包括：
① Ba(OH)₂·8H₂O(s) + 2NH₄Cl(s) = BaCl₂ + 2NH₃↑ + 10H₂O
② NH₃ + HCl = NH₄Cl（白烟）
其中反应①的离子方程式中，Ba(OH)₂·8H₂O和NH₄Cl都不拆——为什么？
🔍 查看答案
因为两者都是固体！离子方程式中只有可溶性强电解质在水溶液中才写成离子形式。固态物质无论溶解性如何，一律写化学式！
⚠️ 陷阱
⚠️ 查看陷阱

① 弱酸全拆成H⁺——❌ CH₃COOH、HF、H₂CO₃、H₂SO₃、HClO、H₂SiO₃都是弱酸，必须写分子式！

② 弱碱/难溶碱拆了——❌ NH₃·H₂O、Fe(OH)₃、Cu(OH)₂、Mg(OH)₂都不能拆！

③ 特殊氧化物/过氧化物乱拆——❌ Na₂O₂、CaC₂、Mg₃N₂等特殊物质写化学式！

④ 配合物/络合物的内界被拆散——❌ [CuCl₂]⁻、[Ag(NH₃)₂]⁺、[Fe(SCN)]²⁺等的配离子是一个整体！

⑤ 忽略状态！——❌ 固体、气体、氧化物不管溶解性如何一律不拆！只有水溶液中的可溶性强电解质才拆

✅ 避坑指导
✅ 查看指导

"拆与不拆"速查表（背下来！）：

✅ 必须拆（写离子形式）❌ 绝对不能拆（写化学式）
强酸：HCl、HBr、HI、HClO₄、HNO₃、H₂SO₄弱酸：CH₃COOH、HF、H₂CO₃、HClO等
强碱：NaOH、KOH、Ca(OH)₂、Ba(OH)₂、澄清石灰水弱碱：NH₃·H₂O、不溶性碱
所有可溶性盐（钠盐、钾盐、铵盐、硝酸盐等）难溶物：沉淀、固体、气体、氧化物
——特殊：过氧化物(Na₂O₂)、配合物内界
——单质：H₂、Cl₂、Fe、Cu 等

🗣️ 一句话总结："强酸强碱可溶盐拆；弱酸弱碱氧化物沉淀气体过氧化物配合物不拆；固体气体一律不拆！"

角度③

守恒检查 —— "电荷守恒 + 原子守恒 + 电子守恒"

🔴 必查项
📝 真题举例
经典守恒判断 · 广东卷高频考查方式
逐一检查下列离子方程式的守恒情况：
① Fe³⁺ + Fe = 2Fe²⁺
② 2Fe²⁺ + Cl₂ = 2Fe³⁺ + 2Cl⁻
③ 2MnO₄⁻ + 5H₂O₂ + 6H⁺ = 2Mn²⁺ + 5O₂↑ + 8H₂O
④ 2Al + 2OH⁻ + 2H₂O = 2AlO₂⁻ + 3H₂↑
⑤ MnS - 2e⁻ = S + Mn²⁺ （2025·广东·第14题C选项）
🔍 查看答案
① ❌ 电荷：左(+3+0=+3) ≠ 右(+4) → 正确：2Fe³⁺ + Fe = 3Fe²⁺
② ✅ 电荷：左(+4+0=+4) = 右(+6-2=+4)；原子也守恒
③ ✅ 电荷：左(-2+0+6=+4) = 右(+4)；原子：Mn 2=2, O 10+6=16=10+6, H 6+10=16=16 ✅
④ ✅ 电荷：左(0-2+0=-2) = 右(-2+0=-2) ✅
⑤ ✅ 电荷：左(0-2=-2) = 右(0+2=+2)? 不对！等等——MnS是固体，Mn从+2→+2（没变！），S从-2→0失2e⁻。电荷：左边(-2)，右边(0+2=+2)？
实际上：原电池负极反应，电荷不一定平衡（因为有外电路电子转移）！电极反应的电荷守恒需要结合电解质环境来看
2025·广东·非选择题（节选）
次氯酸在光照条件下分解的离子方程式为：_______________。
🔍 查看答案
✅ 2HClO =(光照)= 2H⁺ + 2Cl⁻ + O₂↑
守恒检查：
⚛ 原子：2H 2Cl 2O → 2H 2Cl 2O ✅
⚡ 电荷：0 → (+2) + (-2) + 0 = 0 ✅
⚠️ 陷阱
⚠️ 查看陷阱

① 只查原子守恒，忘查电荷守恒——❌ 这是最多丢分的地方！养成习惯：先算电荷，再算原子

② 氧化还原反应只查两样——❌ 涉及氧化还原的还要查电子转移守恒（得失电子数相等）

③ 电极反应的电荷"不平衡"——⚠️ 注意！电极反应（半反应）的电荷不需要左右相等（因为电子已经写在方程式里了），但总反应的电荷必须守恒！

④ 系数约简导致"假守恒"——❌ 有时约分后看起来守恒了，但实际电子转移数不对。如将2MnO₄⁻+...约掉公因数后可能丢失信息

✅ 避坑指导
✅ 查看指导

三步守恒检查法（每次必做！）：

  • 第一步：算电荷。左边所有离子电荷之和 = 右边所有离子电荷之和？（注意系数！）
  • 第二步：算原子。各元素原子个数左右分别相等？
  • 第三步：若涉及氧化还原 → 算电子。化合价升高的总数 = 化合价降低的总数？

⚡ 考试技巧：看到离子方程式判断题，先用3秒扫一眼电荷是否相等——50%的错误可以通过这一秒识别出来！

角度④

量的影响 —— "少量/过量/足量，结果天差地别"

🔴 区分题
📝 真题举例
经典量变题型 · Ca(HCO₃)₂ 与 NaOH
少量NaOH：Ca²⁺ + HCO₃⁻ + OH⁻ = CaCO₃↓ + H₂O
过量NaOH：Ca²⁺ + 2HCO₃⁻ + 2OH⁻ = CaCO₃↓ + CO₃²⁻ + 2H₂O

为什么同一个反应物组合，量不同方程式不同？
🔍 查看解析
"少量定1法"核心思想：
🔸 少量NaOH → 以NaOH（即OH⁻）计量数为1 → 1个OH⁻只能中和1个HCO₃⁻ → 另一个HCO₃⁻旁观
🔸 过量NaOH → 以Ca(HCO₃)₂计量数为1 → 2个HCO₃⁻都被中和 → 1个生成CaCO₃沉淀，另1个变成CO₃²⁻留在溶液中

🗣️ 口诀："谁少谁是1，跟着比例走"
经典量变题型 · NH₄Al(SO₄)₂ 与 Ba(OH)₂
向NH₄Al(SO₄)₂溶液中逐滴加入Ba(OH)₂溶液：
少量Ba(OH)₂： Al³⁺ + 2SO₄²⁻ + 2Ba²⁺ + 4OH⁻ = Al(OH)₃↓ + 2BaSO₄↓
过量Ba(OH)₂： NH₄⁺ + Al³⁺ + 2SO₄²⁻ + 2Ba²⁺ + 5OH⁻ = NH₃·H₂O + AlO₂⁻ + 2BaSO₄↓ + 2H₂O

💡 注意：过量强碱会使Al(OH)₃溶解！
🔍 查看解析
少量时：OH⁻优先与Al³⁺生成Al(OH)₃沉淀，同时Ba²⁺与SO₄²⁻生成BaSO₄沉淀
过量时：Al(OH)₃继续与OH⁻反应生成AlO₂⁻（偏铝酸根），同时NH₄⁺与OH⁻反应生成NH₃·H₂O
关键顺序：OH⁻先沉淀Al³⁺→ 再中和NH₄⁺→ 最后溶解Al(OH)₃
2025·广东·第15题（节选）——涉及量的问题
按如图组装装置并进行实验：将铜丝插入溶液中，当c中红色（酚酞）刚好褪去时，将铜丝拔离液面。下列叙述错误的是
A. a中有化合反应发生并有颜色变化
B. b中气体变红棕色时，所含氮氧化物至少有两种
C. c中溶液红色刚好褪去时，HCO₃⁻恰好完全反应
D. 若将a中稀硝酸换为浓硫酸并加热，则c中颜色会褪去
🔍 查看答案
✅ C 是错误叙述（本题答案）。
c中溶液原本呈碱性（酚酞红色），通入NO₂后：NO₂ + H₂O = HNO₃ + NO，生成的HNO₃先中和OH⁻，再与HCO₃⁻反应。红色褪去说明碱性消失（pH≤8.2），但此时HCO₃⁻还未完全反应（HCO₃⁻溶液本身pH≈8.3，酚酞已无色）。
💡 这体现了反应进程与量的关系——"恰好完全反应"需要精确计算，不能凭直觉！
⚠️ 陷阱
⚠️ 查看陷阱

① 不看"少量/过量"，想当然地写——❌ 这是量变型题目第一大丢分原因！题目写了"少量""过量""逐滴加入"就一定有讲究

② 分不清"谁少定谁"——❌ 记住：以少量的物质为基准（计量数=1），按比例推导其他物质的系数

③ 忽略后续反应——❌ 如Al(OH)₃的两性（过量强碱溶解）、CO₃²⁻的继续反应等，过量时可能有第二步反应

④ "恰好完全反应"理解偏差——❌ 如上题C选项，酚酞褪色≠HCO₃⁻完全反应，要具体分析溶液pH变化过程

✅ 避坑指导
✅ 查看指导

"少量定1法"操作步骤：

  1. 找到"少量"的那个反应物
  2. 将其化学计量数定为 1
  3. 根据化学方程式比例确定其他物质的系数
  4. 写出离子方程式（别忘了拆分规则）

角度⑤

介质匹配 —— "酸性/碱性/中性决定产物形式"

🟡 高频易错
📝 真题举例
经典介质题 · KMnO₄在不同介质中的反应
KMnO₄在三种介质中被还原的产物完全不同：
酸性介质（稀H₂SO₄）：2MnO₄⁻ + 5H₂O₂ + 6H⁺ = 2Mn²⁺ + 5O₂↑ + 8H₂O
碱性介质（NaOH）：2MnO₄⁻ + SO₃²⁻ + 2OH⁻ = 2MnO₄²⁻ + SO₄²⁻ + H₂O
中性介质（水）：2MnO₄⁻ + 3SO₃²⁻ + H₂O = 2MnO₂↓ + 3SO₄²⁻ + 2OH⁻

🎯 同一个氧化剂KMnO₄，介质不同 → 产物不同 → 离子方程式不同！
2025·广东·第14题（节选）——MnS/MnO₂原电池的介质
MnS/MnO₂二次电池放电时，正极（MnO₂极）反应在酸性介质中进行：
MnO₂ + 2e⁻ + 4H⁺ = Mn²⁺ + 2H₂O
若介质改为碱性，这个反应还能成立吗？
🔍 查看答案
❌ 不能成立！酸性介质中有大量H⁺参与反应，产物为Mn²⁺和水。
若改为碱性介质：①没有大量H⁺可用；②Mn²⁺在碱性条件下会生成Mn(OH)₂沉淀。
碱性条件下MnO₂得电子的合理产物应该是MnO₂ itself被还原为MnO(OH)或Mn(OH)₂，或者在其他体系中变为MnO₄²⁻（如果电位合适）。
💡 核心：写离子方程式前必须看清楚反应环境的酸碱性！
经典介质题 · NaClO溶液中的反应
向NaClO溶液中通入SO₂气体，下列离子方程式正确的是
A. ClO⁻ + SO₂ = Cl⁻ + SO₄²⁻
B. ClO⁻ + SO₂ + H₂O = Cl⁻ + SO₄²⁻ + 2H⁺
C. 2ClO⁻ + SO₂ = 2Cl⁻ + SO₃²⁻ + O₂
D. ClO⁻ + SO₂ + 2OH⁻ = Cl⁻ + SO₄²⁻ + H₂O
🔍 查看答案
✅ D 正确！
❌ A：漏掉了氧原子来源（需要H₂O参与）
❌ B：NaClO溶液呈碱性（水解：ClO⁻+H₂O ⇌ HClO+OH⁻），不可能出现大量H⁺！产物有H⁺违反介质匹配
❌ C：电子转移不守恒
✅ D：考虑了NaClO溶液的碱性环境（有OH⁻参与），原子守恒、电荷守恒均满足
⚠️ 陷阱
⚠️ 查看陷阱

① 在碱性环境中写H⁺作为产物——❌ 最常见的介质错误！NaClO溶液、Na₂CO₃溶液、NaAlO₂溶液等都是碱性环境，产物中不能出现大量游离H⁺

② 在酸性环境中写OH⁻作为产物——❌ 同理，酸性介质中不应该有大量OH⁻生成

③ 忽略盐类水解导致的酸碱性——❌ Na₂CO₃溶液碱性（CO₃²⁻水解）、NH₄Cl溶液酸性（NH₄⁺水解）、NaHCO₃溶液弱碱性

④ KMnO₄介质产物记混——❌ 酸性→Mn²⁺（无色/肉色）/ 碱性→MnO₄²⁻（绿色）/ 中性→MnO₂（棕色沉淀）

✅ 避坑指导
✅ 查看指导

介质判断三步法：

  • 第一步：看清反应环境。 题目说了"酸性""碱性""中性"？还是给了具体的酸/碱？
  • 第二步：推断溶液酸碱性。 强酸弱碱盐→酸性；强碱弱酸盐→碱性；正盐看水解
  • 第三步：验证产物匹配。 酸性环境产物不能有大量OH⁻；碱性环境产物不能有大量H⁺

🗣️ 口诀："酸性不生碱，碱性不产酸"

角度⑥

隐含条件 —— "氧化性酸、还原性离子、竞争反应"

🔴 压轴级
📝 真题举例
经典隐含条件 · 还原性酸的"暗箭"
Fe₂O₃溶于氢碘酸（HI），离子方程式应该写：
A. Fe₂O₃ + 6H⁺ = 2Fe³⁺ + 3H₂O
B. Fe₂O₃ + 6H⁺ + 2I⁻ = 2Fe²⁺ + I₂ + 3H₂O
C. Fe₂O₃ + 2I⁻ = 2FeI₃ + ... ？
🔍 查看答案
✅ B正确！
💡 隐含条件：HI不仅是酸（提供H⁺），更是还原性酸（I⁻具有还原性）！
当Fe₂O₃溶于HI时，发生了两个过程：
❶ Fe₂O₃ + 6H⁺ → 2Fe³⁺ + 3H₂O（酸溶解）
❷ 2Fe³⁺ + 2I⁻ → 2Fe²⁺ + I₂（氧化还原，因为Fe³⁺能氧化I⁻）
总反应合并就是B！

🗣️ 类似的还原性酸还有HBr（Br⁻也有还原性）
经典隐含条件 · 竞争性氧化还原（"争"）
场景一：FeBr₂中通入Cl₂
少量Cl₂：2Fe²⁺ + Cl₂ = 2Fe³⁺ + 2Cl⁻（先氧化Fe²⁺）
足量Cl₂：2Fe²⁺ + 4Br⁻ + 3Cl₂ = 2Fe³⁺ + 2Br₂ + 6Cl⁻（再氧化Br⁻）

场景二：FeI₂中通入Cl₂
少量Cl₂：2I⁻ + Cl₂ = I₂ + 2Cl⁻（先氧化I⁺！）
足量Cl₂：2Fe²⁺ + 4I⁻ + 3Cl₂ = 2Fe³⁺ + 2I₂ + 6Cl⁻（再氧化Fe²⁺）

🤔 为什么两个场景的氧化顺序不一样？
🔍 查看解析
因为还原性不同！
还原性顺序：I⁻ > Fe²⁺ > Br⁻ > Cl⁻

所以Cl₂（氧化剂）优先氧化还原性最强的那个：
🔸 FeBr₂中：Fe²⁺ > Br⁻ → 先氧化Fe²⁺
🔸 FeI₂中：I⁻ > Fe²⁺ → 先氧化I⁻

🗣️ 口诀："碘铁溴，还原能力排排坐，从大到小依次挨"
2025·广东·第11题（节选）——隐含的氧化还原竞争
利用装置进行实验：打开K₁、K₂，一定时间后a中溶液变蓝。下列说法错误的是
A. a中现象体现了I⁻的还原性
B. b中H₂O₂既作氧化剂也作还原剂
C. 乙醇滴加过程中c中铜丝由黑变红，说明乙醇被氧化
D. d中有银镜反应发生，说明c中产物有乙酸
🔍 查看答案
✅ D 是错误选项（本题答案）。
d中银镜反应证明c中产物有醛基（乙醛），不是乙酸！乙酸不会发生银镜反应。
💡 这里隐含的条件是：乙醇→乙醛（有醛基）→乙酸，铜丝催化下乙醇被氧化为乙醛（铜丝由CuO黑色被还原为Cu红色），乙醛能使银氨溶液发生银镜反应。
⚠️ 陷阱
⚠️ 查看陷阱

① 把HBr、HI当成普通酸——❌ Br⁻和I⁻都有还原性，遇到氧化剂（Fe³⁺、MnO₄⁻、浓H₂SO₄等）会发生氧化还原！

② FeI₂和FeBr₂混淆氧化顺序——❌ FeI₂先氧化I⁻，FeBr₂先氧化Fe²⁺，顺序完全相反！

③ HNO₃（稀）+ 任何还原性离子——❌ 酸性条件下NO₃⁻具有强氧化性（=稀硝酸），能氧化Fe²⁺、I⁻、S²⁻、SO₃²⁻等

④ 忽略H₂O₂的两性——❌ H₂O₂既可以是氧化剂（得电子→H₂O/OH⁻），也可以是还原剂（失电子→O₂），要看跟谁反应

⑤ "酸性NO₃⁻不氧化任何人"——❌ 大错特错！H⁺+NO₃⁻在一起就是HNO₃，强氧化性！

✅ 避坑指导
✅ 查看指导

隐含条件排查清单（写完方程式后自查）：

  1. 是不是还原性酸？ HI、HBr → 注意阴离子可能被氧化
  2. 有没有强氧化剂+强还原剂共存？ Fe³⁺+I⁻、MnO₄⁻+Cl⁻(浓)、HNO₃(酸)+Fe²⁺ → 优先氧化还原
  3. 多个还原剂谁先被氧化？ 按I⁻ > Fe²⁺ > Br⁻ > Cl⁻顺序
  4. 产物会不会继续反应？ Al(OH)₃+过量碱、NH₄⁺+OH⁻继续受热等

① 反应真不真 → ② 该拆不该拆 → ③ 电荷守不守 → ④ 少量定为一 → ⑤ 介质配产物 → ⑥ 还有暗箭防
