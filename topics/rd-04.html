考点 1

海水提溴

必考
📝 真题举例
2025年天津卷 · 选择题（改编）
下列关于海水制溴的说法正确的是（  ）
A.海水经氯气氧化可直接得到液溴
B.海水中的Br⁻需经氧化生成Br₂，但需进一步蒸馏或萃取才能得到液溴
C.海水提溴过程中不需要进行酸化
D.溴蒸气能用NaOH溶液吸收，但不能回收利用

答案：B

A错误：海水通入Cl₂只能将Br⁻氧化为Br₂（水溶液/蒸气），不能直接得到液溴。Br₂需经蒸馏（利用沸点58.8℃）或萃取进一步分离。

B正确：Cl₂ + 2Br⁻ → Br₂ + 2Cl⁻，生成Br₂后浓度很低，需空气吹出→浓Na₂SO₃吸收→蒸馏得到液溴。

C错误：海水提溴通常需要用硫酸酸化，防止Cl₂在碱性海水中歧化为ClO⁻和Cl⁻。

D错误：Br₂ + 2NaOH → NaBr + NaBrO + H₂O，吸收后的溶液可用酸处理回收Br₂（Br⁻ + BrO⁻ + 2H⁺ → Br₂ + H₂O）。

2025年山东卷 · 选择题（改编）
下列关于液溴的保存和性质，说法正确的是（  ）
A.液溴加水封保存在广口试剂瓶中
B.液溴可保存在铁制容器中
C.液溴有毒，但不会挥发
D.Br₂的CCl₄溶液呈橙红色

答案：A

A正确：液溴极易挥发且有毒，需"水封"（在液溴上方加一层水），保存在带玻璃塞的广口试剂瓶（棕色更好，见光分解）。

B错误：液溴会腐蚀铁（常温下浓溴能氧化铁），不能保存在铁制容器中。

C错误：液溴易挥发（红棕色蒸气），有毒。

D错误：Br₂的CCl₄溶液呈橙红色→红棕色（浓度高时深红棕色）；I₂的CCl₄溶液才是紫色。

⚠️ 陷阱提示

陷阱1："海水直接得到液溴"是经典错误选项

海水含Br⁻浓度很低（约0.065g/L），通入Cl₂后Br₂浓度极低。工业上需要：
① 酸化海水 → ② 通Cl₂氧化 → ③ 空气吹出Br₂ → ④ Na₂SO₃吸收（富集）→ ⑤ 蒸馏得到液溴

直接说"海水通Cl₂得液溴"是高考最常见的错误选项之一！

陷阱2：萃取剂的选择

萃取Br₂/I₂常用的萃取剂：CCl₄（密度大于水，下层）、苯（密度小于水，上层）。
不能用的"萃取剂"：①酒精（与水互溶）②裂化汽油（含烯烃，与Br₂加成）③乙酸（与水互溶）

✅ 避坑指导

海水提溴完整流程：

海水 → 酸化(H₂SO₄) → 通入Cl₂(Cl₂ + 2Br⁻ → Br₂ + 2Cl⁻) → 空气吹出 → Na₂SO₃吸收(Br₂ + SO₃²⁻ + H₂O → 2Br⁻ + SO₄²⁻ + 2H⁺) → 蒸馏 → 液溴

关键试剂和反应：

• 氧化剂：Cl₂（将Br⁻→Br₂）

• 吸收剂：Na₂SO₃（将Br₂→Br⁻，实现富集）

• 尾气处理：NaOH溶液（吸收Br₂蒸气）

📌 考点1核心结论：海水提溴的核心是"氧化Br⁻→Br₂→富集→蒸馏"。高考选择题最爱考"直接得到液溴"这个陷阱，以及萃取剂的正确选择。
考点 2

海带提碘

常考
📝 真题举例
经典题型 · 海带提碘实验
从海带中提取碘的实验流程如下：

海带 → 灼烧 → 灰分 → 加水浸取 → 过滤 → 滤液 → 氧化(Cl₂/H₂O₂) → 萃取(CCl₄) → 分液 → 蒸馏CCl₄ → I₂

下列说法正确的是（  ）
A.灼烧海带时需要在坩埚中进行
B.过滤操作中使用到的玻璃仪器有烧杯、漏斗、玻璃棒
C.氧化时最好使用浓硫酸
D.萃取后CCl₄层在上层

答案：A、B

A正确：海带灼烧需在坩埚（或蒸发皿）中进行，不能在烧杯中进行。

B正确：过滤使用的玻璃仪器：烧杯、漏斗、玻璃棒（引流）。

C错误：氧化I⁻→I₂常用H₂O₂（绿色氧化剂，不引入新杂质）或适量Cl₂，不能用浓硫酸（会氧化有机物产生SO₂等）。

D错误：CCl₄密度大于水（1.59g/mL），萃取后CCl₄层在下层。若用苯（密度0.88g/mL）萃取，苯层在上层。

⚠️ 陷阱提示

陷阱1：萃取层的上/下判断

密度 > 水的萃取剂（CCl₄、二硫化碳等）→ 有机层在下层

密度 < 水的萃取剂（苯、汽油、乙酸乙酯等）→ 有机层在上层

这是实验题常考考点，务必记住常见萃取剂的密度关系！

陷阱2：氧化剂的选择

海带提碘常用H₂O₂（在酸性条件下）：2I⁻ + H₂O₂ + 2H⁺ → I₂ + 2H₂O
优点：不引入新杂质（H₂O₂产物是H₂O）。
若用Cl₂：Cl₂ + 2I⁻ → I₂ + 2Cl⁻，但如果Cl₂过量，会进一步将I₂氧化为IO₃⁻！

✅ 避坑指导

海带提碘完整流程：

海带 → 灼烧（破坏有机物，使I⁻进入灰分）→ 加水溶解 → 过滤 → 滤液（含I⁻）→ 加稀H₂SO₄酸化 + H₂O₂氧化 → CCl₄萃取 → 分液 → 蒸馏回收CCl₄得I₂

关键操作及目的：

操作目的
灼烧破坏有机物，使I⁻溶入水
过滤分离不溶灰分
酸化提供酸性环境，促进H₂O₂氧化I⁻
萃取富集I₂（I₂在CCl₄中溶解度远大于水）
分液分离有机层（CCl₄+I₂）和水层
蒸馏回收CCl₄溶剂，得到I₂固体
📌 考点2核心结论：海带提碘是实验题的经典背景，关键操作是灼烧→浸取→氧化→萃取→分液→蒸馏。高考常考萃取剂的密度判断和氧化剂的选择。
考点 3

碘的回收与卤素的综合利用

常考
📝 真题举例
2025年新课标卷 · 第8题（大题）
碘具有重要经济价值。实验室利用沉淀法从含碘废液中回收碘。

原理：第一步：含碘废液中加入AgNO₃和CuSO₄，生成CuI沉淀（富集碘）；第二步：加浓HNO₃将CuI氧化为I₂。

实验步骤：
① 向A中加入CuSO₄，搅拌溶解。将饱和AgNO₃溶液加入B中。
② 加热至65℃，逐滴加入饱和AgNO₃溶液，静置沉降。
③ 确认沉淀完全后弃去上层清液。
④ 将溶液更换为浓硝酸，逐滴加入，搅拌。
⑤ 待I₂析出后，过滤、洗涤、干燥。

(1) 仪器A和C的名称分别是______、______。
(3) 确认沉淀完全的操作：取少量清液，加入一滴淀粉溶液，______。

答案要点：

(1) A为三颈烧瓶（或蒸馏烧瓶），C为冷凝管。

(3) 取少量清液，加入一滴淀粉溶液，若无蓝色出现，说明I⁻已沉淀完全（CuI沉淀不含游离I⁻，淀粉不显色）。

本题亮点：用"沉淀法"回收碘，不同于常规的萃取法。先将I⁻转化为CuI沉淀（富集），再用浓HNO₃氧化得到I₂。

2021年广东卷 · 第6题（选择题）
下列实验操作能达到实验目的的是（  ）
A.将铜丝插入浓硝酸中，验证NO₂的生成
B.将NO₂降温加压，验证颜色变化
C.用乙醇从碘水中萃取碘
D.将剩余试剂放回原试剂瓶

答案：B

C错误：乙醇与水互溶，不能作碘的萃取剂！萃取剂必须满足：①与原溶剂不互溶 ②溶质在萃取剂中溶解度远大于原溶剂 ③不与溶质反应。

常用碘的萃取剂：CCl₄、苯、汽油等。

⚠️ 陷阱提示

陷阱1：萃取的三个条件缺一不可

① 萃取剂与原溶剂不互溶 ② 溶质在萃取剂中溶解度更大 ③ 萃取剂与溶质不反应

违反①的典型：乙醇（与水互溶）
违反③的典型：裂化汽油（含烯烃，与Br₂发生加成反应）
两者都不能做Br₂/I₂的萃取剂！

陷阱2：Cl₂过量氧化I₂

Cl₂ + 2I⁻ → I₂ + 2Cl⁻（适量）
但Cl₂过量时：5Cl₂ + I₂ + 6H₂O → 2HIO₃ + 10HCl（I₂被进一步氧化）
所以氧化I⁻时Cl₂要适量，或改用H₂O₂更安全。

✅ 避坑指导

卤素提取方法对比：

元素来源提取方法核心反应
Cl₂海水(食盐水)电解饱和食盐水2NaCl+2H₂O →⁄通电 2NaOH+H₂↑+Cl₂↑
Br₂海水(Br⁻)Cl₂氧化→吹出→吸收→蒸馏Cl₂+2Br⁻ → Br₂+2Cl⁻
I₂海带/海藻(I⁻)灼烧→浸取→H₂O₂氧化→萃取2I⁻+H₂O₂+2H⁺ → I₂+2H₂O
📌 考点3核心结论：氯、溴、碘的提取都是"氧化卤离子→单质→分离提纯"的思路。但各有差异：Cl₂是电解法，Br₂需富集（浓度太低），I₂常用萃取。掌握萃取操作规范是实验题拿分的关键。
