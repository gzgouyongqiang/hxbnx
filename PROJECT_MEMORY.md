## 三、炼丹炉宠物系统（2026-06-22 新增）

### 3.1 系统定位
- **纯娱乐放松系统**，与学习完全分离
- 但保持化学沉浸感（宠物名字、技能、对话都是化学知识）
- 积分只能通过学习系统获得，炼丹消耗积分

### 3.2 宠物数据库（43种）

每种宠物有独立拟人化emoji图标：

| 类型 | 数量 | 示例 |
|------|------|------|
| 单质-金属 | 10 | 钠宝宝👶、铁块头👷、金闪闪👑、银月姬🧝‍♀️ |
| 单质-非金属 | 4 | 碳弟弟👦、磷火灵👻、硅晶人🤖 |
| 稀有气体 | 3 | 氦气绅士🎩、氖灯侠💡 |
| 氧化物 | 4 | 水精灵🧜‍♀️、铁锈怪🧟 |
| 酸 | 3 | 硫酸暴君👹、硝酸炸弹💥 |
| 碱 | 3 | 氢氧化钠魔女🧙‍♀️、氨气仙子🧚‍♀️ |
| 盐 | 4 | 硫酸铜蓝晶🔷、高锰酸钾紫晶💜 |
| 有机物 | 7 | 乙烯舞者💃、乙醇醉仙🍺、葡萄糖甜心🍬、DNA螺旋🧬 |
| 特殊 | 5 | 氟气女皇👸、臭氧卫士🌍、富勒烯球⚽ |

### 3.3 炼丹配置

| 丹炉 | 积分消耗 | 产出品质概率 |
|------|----------|--------------|
| 凡火丹炉 | 10分 | 普通50% / 稀有35% / 史诗12% / 传说3% |
| 地火丹炉 | 30分 | 普通30% / 稀有40% / 史诗22% / 传说7% / 神话1% |
| 天火丹炉 | 60分 | 普通15% / 稀有30% / 史诗35% / 传说16% / 神话4% |
| 三昧真火 | 100分 | 普通5% / 稀有20% / 史诗35% / 传说30% / 神话10% |
| 混沌神火 | 200分 | 稀有10% / 史诗25% / 传说40% / 神话25% |

- 每日炼丹上限：10次
- 炼丹师等级：学徒→师→大师→宗师→圣→神
- 每日运势：7种元素（火/水/金/木/土/光/暗），对应宠物类型概率加成

### 3.4 宠物交易中心（pets.html）

- **我的宠物**：查看已收集宠物，重复宠物可分解（品级×10积分）
- **宠物合成**：2只同品级合成1只更高品级（随机）
- **宠物图鉴**：查看全部43种宠物收集进度

### 3.5 文件位置

- `assets/alchemy.js` — 宠物数据库 + 炼丹API
- `alchemy.html` — 炼丹炉页面
- `pets.html` — 宠物交易中心

## 四、积分系统（v4精简版）

### 4.1 积分获取
- 完成周次学习任务
- 辅助练习（俗称/方程式/元素推断）答题正确

### 4.2 积分消耗
- 炼丹炉消耗积分获取宠物
- 宠物合成消耗积分

### 4.3 数据存储
- LocalStorage key: `hxbnx_game_v2`
- 包含：score（积分）、streak（连续打卡）、wrongQuestions（错题）、petCollection（宠物收集）
