## 二、网站核心架构

### 2.1 大关卡结构（68关 · 6章节）

基于《高三化学一轮复习·100课时进度表》，移除了32个测试/讲评/机动课时：

| 章节 | 图标 | 课时编号 | 关卡数 |
|------|------|----------|--------|
| 化学基本概念与计算 | 🧪 | 1~9 | 9关 |
| 元素化合物 | ⚗️ | 15~25 | 11关 |
| 物质结构与元素周期律 | 🔬 | 31~39 | 9关 |
| 化学反应原理 | ⚡ | 43~51, 53~63 | 20关 |
| 有机化学基础 | 🧬 | 67~78 | 12关 |
| 化学实验 | 🔭 | 83~89 | 7关 |
| **合计** | | | **68关** |

每关分 **铜牌/银牌/金牌** 三个小关，通关分别获得 10/50/100 积分。

### 2.2 已移除的课时（不做网站上）

第10~14, 26~30, 40~42, 52, 64~66, 79~82, 90~100 课时为测试/讲评/机动/综合训练内容。
