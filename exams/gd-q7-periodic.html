🔒 三步解题法

1
定元素
位置→元素；
结构式→共价键数定族；
电离能图→峰位定族
2
用规律
原子半径(层多径大/同层核多径小)
电负性(右→↑，上→↑)

3

氢键→沸点反常
ⅡA/VA第一电离能反超
Si最高价酸为H2SiO3弱酸

⚠️ 四大高频陷阱

陷阱1 · 氢化物沸点：有氢键(NH3/H2O/HF)的化合物沸点反常高，不能用分子量规律判断！PH3<NH3、H2S<H2O
陷阱2 · 第一电离能反常：ⅡA族>ⅢA族（如Be>B、Mg>Al），VA族>ⅥA族（如N>O、P>S），因为半满/全满稳定
陷阱3 · 最高价氧化物水化物酸性：非金属性越强酸性越强，但Si对应H2SiO3是弱酸，N对应HNO3是强酸——不能光看族
陷阱4 · "一定"类绝对词：选项中出现"一定"要高度警惕——如"甲的氢化物遇氯化氢一定有白烟"（只有NH3有，CH4没有！）
📌 2022年 · 第7题2分 · 位置推断类
甲～戊均为短周期元素，在元素周期表中的相对位置如图所示。戊的最高价氧化物对应的水化物为强酸。下列说法不正确的是
A. 原子半径：丁＞戊＞乙
B. 非金属性：戊＞丁＞丙
C. 甲的氢化物遇氯化氢一定有白烟产生
D. 丙的最高价氧化物对应的水化物一定能与强碱反应
元素周期表位置推断原子半径非金属性
【答案】C

【分析】戊的最高价氧化物水化物为强酸 → 戊可能是S(硫酸)或Cl(高氯酸)。
若戊=Cl：甲=N、乙=F、丙=P、丁=S
若戊=S：甲=C、乙=O、丙=Si、丁=P

选项详解：
A ✅ 原子半径：层多径大，同层核多径小 → 丁(S/P)＞戊(Cl/S)＞乙(F/O)，正确
B ✅ 非金属性同周期从左→右增强 → 戊(Cl/S)＞丁(S/P)＞丙(P/Si)，正确
C ❌ 甲可能是C(氢化物CH4)遇HCl不反应，无白烟！只有NH3+HCl→NH4Cl才有白烟
D ✅ 丙对应H3PO4或H2SiO3(或H4SiO4)，均能与强碱反应，正确
⚠️ 这道題的致命陷阱：

🔤 "一定有白烟"的绝对表述——甲可能是C也可能是N。C的氢化物(CH4)遇HCl不反应，无白烟！只有NH3+HCl→NH4Cl才有白烟。

🔤 戊的多种可能——短周期元素最高价氧化物水化物为强酸的不仅Cl(高氯酸)，还有S(硫酸)。两种推法都正确，所以需要分情况讨论！
✅ 避坑指导：

① 看到"一定"要警惕 → 找反例
② 位置图推断不能只有一个答案 → 两种可能都要验证
③ NH3+HCl=NH4Cl（白烟），CH4+HCl不反应 → 这是关键区分点
④ 原子半径口诀："层多径大，同层核多径小"反复用
📌 2023年 · 第14题4分 · 结构式推断类
化合物可作肥料，所含的5种元素位于主族，在每个短周期均有分布，仅有Y和M同族。Y的基态原子价层p轨道半充满，X的基态原子价层电子排布式为nsⁿ，X与M同周期，E在地壳中含量最多。下列说法正确的是
A. 元素电负性：E＞Y＞Z
B. 氢化物沸点：M＞Y＞E
C. 第一电离能：Y＞E＞X
D. YE3⁻和XE3²⁻空间结构均为三角锥形
电负性第一电离能氢键沸点空间结构
【答案】A

【推断过程】
E在地壳中含量最多 → E=O
X基态价层电子排布nsⁿ，n=2→2s²(Mg)或n=1→1s¹(Li)
Y价层p轨道半充满 → Y=N或P
Y和M同族 → M=N或P
X与M同周期 → 若M=N(第二周期)，X=Li；若M=P(第三周期)，X=Mg
化合物含5种元素→Z=H
最终：E=O，Y=N，Z=H，M=P，X=Mg（化合物NH4MgPO4）

选项详解：
A ✅ 电负性：O(3.44)＞N(3.04)＞H(2.20)，正确
B ❌ 沸点：NH3有氢键(沸点高)，H2O有氢键(沸点更高)，PH3无氢键(沸点低)→顺序应为 H2O＞NH3＞PH3
C ❌ 第一电离能：ⅡA族Mg大于ⅢA族Al，ⅤA族N大于ⅥA族O → 顺序应为N＞O＞Mg
D ❌ NO3⁻价层电子对=3，sp²杂化平面三角形；MgO3²⁻不存在(CO3²⁻才是平面三角形)
⚠️ 这道題的致命陷阱：

🔤 氢键干扰沸点比较——B选项"氢化物沸点"不是"分子晶体符合分子量规律"，NH3和H2O有分子间氢键，沸点反常高。PH3无氢键最低。

🔤 第一电离能ⅡA/ⅤA反超——N(ⅤA)＞O(ⅥA)，Mg(ⅡA)＞Al(ⅢA)。C选项Y(N)＞E(O)正确，但E(O)＞X(Mg)错误——因为Mg是ⅡA族！

🔤 空间结构辨析——NO3⁻是平面三角形(sp²)，不是三角锥形(sp³有孤电子对)。三角锥形是NH3、PH3等。
✅ 避坑指导：

① 元素推断：E在地壳中含量最多→锁定O，这是送分线索
② 氢化物沸点：先看有无氢键(N/O/F)，再看分子量
③ 第一电离能：背熟ⅡA>ⅢA、ⅤA>ⅥA这两个反超
④ 空间结构：AX3型→看孤电子对。0对→平面三角形，1对→三角锥形
📌 2024年 · 第12题4分 · 结构式推断类
一种可为运动员补充能量的物质，其分子结构式如图。已知R、W、Z、X、Y为原子序数依次增大的短周期主族元素，Z和Y同族，则
A. 沸点：ZR3 < YR3
B. 最高价氧化物的水化物的酸性：Z < W
C. 第一电离能：Z < X < W
D. ZX3⁻和WX3²⁻空间结构均为平面三角形
结构式推断共价键数空间结构第一电离能
【答案】D

【推断过程】
Y形成5个共价键 → Y=P或N（但Z和Y同族且Z原子序数小）
Z形成3个共价键 → Z=N（3个键+N→P同族）→ Y=P
W形成4个共价键 → W=C（原子序数在N之前）
R形成1个共价键 → R=H（原子序数最小）
X形成2个共价键且在N和P之间 → X=O
综上：R=H, W=C, Z=N, X=O, Y=P

选项详解：
A ❌ NH3有氢键，PH3无氢键 → 沸点NH3＞PH3，错误
B ❌ 非金属性N＞C → 酸性HNO3＞H2CO3 → Z(N)＞W(C)，错误
C ❌ 第一电离能：N(ⅤA)＞O(ⅥA)＞C，即Z＞X＞W，不是Z＜X，错误
D ✅ NO3⁻(sp²，平面三角形)，CO3²⁻(sp²，平面三角形)，正确！
⚠️ 这道題的致命陷阱：

🔤 第一电离能方向陷阱——C选项说 Z＜X＜W 即 N＜O＜C。实际上N(ⅤA)＞O(ⅥA)，N最大！正确顺序是 N＞O＞C。如果只看元素位置从左到右"增大"会出错。

🔤 氢键沸点——NH3有氢键沸点远高于PH3，不能用分子量比较。

🔤 共价键数推断——看结构式要会数键：H成1键、C成4键、N成3键、O成2键、P成5键——这是基本功！
✅ 避坑指导：

① 结构式推断：数共价键数→定族→推元素
② 第一电离能：涉及N、O、F时要画出来验证！N＞O（ⅤA反超）
③ 氢键沸点：NH3(有氢键) vs PH3(无氢键)，先看有无氢键
④ 空间结构：用VSEPR，AX3 + 无孤电子对 = sp² = 平面三角形
📌 2025年 · 第9题2分 · 图表推断类（新趋势）
元素a~i为短周期元素，其第一电离能与原子序数的关系如图。下列说法正确的是
A. a和g同主族
B. 金属性：g > h > i
C. 原子半径：e > d > c
D. 最简单氢化物沸点：b > c
第一电离能图表推断金属性原子半径
【答案】B

【推断过程】利用第一电离能特征峰推断元素：
同周期第一电离能：ⅡA族>ⅢA族，ⅤA族>ⅥA族，0族最大
a~i依次为：B、C、N、O、F、Ne、Na、Mg、Al

选项详解：
A ❌ a是B(ⅢA)，g是Na(ⅠA)，不同主族，错误
B ✅ g=Na、h=Mg、i=Al同周期，从左到右金属性减弱→Na＞Mg＞Al，正确
C ❌ c=N、d=O、e=F同周期，原子半径从左到右减小→c＞d＞e，不是e＞d＞c，错误
D ❌ b=C，c=N→氢化物CH4 vs NH3。NH3有氢键沸点高→NH3＞CH4，不是b＞c，错误
⚠️ 这道題的致命陷阱：

🔤 原子半径方向——C选项"e＞d＞c"=F＞O＞N。同周期原子半径从左→右减小！正确是N＞O＞F(即c＞d＞e)。如果记反了"电负性大半径大"就完蛋。

🔤 氢化物沸点——D选项CH4 vs NH3，NH3有氢键沸点远高于CH4。不能单纯比分子量！

🔤 第一电离能图读峰位——a~i贯穿三个周期，要通过峰位特征(ⅡA>ⅢA、ⅤA>ⅥA、0族最大)来定位元素，不能凭感觉猜。
✅ 避坑指导：

① 第一电离能图：记住三大特征——ⅡA峰、ⅤA峰、0族峰
② 金属性强弱：同周期从左→右金属性减弱，同主族从上→下金属性增强
③ 原子半径：同周期从左→右减小（核电荷增大），同主族从上→下增大（电子层增多）
④ 氢键=沸点异常：含N-H、O-H、F-H的分子沸点比同类高很多

💡 复习建议

  1. 背熟"铁三角"口诀：层多径大→同层核多径小；右上电负大→左下金属强
  2. 第一电离能两处反超必须刻在脑子里：Be＞B、Mg＞Al（ⅡA＞ⅢA）；N＞O、P＞S（ⅤA＞ⅥA）
  3. 氢键三兄弟：NH3、H2O、HF——比沸点看到N/O/F就先想有没有氢键
  4. 结构式推元素：数共价键→定族（H1/C4/N3/O2/F1/P3或5/S2/Cl1）
  5. VSEPR速判空间结构：AX3无孤电子对=平面三角形，有1对=三角锥形
