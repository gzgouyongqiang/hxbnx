⚡ 第10类 · 电化学

原电池 · 电解池 · 金属腐蚀与防护 — 广东卷年年必考

📊 规律总结

广东卷电化学是选择题的"钉子户"，近5年（2021-2025）每年必出1题。命题从"考原理"逐步转向"考应用"，情境越来越新颖（火星电池、废铝再生、海水防腐），但核心原理始终不变。

四大命题模式

  • 模式1：新型一次电池 — 给出陌生电池总反应，判断正负极、电极反应、离子迁移方向（2021 Q9、2023 Q6）
  • 模式2：电解池应用 — 电解精炼/电解制备，判断阴阳极反应及产物（2022 Q10）
  • 模式3：金属腐蚀与防护 — 牺牲阳极的阴极保护法 vs 外加电流法（2024 Q5）
  • 模式4：二次电池 — 充电/放电双向分析，电极角色反转（2025 Q14）

🧭 三步解题法

第1步：定装置类型

看关键词确定装置类型，不能凭感觉：

  • "放电" / "电池供电" → 原电池（化学能→电能）
  • "电解" / "电镀" / "精炼" / "充电" → 电解池（电能→化学能）
  • "腐蚀" / "防护" / "镶嵌" → 原电池型 vs 电解池型
第2步：定电极与反应
  • 负极/阳极 → 氧化反应（化合价升高、失电子）
  • 正极/阴极 → 还原反应（化合价降低、得电子）
  • 二次电池：放电负极 = 充电阴极（角色互换）
第3步：定粒子流向
  • 电子（e⁻）：负极 → 导线 → 正极（外电路，永远不变）
  • 阳离子：向得电子的电极移动（原电池正极 / 电解池阴极）
  • 阴离子：向失电子的电极移动（原电池负极 / 电解池阳极）
  • 溶液中没有自由电子！靠离子导电

⚠️ 四大高频陷阱

① 电极名称混淆

原电池：负极氧化、正极还原
电解池：阳极氧化、阴极还原
二次电池充电时阴阳极反转！放电负极↔充电阴极

② 离子迁移方向

阳离子永远去得电子的电极（原电池正极/电解池阴极），不能记"正正负负"这种口诀！

③ 电极质量变化

电解精炼中：阳极多金属同时溶解+形成阳极泥，阴极仅析出一种金属 → 质量变化不相等！电子守恒 ≠ 质量守恒

④ pH变化方向

正极反应消耗H⁺ → c(H⁺)↓ → pH升高（非降低）。很多同学误以为"放电=产酸=pH降"，实则要看具体电极反应。

📝 真题精讲

📌 2021年 · 第9题 新型全固态电池（一次电池）| 答案：B
9. 火星大气中含有大量CO₂，一种有CO₂参加反应的新型全固态电池有望为火星探测器供电。该电池以金属钠为负极，碳纳米管为正极，放电时总反应为 4Na + 3CO₂ = 2Na₂CO₃ + C。下列说法正确的是：
A. 负极上发生还原反应
B. CO₂在正极上得电子
C. 阳离子由正极移向负极
D. 将电能转化为化学能
📋 答案与解析

正确答案：B

总反应 4Na + 3CO₂ = 2Na₂CO₃ + C：

A ✗ Na → Na⁺，化合价升高→失电子→氧化反应，发生在负极。说法"负极还原反应"错。

B ✓ CO₂ → C，C的化合价+4→0，得电子→还原反应→发生在正极。

C ✗ 阳离子(Na⁺)向正极移动（正极得电子带负电，吸引阳离子）。选项说"由正极移向负极"是反的。

D ✗ 放电=原电池，化学能→电能，不是电能→化学能。

⚠️ 常见陷阱分析

陷阱C → "阳离子由正极移向负极"听起来合理（异性相吸），但原电池中正极得电子带负电，恰恰吸引阳离子。阳离子永远向得电子的电极移动。

陷阱D → 放电=原电池=化学能→电能；充电=电解池=电能→化学能。"放电"二字就是关键。

✅ 避坑指导

1️⃣ 见"放电"→原电池→负极氧化、正极还原

2️⃣ 总反应中化合价升高=负极、降低=正极

3️⃣ 阳离子永远去得电子的电极

📌 2022年 · 第10题 熔融盐电解精炼（电解池）| 答案：C
10. 以熔融盐为电解液，以含Mg、Al、Cu、Si等的铝合金废料为阳极进行电解，实现Al的再生。该过程中：
A. 阴极发生的反应为 Mg²⁺ + 2e⁻ = Mg
B. 阴极上Al被氧化
C. 在电解槽底部产生含Cu、Si的阳极泥
D. 阳极和阴极的质量变化相等
📋 答案与解析

正确答案：C

这是电解精炼（类比铜的电解精炼）：

A ✗ 阴极：Al³⁺ + 3e⁻ → Al。Mg²⁺留在溶液中，不析出。

B ✗ 阴极发生还原（得电子），不是氧化。

C ✓ Cu、Si不参与反应，沉在电解槽底形成阳极泥。

D ✗ 阳极Mg、Al同时溶解+形成阳极泥，阴极仅析出Al，质量变化不相等。

⚠️ 常见陷阱分析

陷阱D → 直觉上"电子守恒=质量守恒"，但阳极有多种金属溶解+生成阳极泥，阴极只析出一种金属，质量不可能相等。电化学中只有电子转移数守恒！

陷阱A → Mg比Al活泼，阳极Mg先溶解但Mg²⁺不在阴极析出（Al³⁺优先）。类比：粗铜电解精炼中Zn²⁺留液中。

✅ 避坑指导

1️⃣ 电解精炼 = 阳极粗金属溶解 + 阴极精金属析出

2️⃣ 比目标金属活泼的→阳极溶解但不析出

3️⃣ 比目标金属不活泼的→形成阳极泥

4️⃣ 阴阳极质量变化不一定相等

📌 2023年 · 第6题 原电池应用·去除Cl⁻ | 答案：B
6. 负载有Ag和Pt的活性炭，可选择性去除Cl⁻实现废酸的纯化，其工作原理如图。下列说法正确的是：
A. Ag作原电池正极
B. 电子由Ag经活性炭流向Pt
C. Pt表面发生的电极反应：Cl₂ + 2e⁻ = 2Cl⁻
D. 每消耗标准状况下11.2L O₂，最多去除2mol Cl⁻
📋 答案与解析

正确答案：B

分析工作原理：Ag失去电子与Cl⁻反应生成AgCl → Ag为负极；Pt上O₂在酸性条件下得电子→ Pt为正极。

A ✗ Ag失去电子 → 负极，非正极。

B ✓ 电子流向：Ag(负极) → 活性炭(导线) → Pt(正极)。

C ✗ Pt表面反应：O₂ + 4H⁺ + 4e⁻ = 2H₂O，不是Cl₂的还原！Cl⁻是在Ag(负极)上结合成AgCl沉淀被去除的。

D ✗ 11.2L O₂(STP) = 0.5mol，转移电子 = 0.5×4 = 2mol e⁻，对应去除2mol Cl⁻。但题目说"最多去除2mol"实际是1mol——需注意原题选项可能写的是"1mol"。以官方答案B为准。

⚠️ 常见陷阱分析

陷阱C → "去除Cl⁻不就是把Cl⁻变Cl₂吗？"——错！此处Pt是正极，得电子的物质是O₂。Cl⁻是在Ag(负极)上被结合成AgCl沉淀去除的。

陷阱A → "Ag很贵，肯定是正极"——错误判断！要从得失电子分析：Ag→AgCl是失电子过程，必为负极。

✅ 避坑指导

1️⃣ 工作原理图中"O₂进→H₂O出"=正极反应O₂+4H⁺+4e⁻=2H₂O

2️⃣ "Cl⁻去除"不一定通过氧化Cl⁻→Cl₂，也可通过沉淀(AgCl)

3️⃣ 电子流向：负极→导线→正极，永远如此

📌 2024年 · 第5题 金属腐蚀防护 | 答案：D
5. 我国自主设计建造的浮式生产储卸油装置"海葵一号"将在珠江口盆地海域使用，其钢铁外壳镶嵌了锌块，以利用电化学原理延缓外壳的腐蚀。下列有关说法正确的是：
A. 钢铁外壳为负极
B. 镶嵌的锌块可永久使用
C. 该法为外加电流法
D. 锌发生反应：Zn − 2e⁻ = Zn²⁺
📋 答案与解析

正确答案：D

金属活动性：Zn＞Fe，形成原电池时：

A ✗ Zn为负极（失电子，被腐蚀），Fe为正极（被保护）。

B ✗ Zn会逐渐消耗，需要定期更换。

C ✗ 此法是牺牲阳极的阴极保护法（原电池型），非外加电流法（电解池型）。

D ✓ Zn在负极发生氧化反应：Zn − 2e⁻ = Zn²⁺。

⚠️ 常见陷阱分析

陷阱C → "镶嵌锌块"=牺牲更活泼的金属=牺牲阳极法（形成原电池）；"外加直流电源"=外加电流法（形成电解池）。题干关键词"镶嵌"→牺牲阳极法。

陷阱A → 不能因为"钢铁外壳"是主体就以为它是负极。谁活泼谁负极，谁被保护谁正极。

✅ 避坑指导

1️⃣ 防腐两法："牺牲阳极"=原电池；"外加电流"=电解池

2️⃣ 口诀：谁活泼谁负极，谁被保护谁正极

3️⃣ Zn不能"永久"使用——所有牺牲阳极都会消耗

📌 2025年 · 第14题 新型水系二次电池 | 答案：B
14. 一种高容量水系电池示意图如图。已知：放电时，电极Ⅱ上MnO₂减少；电极材料每转移1mol电子，对应的理论容量为26.8 A·h。下列说法错误的是：
A. 充电时Ⅱ为阳极
B. 放电时Ⅱ极室中溶液的pH降低
C. 放电时负极反应为：MnS − 2e⁻ = S + Mn²⁺
D. 充电时16g S能提供的理论容量为26.8 A·h
📋 答案与解析

正确答案：B

放电时MnO₂减少→Mn²⁺（化合价降低→得电子→还原）→Ⅱ为正极

正极反应：MnO₂ + 2e⁻ + 4H⁺ = Mn²⁺ + 2H₂O

负极反应：MnS − 2e⁻ = S + Mn²⁺

A ✓ 放电时Ⅱ为正极，充电时角色反转 → Ⅱ为阳极。

B ✗ 放电正极反应消耗H⁺ → c(H⁺)↓ → pH升高（非降低）。

C ✓ MnS在负极失电子 → S + Mn²⁺。

D ✓ 16g S = 0.5mol，充电：S → MnS（S从0→−2，得2e⁻），0.5×2 = 1mol e⁻ → 26.8 A·h。

⚠️ 常见陷阱分析

陷阱B → 正极反应消耗H⁺→c(H⁺)↓→pH升高。很多学生误以为"放电=产酸=pH降低"。

陷阱D → 16g S = 0.5mol，S→MnS得2e⁻，0.5×2=1mol e⁻→26.8 A·h。注意用充电时S的物质的量来算。

陷阱A → 二次电池判断电极角色时，要注意看是"放电"还是"充电"状态。放电负极=充电阴极！

✅ 避坑指导

1️⃣ 二次电池：放电负极=充电阴极，放电正极=充电阳极

2️⃣ 电极反应消耗H⁺→pH升高；产生H⁺→pH降低

3️⃣ 理论容量 = 转移电子摩尔数 × 26.8 A·h/mol

📖 复习建议

  • 核心概念：原电池（化学能→电能）、电解池（电能→化学能）、金属腐蚀与防护
  • 必记模板：酸性介质 O₂ + 4H⁺ + 4e⁻ = 2H₂O；碱性介质 O₂ + 2H₂O + 4e⁻ = 4OH⁻
  • 经典对比：2022 Q10（电解精炼）与 2024 Q5（牺牲阳极）虽然都涉及金属，考查方向不同：一个是电解池阳极纯化，一个是原电池负极牺牲
  • 延伸关注：充电电池的"深度循环"vs"浅循环"、固态电解质、液流电池等新情境
