⚗️ 第13类 · 离子方程式正误判断

拆分规则 · 守恒检验 · 弱电解质 · 氧化还原 · 特殊反应

📊 规律总结

离子方程式正误判断是高考化学选择题中的高频经典考点，全国各卷几乎年年出现。广东卷在2022年考查后暂未再出，但全国卷（河南、广西、浙江等）持续命制。

六大常见设错类型

  • ① 弱电解质不拆 — CH₃COOH、H₂CO₃、NH₃·H₂O、H₂O、HClO等写离子方程式时应保留分子式（2025河南Q5A、广西Q8A）
  • ② 强电解质该拆不拆 — Na₂SiO₃等可溶性强电解质在溶液中应拆成离子（2022广东Q14D）
  • ③ 氧化还原反应得失电子不守恒 — SO₂与KMnO₄反应中S和Mn的化合价变化要配平（2025广西Q8D）
  • ④ 漏写气体/沉淀/弱电解质符号 — Al³⁺与HS⁻双水解应标↑和↓（2025广西Q8B）
  • ⑤ 忽略过量/少量问题 — CO₂少量和过量与碱反应的产物不同（2025浙江Q7D）
  • ⑥ 混淆水解与电离方程式 — CO₃²⁻的水解≠电离（2025浙江Q7A）

🧭 五步检验法

第1步：检查物质拆分
  • 强酸、强碱、可溶盐 → 拆成离子（NaOH→Na⁺+OH⁻，NaCl→Na⁺+Cl⁻）
  • 弱酸、弱碱、水、气体、沉淀、氧化物、单质 → 不拆（保留化学式）
  • 易错：Na₂O₂、Ca(OH)₂（石灰乳）不拆
第2步：检查原子守恒

左右两边各元素原子数相等

第3步：检查电荷守恒

左边离子所带正负电荷总数 = 右边离子所带正负电荷总数（氧化还原题必检！）

第4步：检查得失电子守恒（氧化还原型）

化合价升高的总价数 = 化合价降低的总价数

第5步：检查反应合理性
  • 过量/少量是否正确区分
  • 产物是否符合实际（如Fe²⁺不能被HNO₃还原为Fe）
⚠️ 高频陷阱
  • 陷阱1：弱酸写成离子 — CH₃COOH是最常见的"被误拆"弱酸
  • 陷阱2：忽略"少量"条件 — CO₂少量通入苯酚钠溶液只生成NaHCO₃，不生成Na₂CO₃
  • 陷阱3：Na₂O₂写成Na⁺+O₂²⁻ — Na₂O₂是过氧化物，离子方程式中不能拆分（应保留化学式）
  • 陷阱4：漏写↑↓ — 产生气体或沉淀时必须标注

📝 真题精练

2022年广东 · 第14题 钠的化合物 · 离子方程式正误
14. 下列关于Na的化合物之间转化反应的离子方程式书写正确的是
A. 碱转化为酸式盐（NaOH + CO₂）：NaOH溶液中通入足量CO₂ ❌ CO₂ + 2OH⁻ = CO₃²⁻ + H₂O ✅ CO₂ + OH⁻ = HCO₃⁻
B. 碱转化为两种盐（Cl₂ + NaOH）： ✅ Cl₂ + 2OH⁻ = Cl⁻ + ClO⁻ + H₂O
C. 过氧化物转化为碱（Na₂O₂ + H₂O）： ❌ 2Na⁺ + 2O₂²⁻ + 2H₂O = 4Na⁺ + 4OH⁻ + O₂↑ ✅ Na₂O₂不能拆！应写：2Na₂O₂ + 2H₂O = 4Na⁺ + 4OH⁻ + O₂↑
D. 盐转化为另一种盐（Na₂SiO₃ + H⁺）： ❌ Na₂SiO₃是可溶盐，离子方程式中应拆成Na⁺和SiO₃²⁻
✅ 答案：B

A错误：向NaOH溶液中通入足量CO₂，碱转化为酸式盐NaHCO₃，离子方程式应为CO₂ + OH⁻ = HCO₃⁻。原式写的是CO₂ + 2OH⁻ = CO₃²⁻ + H₂O，这是CO₂不足量时的产物。

B正确：Cl₂通入NaOH溶液中生成NaCl和NaClO两种盐，离子方程式为Cl₂ + 2OH⁻ = Cl⁻ + ClO⁻ + H₂O。原子守恒、电荷守恒均满足。

C错误：Na₂O₂是过氧化物，在离子方程式中不能拆成Na⁺和O₂²⁻（O₂²⁻不是溶液中真实存在的离子形式）。应保留化学式Na₂O₂。

D错误：Na₂SiO₃是可溶性盐，在离子方程式中应拆成Na⁺和SiO₃²⁻，不能保留化学式。

⚠️ 陷阱分析

陷阱1（选项A）："足量"和"不足量"导致产物不同。CO₂不足→CO₃²⁻；CO₂足量→HCO₃⁻。题目明确说"碱转化为酸式盐"，所以产物必须是HCO₃⁻。

陷阱2（选项C）：Na₂O₂的拆分是经典陷阱。Na₂O₂中虽有Na⁺，但O₂²⁻不能单独存在（不同于Na₂O中的O²⁻），因此整体不拆。类似不拆的还有：Ca(OH)₂（石灰乳/浊液）、Cu、Fe等。

陷阱3（选项D）：与选项C相反——可溶盐应拆。Na₂SiO₃是强电解质且可溶，必须拆成Na⁺和SiO₃²⁻。

💡 避坑指导

核心口诀："强拆弱不拆，溶拆不溶不拆"。Na₂O₂特殊——虽然含Na⁺，但整体不拆。做题时先判断每种物质能否拆，再检查守恒。

速记：Na₂O₂ + H₂O的方程式是固定考法，记住不拆即可秒杀。

2025年广西 · 第8题 含硫物质转化 · 离子方程式正误
8. 不同含硫物质间可相互转化。下列离子方程式书写正确的是
A. 反应①（S²⁻ + CH₃COOH → HS⁻）： ❌ S²⁻ + 2CH₃COOH = H₂S↑ + 2CH₃COO⁻ ✅ S²⁻ + CH₃COOH = HS⁻ + CH₃COO⁻
B. 反应②（Al³⁺ + HS⁻ → ）： ❌ Al³⁺ + 3HS⁻ = Al(OH)₃↓ + H₂S↑（漏了H₂O） ✅ Al³⁺ + 3HS⁻ + 3H₂O = Al(OH)₃↓ + 3H₂S↑
C. 反应③（S₂O₃²⁻ 酸性歧化 → S + SO₂）： ✅ S₂O₃²⁻ + 2H⁺ = S↓ + SO₂↑ + H₂O
D. 反应④（SO₂ + MnO₄⁻ + H⁺ → ）： ❌ SO₂ + 2MnO₄⁻ + 2H₂O = SO₄²⁻ + 2Mn²⁺ + 4OH⁻ ✅ 5SO₂ + 2MnO₄⁻ + 2H₂O = 5SO₄²⁻ + 2Mn²⁺ + 4H⁺
✅ 答案：C

A错误：CH₃COOH是弱酸，离子方程式中应保留化学式（不拆）。正确写法为S²⁻ + CH₃COOH = HS⁻ + CH₃COO⁻。

B错误：Al³⁺与HS⁻发生双水解反应，生成Al(OH)₃沉淀和H₂S气体，但漏写了H₂O。正确方程式为Al³⁺ + 3HS⁻ + 3H₂O = Al(OH)₃↓ + 3H₂S↑。

C正确：S₂O₃²⁻在酸性条件下发生歧化反应，S从+2价分别变为0价（S）和+4价（SO₂），离子方程式S₂O₃²⁻ + 2H⁺ = S↓ + SO₂↑ + H₂O满足原子守恒、电荷守恒和得失电子守恒。

D错误：SO₂被KMnO₄氧化时，S从+4价升至+6价（升2），Mn从+7价降至+2价（降5）。得失电子守恒要求SO₂:KMnO₄ = 5:2，原式系数比错误。

⚠️ 陷阱分析

陷阱1（选项A）：S²⁻是强碱弱酸盐的酸根，与弱酸CH₃COOH反应生成弱酸H₂S是正常反应。但CH₃COOH不能拆——弱酸保留化学式是铁律。

陷阱2（选项D）：氧化还原反应配平的核心是得失电子守恒。S(+4→+6)升2，Mn(+7→+2)降5，最小公倍数10，所以SO₂系数=10/2=5，MnO₄⁻系数=10/5=2。

💡 避坑指导

配平技巧：看到氧化还原离子方程式，第一步标化合价，第二步算电子转移，第三步根据最小公倍数定系数，第四步用H⁺/OH⁻/H₂O配平H和O。

复习提示：S₂O₃²⁻ + 2H⁺的歧化反应是常考反应，产物是S↓和SO₂↑，记住直接秒选。

2025年浙江 · 第7题 基础离子方程式 · 水解/电离/氧化还原
7. 下列离子方程式正确的是
A. CO₃²⁻的水解反应： ❌ CO₃²⁻ + 2H₂O ⇌ H₂CO₃ + 2OH⁻（原式混淆了水解与电离） ✅ CO₃²⁻ + H₂O ⇌ HCO₃⁻ + OH⁻（第一步水解）
B. Fe与醋酸反应： ❌ Fe + 2CH₃COO⁻ = Fe²⁺ + H₂↑ + 2CH₃COO⁻ ✅ Fe + 2CH₃COOH = Fe²⁺ + H₂↑ + 2CH₃COO⁻（CH₃COOH不拆）
C. FeO与稀硝酸反应： ❌ FeO + 2H⁺ = Fe²⁺ + H₂O ✅ 3FeO + 10H⁺ + NO₃⁻ = 3Fe³⁺ + NO↑ + 5H₂O
D. 苯酚钠溶液中通入少量CO₂： ✅ C₆H₅O⁻ + CO₂ + H₂O → C₆H₅OH + HCO₃⁻
✅ 答案：D

A错误：CO₃²⁻的水解方程式为CO₃²⁻ + H₂O ⇌ HCO₃⁻ + OH⁻（第一步）。原式写的是电离方程式（CO₃²⁻ ⇌ CO₂ + O²⁻之类的错误形式），混淆了水解和电离。

B错误：CH₃COOH是弱酸，在离子方程式中应保留化学式。正确为Fe + 2CH₃COOH = Fe²⁺ + H₂↑ + 2CH₃COO⁻。

C错误：稀HNO₃具有强氧化性，将FeO中的Fe²⁺氧化为Fe³⁺。正确方程式为3FeO + 10H⁺ + NO₃⁻ = 3Fe³⁺ + NO↑ + 5H₂O。

D正确：苯酚的酸性弱于H₂CO₃但强于HCO₃⁻，因此苯酚钠与少量CO₂反应生成苯酚和NaHCO₃（而非Na₂CO₃）。离子方程式为C₆H₅O⁻ + CO₂ + H₂O → C₆H₅OH + HCO₃⁻。

⚠️ 陷阱分析

陷阱1（选项A）：水解方程式的书写格式：弱酸根 + H₂O ⇌ 弱酸 + OH⁻。CO₃²⁻的第一步水解产物是HCO₃⁻（不是H₂CO₃）。

陷阱2（选项C）：HNO₃ + Fe²⁺是经典氧化还原组合，HNO₃会将Fe²⁺氧化为Fe³⁻。看到HNO₃就想到"具有强氧化性"，这是必考点。

陷阱3（选项D）：酸性强弱顺序 H₂CO₃ > C₆H₅OH > HCO₃⁻。"强酸制弱酸"原则决定了CO₂与苯酚钠反应只能生成HCO₃⁻，不能继续生成CO₃²⁻。

💡 避坑指导

酸性顺序记忆：H₂CO₃ > C₆H₅OH > HCO₃⁻。遇到CO₂与酚钠/苯酚钠反应，产物一定是NaHCO₃而非Na₂CO₃。

HNO₃的铁律：HNO₃参与的反应，优先考虑氧化性，Fe²⁺→Fe³⁺是标准考法。

2025年河南 · 第5题 溶液中离子反应 · 弱电解质 · 氧化还原
5. 对于下列过程中发生的化学反应，相应离子方程式正确的是
A. 磷酸二氢钠水解： ❌ H₂PO₄⁻ + H₂O ⇌ H₃PO₄ + OH⁻（水解应生成HPO₄²⁻+OH⁻）
B. 用稀盐酸浸泡氧化银： ✅ Ag₂O + 2H⁺ = 2Ag⁺ + H₂O
C. 向次氯酸钠溶液中加入碘化氢溶液： ❌ ClO⁻ + 2HI = Cl⁻ + I₂ + H₂O ✅ ClO⁻ + 2H⁺ + 2I⁻ = Cl⁻ + I₂ + H₂O（HI是强酸，应拆成H⁺+I⁻）
D. 向硫酸氢钠溶液中滴加少量碳酸氢钡溶液： ❌ HCO₃⁻ + H⁺ = CO₂↑ + H₂O（忽略了Ba²⁺与SO₄²⁻的沉淀反应） ✅ Ba²⁺ + 2HCO₃⁻ + 2H⁺ + SO₄²⁻ = BaSO₄↓ + 2CO₂↑ + 2H₂O
✅ 答案：B

A错误：H₂PO₄⁻是酸式盐的酸根，其水解方程式为H₂PO₄⁻ + H₂O ⇌ H₃PO₄ + OH⁻（这是电离式方向的写法错误），正确的水解应为H₂PO₄⁻ + H₂O ⇌ HPO₄²⁻ + H₃O⁺（酸式根以电离为主），或H₂PO₄⁻发生第二步水解的写法需要根据实际题目。

B正确：Ag₂O是碱性氧化物，与稀盐酸反应的离子方程式为Ag₂O + 2H⁺ = 2Ag⁺ + H₂O。原子守恒、电荷守恒均正确。

C错误：HI是强酸，在离子方程式中应拆成H⁺和I⁻。正确方程式为ClO⁻ + 2H⁺ + 2I⁻ = Cl⁻ + I₂ + H₂O。

D错误：少量Ba(HCO₃)₂滴入NaHSO₄溶液中，HCO₃⁻与H⁺反应产生CO₂，同时Ba²⁻与SO₄²⁻反应产生BaSO₄沉淀。由于Ba(HCO₃)₂少量，H⁺和SO₄²⁻均过量，反应中应体现两个反应的同时发生。

⚠️ 陷阱分析

陷阱1（选项A）：H₂PO₄⁻是两性离子，既能水解又能电离。需要根据题目情境判断主要方向。题干说"水解"，但H₂PO₄⁻的第一步主要以电离为主（H₂PO₄⁻ ⇌ H⁺ + HPO₄²⁻）。

陷阱2（选项C）：HI是强酸（氢卤酸中HF是弱酸，其余均为强酸），必须拆。但HI具有还原性，会被ClO⁻氧化。

陷阱3（选项D）："少量"Ba(HCO₃)₂意味着NaHSO₄过量，需要考虑SO₄²⁻与Ba²⁺的沉淀反应。不能只写HCO₃⁻ + H⁺ = CO₂↑ + H₂O。

💡 避坑指导

强酸判断口诀："除硅氟外全强酸"——HCl、HBr、HI、HNO₃、H₂SO₄、HClO₄是强酸，HF和H₂SiO₃是弱酸。有机酸（CH₃COOH等）一律弱酸。

过量/少量判断：反应物不过量的那个写化学式，过量的拆成离子。做题时先判断谁过量，再决定拆不拆。
