⚗️ 第9类 · 水溶液离子平衡

弱电解质的电离 · 盐类水解 · 配离子平衡 · 电荷守恒

📊 规律总结

广东卷水溶液离子平衡在选择题中出现较少，但考查深度高。近5年（2021-2025）仅出现2道纯水溶液离子平衡选择题，但相关知识点（水解、电荷守恒、pH）频繁出现在其他题型中。

三大命题模式

  • 模式1：弱碱盐的水解 — 给定有机弱碱盐酸盐，分析pH、稀释效应、电离方程式、电荷守恒（2021 Q8）
  • 模式2：配离子平衡 + 电荷守恒 — 配离子在溶液中的稀释效应、平衡移动、电荷守恒书写（2025 Q12）
  • 模式3（混合考法）： Ksp计算、滴定终点判断 — 虽不单独成题，但在NA题和实验题中频繁涉及

🧭 三步解题法

第1步：识别体系

快速判断溶液类型：

  • 强酸弱碱盐（如GHCl、NH₄Cl）→ 弱碱阳离子水解，溶液呈酸性
  • 强碱弱酸盐（如Na₂CO₃、NaAc）→ 弱酸根水解，溶液呈碱性
  • 配离子体系（如[CuCl₂]⁻、[CuCl₃]²⁻）→ 存在配位平衡
第2步：分析平衡移动

判断操作（加水、加盐、加热）对各平衡的影响方向和程度：

  • 加水稀释 → 促进水解/电离，但离子浓度整体下降
  • 加同离子 → 抑制水解/电离，平衡逆向移动
  • 加热 → 促进水解（水解吸热），但注意气体逸出等因素
第3步：检验守恒关系

遇到离子浓度关系式，必须逐一核对：

  • 电荷守恒：所有阳离子正电荷 = 所有阴离子负电荷（别忘了配离子的电荷数！）
  • 物料守恒：某元素所有存在形式之和为定值
  • 质子守恒：质子得失平衡（可由电荷+物料推导）

⚠️ 四大高频陷阱

陷阱1：稀释时离子浓度 vs 水解程度

加水稀释：水解程度增大 ≠ 离子浓度增大！水解程度增大是百分比概念，但溶液体积增大更多，c(H⁺)实际减小，pH升高（2021 Q8-B）。

陷阱2：电离方程式写法

盐的电离：GHCl = GH⁺ + Cl⁻ ✓ | GHCl = G + H⁺ + Cl⁻ ✗
H[CuCl₂] = H⁺ + [CuCl₂]⁻ ✓ | H[CuCl₂] = H⁺ + Cu⁺ + 2Cl⁻ ✗
规律：盐电离时配离子保持完整，不能拆开写（2021 Q8-C, 2025 Q12-C）。

陷阱3：电荷守恒漏系数

配离子的电荷数必须乘系数！[CuCl₃]²⁻带2个负电荷，电荷守恒中为2c([CuCl₃]²⁻)，很多同学忘记乘2（2025 Q12-D）。

陷阱4：平衡移动 ≠ 成分变化

"加入NaCl → [CuCl₂]⁻浓度一定上升"是错误的！虽然Cl⁻增多有利于生成[CuCl₂]⁻，但也可能促进[CuCl₂]⁻进一步转化为[CuCl₃]²⁻，最终[CuCl₂]⁻不一定上升（2025 Q12-B）。

📝 真题精讲

📌 2021年 · 第8题 分值：2分 | 答案：B
8. 鸟嘌呤(G)是一种有机弱碱，可与盐酸反应生成盐酸盐(用GHCl表示)。已知GHCl水溶液呈酸性，下列叙述正确的是
A. 0.001 mol·L⁻¹ GHCl水溶液的pH = 3
B. GHCl水溶液加水稀释，pH升高
C. GHCl在水中的电离方程式为：GHCl ＝ G ＋ H⁺ ＋ Cl⁻
D. GHCl水溶液中：c(OH⁻) ＋ c(Cl⁻) ＝ c(H⁺) ＋ c(GH⁺)
📋 答案与解析

正确答案：B

A ✗ GHCl为强酸弱碱盐，GH⁺发生水解：GH⁺ ＋ H₂O ⇌ G ＋ H₃O⁺。弱离子的水解较为微弱，0.001 mol/L GHCl溶液中c(H⁺) ＜ 0.001 mol/L，因此pH ＞3，而非pH=3。

B ✓ 加水稀释GHCl溶液时，GH⁺水解程度增大（勒夏特列原理），但溶液总体积增大更多，c(H⁺)总体减小，pH升高。

C ✗ GHCl是强酸弱碱盐，电离方程式为：GHCl ＝ GH⁺ ＋ Cl⁻。GHCl不会分解为G＋H⁺＋Cl⁻。

D ✗ GHCl溶液中电荷守恒应为：c(H⁺) ＋ c(GH⁺) ＝ c(OH⁻) ＋ c(Cl⁻)。选项将c(H⁺)和c(GH⁺)写到了等号右侧，实质相等但选项是将OH⁻和Cl⁻放到一边，H⁺和GH⁺放到另一边——这是对的！等号两边交换即可。但真正的问题是：D选项写的是 c(OH⁻)＋c(Cl⁻)＝c(H⁺)＋c(GH⁺)，这与标准形式只是符号交换。
实际上，D选项本身在数学上是正确的（只是左右写法不同），但广东卷官方答案判定为错误。仔细看：c(H⁺)＋c(GH⁺)＝c(OH⁻)＋c(Cl⁻) 与 c(OH⁻)＋c(Cl⁻)＝c(H⁺)＋c(GH⁺) 完全等价！除非题目选项中有别的细微差异（如缺项或符号问题）。原解析说D错误，推测是选项中的化学式写法或顺序有特殊约定。

⚠️ 强哥注：原卷选项D含有WMF嵌入公式，可能实际书写与提取文字有差异。以官方答案B为准。

⚠️ 常见陷阱分析

陷阱A → "0.001 mol/L 一元强酸盐 → pH=3"的错误直觉。忘了弱碱阳离子水解，c(H⁺)小于理论值。

陷阱B → "稀释促进水解 → 酸性增强 → pH降低"的思维惯性。稀释确实促进水解百分比，但实际c(H⁺)因体积增大而下降！这是最经典的易错点。

陷阱C → 把"盐的电离"和"酸的逐级电离"混淆。盐电离是一步完成的，配离子不拆。

✅ 避坑指导

1️⃣ 遇到弱碱盐/弱酸盐，第一时间写水解方程式，别凭直觉判断pH。

2️⃣ 稀释对离子浓度的判断口诀："水解百分比升，离子浓度降"。

3️⃣ 盐的电离永远是一步到位的离子分离，绝不拆配离子。

4️⃣ 离子浓度关系题 = 电荷守恒 + 物料守恒 + 质子守恒，三剑合璧。

📌 2025年 · 第12题 分值：2分 | 答案：A
12. CuCl微溶于水，但在Cl⁻浓度较高的溶液中因形成[CuCl₂]⁻和[CuCl₃]²⁻而溶解。将适量CuCl完全溶于盐酸，得到含[CuCl₂]⁻和[CuCl₃]²⁻的溶液，下列叙述正确的是
A. 加水稀释，[CuCl₃]²⁻浓度一定下降
B. 向溶液中加入少量NaCl固体，[CuCl₂]⁻浓度一定上升
C. H[CuCl₂]的电离方程式为：H[CuCl₂] ＝ H⁺ ＋ Cu⁺ ＋ 2Cl⁻
D. 体系中，c(Cu⁺) ＋ c(H⁺) ＝ c(Cl⁻) ＋ c([CuCl₂]⁻) ＋ c([CuCl₃]²⁻) ＋ c(OH⁻)
📋 答案与解析

正确答案：A

A ✓ 加水稀释，溶液体积增大，[CuCl₃]²⁻的物质的量浓度 = n/V，V增大 → 浓度一定下降。注意"一定"是因为体积的变化是确定的。

B ✗ 加入NaCl增加Cl⁻浓度，体系中存在平衡：
Cu⁺ ＋ 2Cl⁻ ⇌ [CuCl₂]⁻
[CuCl₂]⁻ ＋ Cl⁻ ⇌ [CuCl₃]²⁻
Cl⁻增多，可能促进[CuCl₂]⁻进一步转化为[CuCl₃]²⁻，所以[CuCl₂]⁻浓度不一定上升。

C ✗ H[CuCl₂]应电离为：H[CuCl₂] ＝ H⁺ ＋ [CuCl₂]⁻。配离子[CuCl₂]⁻是一个整体，不能拆成Cu⁺＋2Cl⁻。

D ✗ 正确的电荷守恒应为：
c(Cu⁺) ＋ c(H⁺) ＝ c(Cl⁻) ＋ c([CuCl₂]⁻) ＋ 2c([CuCl₃]²⁻) ＋ c(OH⁻)
[CuCl₃]²⁻带2个单位负电荷，必须乘系数2！选项中漏掉了系数。

⚠️ 常见陷阱分析

陷阱A → 看似"简单常识"，但考生可能怀疑"一定"二字而不敢选。加水稀释浓度下降是物理必然，无需犹豫。

陷阱B → "加Cl⁻→[CuCl₂]⁻升"的一厢情愿。忽视了后续平衡：[CuCl₂]⁻可以再结合Cl⁻变成[CuCl₃]²⁻！这是连续配位平衡的典型陷阱。

陷阱C → 和2021 Q8 C选项一脉相承——配离子电离不拆！H[CuCl₂]是配酸，电离出H⁺和完整的[CuCl₂]⁻。

陷阱D → 电荷守恒最常见的错误：忘记配离子电荷数的系数。2-配离子要乘2，3-配离子要乘3。

✅ 避坑指导

1️⃣ 稀释问题：体积增大 → 浓度一定下降，这是物理定律级别的确定。

2️⃣ 连续平衡体系的"加料"问题：不要只看最近的平衡，要考虑所有串联平衡的净效应。

3️⃣ 配离子电离：配离子 = 一个整体，电离时配离子保持完整。

4️⃣ 电荷守恒书写三步骤：①列出所有离子 ②标注每个离子的电荷数 ③每个离子浓度 × 电荷数绝对值 = 另一侧同理。

📖 复习建议

  • 核心概念：水解平衡（弱碱阳离子/弱酸根）、配位平衡、电荷守恒、物料守恒
  • 必背公式：电荷守恒：Σc(阳离子)×电荷 = Σc(阴离子)×|电荷|
    注：配离子的电荷数不可忽略！
  • 经典对比：2021 Q8（弱碱盐水解）与 2025 Q12（配离子平衡）虽然体系不同，但都考查了电离方程式写法和稀释效应——这是跨越两种题型的核心考点
  • 延伸关注：Ksp计算、滴定终点判断虽不单独MC出题，但在NA题（NH₄⁺水解减少粒子数）和实验大题中频繁出现，需一并掌握
