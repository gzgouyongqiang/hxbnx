🔥 第12类 · 热化学与反应历程

焓变 · 盖斯定律 · 活化能 · 反应历程图 · 催化剂与能量

📊 规律总结

广东卷热化学与反应历程选择题在近5年（2021-2025）中呈现两种命题模式：

两大命题模式

  • 模式1：反应历程图分析 — 给出能量-反应进程图或势能图，判断反应热（吸热/放热）、活化能大小、催化剂效果（2022 Q15、2023 Q15、2024 Q15）
  • 模式2：焓变与实验现象关联 — 通过实验现象（如烧瓶壁变冷）判断反应的焓变（吸热/放热），结合气体压强变化进行综合推理（2024 Q11）

命题趋势

  • 反应历程图题几乎每年必考（2022-2024连续三年出现）
  • 近年新增催化剂对比（两种催化剂I和II的反应历程图对照）
  • 2024年首次将焓变判断融入实验情境（Ba(OH)₂·8H₂O + NH₄Cl经典实验）
  • 题目难度中等偏上，图像分析能力是关键

🧭 三步解题法

第1步：识别反应热类型

快速判断吸热/放热：

  • 反应历程图：产物能量高于反应物 → 吸热；产物能量低于反应物 → 放热
  • 浓度变化：升温后产物浓度增大 → 放热（勒夏特列原理）
  • 实验现象：烧瓶壁变冷 → 吸热；温度升高 → 放热
第2步：读图抓关键数据

反应历程图中的核心信息：

  • 波峰数 = 反应步骤数（每个波峰代表一步过渡态）
  • 最高波峰 = 决速步（活化能最大的步骤控制速率）
  • 实线 vs 虚线：有催化剂的活化能更低，波峰更矮
第3步：排除法快速锁定

常见选项设坑及排除策略：

  • ❌ "无催化剂时反应不能进行" → 催化剂只改变速率，不改变能否反应
  • ❌ 混淆"活化能更低"与"催化效果更好" → 效果好 = 速率快 = 活化能低
  • ❌ "催化剂改变焓变" → 催化剂只改变活化能，不改变化学热力学参数
⚠️ 高频陷阱
  • 陷阱1：混淆催化剂I和II的活化能大小关系 — 速率快的催化剂活化能更低，不是更高
  • 陷阱2：忽略中间产物的积累 — 前期活化能低则中间产物生成快，后期活化能高则消耗慢，最高浓度取决于前后速率的平衡
  • 陷阱3：错误判断放热/吸热 — 不能只看一个方向，要整体看反应物和产物的能量高低
  • 陷阱4：忽略实验中的多重因素 — 吸热导致温度降低（气球缩小），但分解产生气体（气球增大），需综合判断

📝 真题精练

2022年 · 第15题 反应速率 · 催化剂 · 活化能
15. 在相同条件下研究催化剂Ⅰ、Ⅱ对反应的影响，各物质浓度c随反应时间t的部分变化曲线如图，则
A. 无催化剂时，反应不能进行
B. 与催化剂Ⅰ相比，Ⅱ使反应活化能更低
C. a曲线表示使用催化剂Ⅱ时X的浓度随t的变化
D. 使用催化剂Ⅰ时，v(X)=1.0 mol·L⁻¹·min⁻¹
✅ 答案：D

A错误：由图可知，无催化剂时，随反应进行，生成物浓度也在增加，说明反应也在进行，只是速率较慢。催化剂只改变反应速率，不改变反应能否发生。

B错误：由图可知，催化剂Ⅰ比催化剂Ⅱ催化效果好（达到平衡更快），说明催化剂Ⅰ使反应活化能更低，反应更快。催化效果更好 = 活化能更低。

C错误：使用催化剂Ⅱ时，在0~2 min内Y的浓度变化了2.0 mol/L，而a曲线表示的X的浓度变化了2.0 mol/L，二者变化量之比不等于化学计量数之比（应满足 ΔX:ΔY = 化学计量数比），所以a曲线不表示使用催化剂Ⅱ时X的浓度变化。

D正确：使用催化剂Ⅰ时，在0~2 min内，Y的浓度变化了4.0 mol/L，则 v(Y)=4.0/2=2.0 mol·L⁻¹·min⁻¹。由化学计量数关系 v(X) = ½v(Y) = 1.0 mol·L⁻¹·min⁻¹。

⚠️ 陷阱分析

陷阱1（选项A）："无催化剂就不能反应"是经典错误。催化剂降低活化能、加快反应速率，但不改变反应的自发性。没有催化剂，反应依然可以进行，只是速率慢。

陷阱2（选项B）：容易把"活化能更低"和"催化效果更好"的方向搞反。活化能越低 → 速率越快 → 催化效果越好。图中催化剂Ⅰ使反应更快达到平衡，说明Ⅰ的活化能更低。

陷阱3（选项C）：需要通过计量数之比来验证曲线归属。不同物质的浓度变化速率之比 = 化学计量数之比，如果不满足则该曲线对应关系错误。

💡 避坑指导

核心方法：读图时先判断三条曲线分别对应什么条件（无催化剂/催化剂Ⅰ/催化剂Ⅱ），然后通过计量数比验证。记住：催化效果越好 = 活化能越低 = 速率越快。

速算技巧：看到"v(X)=?"直接找Y的变化量和时间，再用计量数比折算。

2023年 · 第15题 反应历程图 · 催化剂对比 · 活化能
15. 催化剂Ⅰ和Ⅱ均能催化反应。反应历程（下图）中，M为中间产物。其它条件相同时，下列说法不正确的是
A. 使用Ⅰ和Ⅱ，反应历程都分4步进行
B. 反应达平衡时，升高温度，R的浓度增大
C. 使用Ⅱ时，反应体系更快达到平衡
D. 使用Ⅰ时，反应过程中M所能达到的最高浓度更大
✅ 答案：C（选不正确的）

A正确：由图可知，两种催化剂均出现四个波峰，所以使用Ⅰ和Ⅱ，反应历程都分4步进行。

B正确：由图可知该反应是放热反应（产物能量低于反应物），所以达平衡时，升高温度平衡逆向移动（向左），R的浓度增大。

C错误：由图可知Ⅰ的最高活化能小于Ⅱ的最高活化能，所以使用Ⅰ时反应速率更快，反应体系更快达到平衡。C说"使用Ⅱ更快"，是反的。

D正确：由图可知在前两个历程中使用Ⅰ活化能较低（反应速率较快，M生成快），后两个历程中使用Ⅰ活化能较高（反应速率较慢，M消耗慢）。生成快+消耗慢 → M积累更多，最高浓度更大。

⚠️ 陷阱分析

陷阱1（选项B）：需要先判断反应是放热还是吸热。图中左侧能量高、右侧能量低，说明正反应放热。升温后平衡左移，R（反应物）浓度增大。

陷阱2（选项C）：关键在于准确比较两种催化剂的最高活化能。最高波峰对应决速步（速率控制步骤），波峰越高的催化剂，决速步活化能越大，整体速率越慢。

陷阱3（选项D）：中间产物M的最高浓度取决于"生成速率"和"消耗速率"的竞争。需要综合分析前半段（M的生成）和后半段（M的消耗）的活化能差异。

💡 避坑指导

核心方法：看到双催化剂反应历程图，分三步分析：① 波峰数 = 步骤数；② 最高波峰对比 → 谁的决速步活化能低 → 谁更快；③ 中间产物浓度看"前快后慢"还是"前慢后快"。

判断放热/吸热：看反应历程图首尾能量差，始端 > 末端 = 放热。

2024年 · 第11题 焓变判断 · 吸热反应 · 实验现象推理
11. 按下图装置进行实验。搅拌一段时间后，滴加浓盐酸。不同反应阶段的预期现象及其相应推理均合理的是
A. 烧瓶壁会变冷，说明存在放热反应
B. 试纸会变蓝，说明有NH₃生成，产氨过程熵增
C. 滴加浓盐酸后，有白烟产生，说明有NH₄Cl升华
D. 实验过程中，气球会一直变大，说明体系压强增大
✅ 答案：B

A错误：烧瓶壁变冷说明Ba(OH)₂·8H₂O与NH₄Cl的反应是焓变大于0的吸热反应（ΔH > 0），而非放热反应。选项推理方向搞反了。

B正确：湿润的红色石蕊试纸变蓝，说明有NH₃生成。该反应Ba(OH)₂·8H₂O(s) + 2NH₄Cl(s) = BaCl₂(aq) + 2NH₃↑ + 8H₂O(l)中，固体反应物生成气体和液体产物，混乱度增大，确实熵增（ΔS > 0）。

C错误：滴加浓盐酸后，有白烟产生是因为浓盐酸挥发出的HCl气体与反应生成的NH₃气体反应生成了NH₄Cl固体小颗粒，不是NH₄Cl的升华。

D错误：实验过程中气球的变化不是"一直变大"。吸热反应使温度降低 → 气压减小 → 气球缩小；NH₃生成使气压增大 → 气球变大；加盐酸后NH₃被消耗 → 气压减小 → 气球变小。正确顺序是：先缩小 → 后增大 → 再缩小。

⚠️ 陷阱分析

陷阱1（选项A）："变冷=放热"是常见的逻辑倒置错误。环境（烧瓶壁）变冷 → 反应从环境吸收热量 → 反应是吸热的！

陷阱2（选项C）：白烟不等于升华。NH₄Cl确实可以受热分解再化合（类似"升华"的假象），但本题的白烟是HCl + NH₃ → NH₄Cl的气相反应产物，与升华无关。

陷阱3（选项D）："一直变大"忽略了吸热降温的效应。本题涉及三个竞争因素：① 吸热降温（缩小）② 产气（增大）③ 加酸消耗气体（缩小），需要分阶段分析。

💡 避坑指导

核心方法：经典实验题先写出化学方程式，再逐选项验证。Ba(OH)₂·8H₂O + 2NH₄Cl = BaCl₂ + 2NH₃↑ + 10H₂O 是高中最经典的吸热反应之一，与浓H₂SO₄稀释（放热）形成对比。

气球变化分析：分三阶段：反应初期（吸热降温为主→缩小）、反应后期（产气为主→增大）、加盐酸后（气体被消耗→缩小）。

2024年 · 第15题 反应历程示意图 · 放热/吸热判断 · 催化剂效果
15. 对反应S ⇌ I ⇌ R（I为中间产物），相同条件下：①加入催化剂，反应达到平衡所需时间大幅缩短；②提高反应温度，[R]增大，[S]减小。基于以上事实，可能的反应历程示意图（——为无催化剂，- - - -为有催化剂）为
A
B
C
D
✅ 答案：A

第一步：判断总反应方向。提高温度后[R]增大、[S]减小，说明平衡逆向移动（向S的方向移动），即正反应为放热反应。由此排除C、D（C、D中产物能量高于反应物，为吸热反应）。

第二步：判断S→I步骤的热效应。[S]减小说明S→I的正反应被促进（升温后平衡正向移动），即S→I属于吸热反应。在反应历程图中，I的能量应高于S（吸热需要能量升高）。

第三步：判断催化剂效果。加入催化剂后反应达到平衡的时间大幅缩短，说明决速步的活化能下降。决速步是活化能最大的步骤，有催化剂时该步的波峰应更矮。

综合以上三条：A选项满足：① 总反应放热（R能量低于S）；② S→I能量升高（吸热）；③ 有催化剂时最高波峰降低。

⚠️ 陷阱分析

陷阱1：误以为"[R]增大=正反应为吸热"。注意在本题S ⇌ I ⇌ R中，升温后[R]增大、[S]减小，说明平衡向S的方向移动，即正反应是放热的。

陷阱2：忽略了中间产物I的步骤方向。S→I是吸热的（升温促进），I→R是放热的（总反应放热）。两个步骤热效应方向相反。

陷阱3：催化剂不只降低一个步骤的活化能，而是降低所有步骤的活化能。但题目说"大幅缩短"，强调的是决速步（最高波峰）的活化能下降幅度最大。

💡 避坑指导

核心方法：文字条件 + 图像对照法。先将文字条件转化为明确的判断：① 总反应放热 → 排除吸热图；② S→I吸热 → I能高于S；③ 催化剂降低活化能 → 有催化波峰更矮。三条全满足的唯一答案就是正确的。

复习提示：勒夏特列原理判断平衡移动方向是热化学与反应历程题的必备工具。看到温度变化对浓度的影响，第一步永远是判断正反应是放热还是吸热。
