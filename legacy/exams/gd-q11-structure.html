🔬 第11类 · 物质结构与性质

共价键 · 大π键 · 杂化方式 · VSEPR · 晶体 · 结构-性质关系

📊 规律总结

广东卷物质结构与性质选择题在近5年（2021-2025）中出现频率较低，纯结构题仅2025年Q2和Q13两道。但结构概念频繁渗透在其他题型中（有机题考查杂化、元素推断考查VSEPR/氢键）。

四大命题角度

  • 角度1：化学键基本判断 — 共价键/离子键/金属键的识别、σ键/π键/大π键辨别（2025 Q2）
  • 角度2：杂化与分子构型 — sp/sp²/sp³ 杂化判断、VSEPR模型、分子共平面（2024 Q9、2025 Q13）
  • 角度3：结构与性质关系 — 由结构能否推测性质（2025 Q13的逆问法）
  • 角度4：物质状态与变化 — 物理变化vs化学变化中的键断裂、晶体水、胶体（2023 Q5）

🧭 三步解题法

第1步：识别键型

快速判断题目涉及的键型层级：

  • 分子内：共价键（σ键/π键/大π键）→ 决定化学性质
  • 分子间：氢键/范德华力 → 决定物理性质（熔沸点、溶解度）
  • 晶体中：离子键/金属键/共价键（原子晶体）→ 决定晶体类型和性质
第2步：判定杂化与构型

由中心原子价层电子对数 → 杂化方式 → 空间构型：

  • 价层电子对=2 → sp杂化 → 直线形（如CO₂、乙炔C）
  • 价层电子对=3 → sp²杂化 → 平面三角形（如SO₃、苯C）
  • 价层电子对=4 → sp³杂化 → 四面体（如CH₄、饱和C）
第3步：结构→性质推导

建立结构特征→化学性质的因果链：

  • π键存在 → 可发生加成、加聚反应
  • 孤对电子存在 → 可形成配位键、氢键
  • 离域电子 → 导电性（石墨）、芳香性（苯）
  • ⚠️ 不是所有性质都能由结构推测：氧化性取决于化合价（如SO₃的氧化性来自S(+6)而非VSEPR构型）

⚠️ 四大高频陷阱

陷阱1：物理变化中"共价键断裂"

干冰升华、冰融化、碘升华 = 分子间作用力被破坏，共价键不断裂！只有化学反应才断裂共价键（2023 Q5-A）。

陷阱2：sp³杂化≠平面结构

sp³杂化的碳具有四面体结构，含sp³碳的环系不可能所有原子共平面（2024 Q9-C）。苯环中所有碳为sp²，才是平面。

陷阱3：大π键≠多个双键

苯环中6个C-C键完全等同，是6中心6电子大π键（Π₆⁶），并非单双键交替！配合物中符合休克尔4n+2规则的环系也有大π键（2025 Q2-B、2023 Q5-C）。

陷阱4：结构-性质的错误关联

"VSEPR结构决定所有化学性质"是错误的！SO₃的VSEPR是平面三角形，但其氧化性由S(+6)高价态决定，与分子几何无关。结构只能推测部分性质（2025 Q13-A）。

📝 真题精讲

📌 2023年 · 第5题 分值：2分 | 答案：D
5. 化学处处呈现美。下列说法正确的是
A. 舞台上干冰升华时，共价键断裂
B. 饱和CuSO₄溶液可析出无水蓝色晶体
C. 苯分子的正六边形结构，单双键交替呈现完美对称
D. 晨雾中的光束如梦如幻，是丁达尔效应带来的美景
📋 答案与解析

正确答案：D

A ✗ 干冰（CO₂）升华是物理变化，只破坏分子间作用力（范德华力），共价键没有断裂。

B ✗ 饱和CuSO₄溶液析出的是五水硫酸铜（CuSO₄·5H₂O）蓝色晶体，含结晶水，不是无水CuSO₄。

C ✗ 苯分子的正六边形结构中，六个碳碳键完全等同，是6中心6电子大π键（Π₆⁶），并非单双键交替。

D ✓ 晨雾是气溶胶（胶体），光束通过时发生散射形成光路，这是典型的丁达尔效应。

⚠️ 常见陷阱分析

陷阱A → 看到"升华"就想到"键断裂"。升华是物理变化，只有分子间力被破坏。判断标准：有无新物质生成。

陷阱B → CuSO₄固体是白色，但溶液中析出的晶体是蓝色的CuSO₄·5H₂O。混淆无水物和水合物的颜色是经典错误。

陷阱C → "正六边形"≠"单双键交替"。凯库勒式只是历史简化表示，苯的真实结构是离域大π键。

✅ 避坑指导

1️⃣ 判断"键是否断裂"的核心标准：是否发生化学反应。物理变化 → 键不断裂。

2️⃣ 常见水合晶体颜色速记：CuSO₄·5H₂O（蓝）、FeSO₄·7H₂O（绿）、CoCl₂·6H₂O（粉红）。

3️⃣ 苯的结构特征是6个等同C-C键 + 大π键，不要画成单双键交替。

📌 2024年 · 第9题 分值：2分 | 答案：C
9. 从我国南海的柳珊瑚中分离得到的柳珊瑚酸(下图)，具有独特的环系结构。下列关于柳珊瑚酸的说法不正确的是
A. 能使溴的四氯化碳溶液褪色
B. 能与氨基酸的氨基发生反应
C. 其环系结构中3个五元环共平面
D. 其中碳原子的杂化方式有sp²和sp³
📋 答案与解析

正确答案：C

A ✓ 该物质含有碳碳双键（C=C），能使溴的四氯化碳溶液褪色（亲电加成）。

B ✓ 该物质含有羧基（-COOH），能与氨基酸的氨基（-NH₂）发生反应生成酰胺键。

C ✗ 环系中存在sp³杂化碳原子，具有类似甲烷的四面体结构，导致3个五元环不可能全部共平面。

D ✓ 饱和碳原子为sp³杂化，形成双键的碳原子为sp²杂化。两种杂化共存。

⚠️ 常见陷阱分析

陷阱C → "五元环 → 平面"的错误直觉。判断分子是否共平面的关键：所有环原子是否都是sp²杂化。sp³碳使环系出现"褶皱"，破坏共平面性。

陷阱A → 不要看到羧基就忽略双键！羧基中的C=O不能使溴水褪色（需要C=C），但本题分子中另有独立的C=C双键。

陷阱D → "杂化方式只有一种"的思维定式。复杂有机分子中sp²和sp³共存是常态。

✅ 避坑指导

1️⃣ 杂化判定口诀：单键碳=sp³、双键碳=sp²、叁键碳=sp。看碳原子周围σ键数+孤对电子。

2️⃣ 共平面判定：所有原子必须都是sp/sp²，有一个sp³就不行。

3️⃣ "环系共平面"是高频陷阱选项，看到环+sp³碳就排除。

📌 2025年 · 第2题 分值：2分 | 答案：A
2. 在法拉第发现苯200周年之际，我国科学家首次制备了以金属M为中心的多烯环配合物。该配合物具有芳香性，其多烯环结构(如图)形似梅花。该多烯环上
A. C—H键是共价键
B. 有8个碳碳双键
C. 共有16个氢原子
D. 不能发生取代反应
📋 答案与解析

正确答案：A

A ✓ C和H原子间通过共用电子对形成的是共价键（σ键）。这是最基本的化学键判断。

B ✗ 该配合物具有芳香性，存在共轭大π键（符合休克尔4n+2规则），不是8个独立碳碳双键。M=C也不是普通的碳碳双键。

C ✗ 根据价键理论分析，该物质有10个H原子，而非16个。需要仔细数结构式中环上的氢。

D ✗ 大π键环上可以发生取代反应（类似苯环的亲电取代）。

⚠️ 常见陷阱分析

陷阱B → "有芳香性 ≠ 有8个双键"。芳香性来自大π键（离域π电子），碳碳键完全等同，不是单双键交替。这个陷阱和2023 Q5-C如出一辙！

陷阱C → 看着图"感觉"有16个H。实际上需要一板一眼地数每个碳上的氢：sp²碳连1个H，边缘sp²碳可能连2个H。总数只有10个。

陷阱D → "芳香性=稳定=不反应"的误解。芳香环上的取代反应（卤代、硝化、磺化）是经典反应类型。

✅ 避坑指导

1️⃣ 共价键的判断是最基础也是最常考的，C-H、C-C、C=C都是共价键。

2️⃣ 芳香性 = 大π键 + 休克尔规则（4n+2），不是"多个独立双键"。

3️⃣ 数氢原子：先标出所有碳的杂化方式，sp²碳一般连1个H（环内不连H），sp³碳连2-3个H。

4️⃣ 含大π键的芳香环可以发生取代反应（保留芳香性），但不易发生加成反应（破坏芳香性）。

📌 2025年 · 第13题 分值：2分 | 答案：A
13. 由结构不能推测出对应性质的是
A. SO₃，VSEPR模型为平面三角形 → 具有强氧化性
B. K和Na，原子结构不同 → 焰色不同
C. 乙烯和乙炔，均含π键 → 能发生加聚反应
D. 石墨，层中含离域p电子 → 能导电
📋 答案与解析

正确答案：A

A ✓ 符合题意 — SO₃的VSEPR模型为平面三角形，这只反映其分子几何结构。但SO₃的氧化性由硫的高价态(+6)决定，与VSEPR构型无直接关联。所以由VSEPR结构不能推测氧化性。

B ✗ — K和Na原子结构差异（核外电子排布不同）→ 电子跃迁能量不同 → 焰色不同。结构与性质对应合理。

C ✗ — 乙烯和乙炔均含π键，π键容易断裂 → 是加聚反应的基础。结构与性质对应合理。

D ✗ — 石墨层中含有离域p电子（大π键），电子可在层内自由移动 → 导电性。结构与性质对应合理。

⚠️ 常见陷阱分析

核心陷阱 → 这道题的逆向问法是最大的陷阱！题目问的是"不能推测"，选A（不能推测的），其他三个都是"能推测"的。很多同学习惯性选"正确"选项而丢分。

陷阱A → 把"几何结构(VSEPR)"和"氧化还原性"混为一谈。氧化性取决于化合价（氧化态），不是分子的空间排列。SO₃的氧化性来自S(+6)→S(+4)的还原趋势，与其平面三角形结构无关。

陷阱D → 石墨层内导电 ≠ 层间导电。石墨是各向异性导体，层内因离域π电子而导电，层间不导电。

✅ 避坑指导

1️⃣ 结构-性质对应关系要区分"能推测"和"不能推测"：

✅ 能推测：π键→加成/加聚、离域电子→导电、原子结构→光谱性质

❌ 不能推测：VSEPR→氧化性、分子极性→酸碱性（不直接）

2️⃣ 氧化还原性由元素化合价和电负性决定，不由分子几何构型决定。

3️⃣ 石墨导电原理：层内sp²碳+剩余p轨道形成离域大π键，电子可在层内自由移动。
